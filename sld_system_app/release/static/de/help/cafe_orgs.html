Die Organisationsverwaltung bietet Zugriff auf Funktionen zum Anlegen und Verwalten von Vertriebsorganisationen. Als Vertriebsorganisation bezeichnet man die Organisation, die als "Besitzer" oder "Wurzel" des Multi-Channel-Vertriebssystems agiert. Sobald die Vertriebsorganisation angelegt und aktiviert ist, kann sie ein System von Channels und Organisationen aufsetzen, das ihr Geschäftsmodell und ihre Nachfragekette abbildet, kann Content publizieren und Bestellungen bearbeiten.

Vertriebsorganisationen anzeigen

Um eine Liste aller Vertriebsorganisationen im System anzuzeigen:

  1. Klicken Sie in der Navigationsleiste auf Vertriebsorganisationen.

    Eine Liste aller Vertriebsorganisationen im System wird angezeigt.

    Hinweis:

    Wenn Sie die Datenbank initialisiert haben, ohne die Beispieldaten in die neue Intershop Commerce Management-Instanz zu importieren, sollte diese Liste leer sein.

Vertriebsorganisation anlegen

Um eine neue Vertriebsorganisationen anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Vertriebsorganisationen.

    Eine Liste aller Vertriebsorganisationen im System wird angezeigt.

  2. Klicken Sie auf Neu.

    Der Dialog Vertriebsorganisation wird angezeigt.

  3. Tragen Sie in die Felder unter dem Register Allgemein alle erforderlichen Angaben ein.

    In der Tabelle unten werden die verfügbaren Felder aufgelistet.

    Tabelle 1: Allgemeine Angaben zur Vertriebsorganisation
    Feld Beschreibung
    Name Name zur Identifizierung der Organisation. Kann bis zu 35 Zeichen lang sein und Sonderzeichen und Ziffern enthalten. Das Feld Name ist obligatorisch.
    Organisations-ID Eine systemweite ID für die Organisation. Kann bis zu 35 Zeichen lang sein und aus den alphanumerischen Zeichen a-z und 0-9 bestehen. Die ID darf keine Leerzeichen oder Sonderzeichen enthalten. Dies ist ein Pflichtfeld. Das System verwendet diesen Wert als Namen der für die neue Organisation erzeugten Site.
    Online (Kontrollkästchen) Setzt den Online-Status der Organisation. Markiert: alle Benutzer können auf die Organisation zugreifen; nicht markiert: auf die Organisation kann nicht zugegriffen werden.
    Beschreibung Angaben zur Beschreibung der Organisation. Kann bis zu 256 Zeichen lang sein und Sonderzeichen und Ziffern enthalten.

  4. Klicken Sie auf Übernehmen.

    Die Vertriebsorganisation wird angelegt, und die Register Kontakt, Attribute, Design und Services werden aktiviert.

    Tragen Sie unter jedem Register die erforderlichen Angaben ein. Weitere Informationen dazu finden Sie in Vertriebsorganisation bearbeiten.

Vertriebsorganisation bearbeiten

Um eine bestehende Vertriebsorganisation zu bearbeiten:

  1. Klicken Sie in der Navigationsleiste auf Vertriebsorganisationen.

    Eine Liste aller Vertriebsorganisationen im System wird angezeigt.

  2. Klicken Sie auf den Namen der Vertriebsorganisation, die Sie bearbeiten wollen.

    Die Details der Vertriebsorganisation werden angezeigt.

  3. Bearbeiten Sie unter jedem Register die erforderlichen Angaben.

    Sie können unter jedem Register Angaben hinzufügen, bearbeiten und löschen. In der Tabelle unten werden die verfügbaren Register und die jeweils anzugebenden Informationen aufgelistet.

    Tabelle 2: Daten für Vertriebsorganisationen
    Register Informationen
    Allgemein Hier können Name, ID und Beschreibung der Organisation angegeben werden.
    Kontakt Hier kann die Standardadresse für die Organisation angegeben werden.
    Attribute Hier können benutzerdefinierte Attribute für die Organisation festgelegt werden.
    Design Hier können Designpakete für die Organisation verwaltet werden.
    Zahlung Hier können Zahlungsarten für die Organisation verwaltet werden.
    Services Hier können, sofern verfügbar, zusätzliche Services für die Organisation verwaltet werden.

Zugriff auf Vertriebsorganisation sperren

Um den Zugriff auf eine Vertriebsorganisation zu sperren oder eine Organisation offline zu schalten:

  1. Klicken Sie in der Navigationsleiste auf Vertriebsorganisationen.

    Eine Liste aller Vertriebsorganisationen im System wird angezeigt.

  2. Klicken Sie auf den Namen der Vertriebsorganisation, die Sie bearbeiten wollen.

    Die Details der Vertriebsorganisation werden angezeigt.

  3. Markieren Sie im Register Allgemein den neuen Status der Organisation.

    Aktivieren oder deaktivieren Sie das Online-Kontrollkästchen.

  4. Klicken Sie auf Übernehmen.

    Die Organisation wird auf den neuen Status gesetzt.

Vertriebsorganisation löschen

Um eine oder mehrere Vertriebsorganisationen zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Vertriebsorganisationen.

    Eine Liste aller Vertriebsorganisationen im System wird angezeigt.

  2. Markieren Sie das Kontrollkästchen neben der Vertriebsorganisation, die Sie löschen wollen.

    Sie können mehr als eine Vertriebsorganisation zum Löschen markieren, indem Sie auf die entsprechenden Kontrollkästchen klicken.

  3. Klicken Sie auf Löschen.

    Die markierten Vertriebsorganisationen werden aus dem System entfernt.

Hinweis:

Beim nächsten Ablauf des Jobs "DeleteDomainReferences" werden Organisationen komplett entfernt und deren IDs wieder verfügbar gemacht.

Designverwaltung

Die Organisationsverwaltung kann für jede Vertriebsorganisation individuell angepasst werden. Anpassungen können dabei Logos, Farben und andere Einstellungen betreffen, die für das äußere Erscheinungsbild der Organisationsverwaltung verantwortlich sind. Das Design beruht auf sogenannten Designpaketen, die zuerst geladen und dann relativ zu einer Locale installiert werden müssen, um wirksam zu werden.

Modul Design öffnen

Um das Modul Design zu öffnen:

  1. Klicken Sie in der Navigationsleiste auf Vertriebsorganisationen.

    Eine Liste aller Vertriebsorganisationen im System wird angezeigt.

  2. Klicken Sie auf den Namen der Vertriebsorganisation, die Sie bearbeiten wollen.

    Die Details der Vertriebsorganisation werden angezeigt.

  3. Öffnen Sie das Register Design.

    Eine Liste der bereits geladenen Designpakete wird angezeigt.

Design-Archivdatei laden

Durch das Laden einer Design-Archivdatei wird ein neues Designpaket erzeugt. Um eine Design-Archivdatei zu laden:

  1. Öffnen Sie das Modul Design.

    Für weitere Informationen, siehe Modul Design öffnen.

  2. Klicken Sie auf Upload.

    Der Dialog Design wird angezeigt.

  3. Geben Sie einen Namen für das Designpaket, eine Beschreibung sowie Pfadangabe und Name der Archivdatei an.

  4. Klicken Sie auf Übernehmen.

    Das Designpaket wird geladen und kann jetzt installiert werden.

Designpaket installieren

Um ein Designpaket zu installieren:

  1. Öffnen Sie das Modul Design.

    Für weitere Informationen, siehe Modul Design öffnen.

  2. Klicken Sie in der Zeile des Designpakets, das installiert werden soll, auf Installieren.

    Eine Liste der vorhandenen Installationen dieses Designpakets wird angezeigt.

  3. Wählen Sie im Feld Neue Installation die Locale aus, für die das Paket installiert werden soll.

    Wählen Sie aus der Dropdown-Liste eine der vorhandenen Locales aus, und klicken Sie auf Installieren.

Designpaket deinstallieren

Um ein Designpaket für eine Organisation zu deinstallieren:

  1. Öffnen Sie das Modul Design.

    Für weitere Informationen, siehe Modul Design öffnen.

  2. Klicken Sie in der Zeile des Designpakets, das deinstalliert werden soll, auf Installieren.

    Eine Liste der vorhandenen Installationen dieses Designpakets wird angezeigt.

  3. Markieren Sie die Installationen, die Sie entfernen wollen.

    Markieren Sie die Kontrollkästchen neben den Installationen, die Sie löschen wollen.

  4. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Die Installation wird entfernt.

Designpaket entfernen

Um ein Designpaket von einer Organisation zu entfernen:

  1. Öffnen Sie das Modul Design.

    Für weitere Informationen, siehe Modul Design öffnen.

  2. Markieren Sie das Designpaket, das Sie löschen wollen.

    Markieren Sie das Kontrollkästchen neben dem Designpaket, das Sie entfernen wollen.

  3. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Das Designpaket wird entfernt.

Zahlungsarten verwalten

Intershop Commerce Management bietet direkte Unterstützung für die folgenden Zahlungsarten:

  • Nachnahme

  • Bankeinzug

  • Rechnungskauf

  • Vorkasse

  • Geschenkkarte

Außerdem besitzt Intershop Commerce Management Zahlungsarten als "Helfer", die als Vorlage für die Implementierung eigener Zahlungsarten dienen:

Achtung:

Demo-Zahlungsarten bei Intershop Commerce Management sind nicht PA-DSS-konform.

Die Demo-Zahlungsarten, die Intershop Commerce Management liefert, sind:

  • Einfache Kreditkarte

    Eine Ersatzmethode ausschließlich für Zwecke der Bestellverwaltung, die für Bestellungen verwendet wird, bei denen die ursprüngliche Zahlungsart nicht mehr verfügbar ist.

  • ISH Credit Card

    Eine Demo-Methode, die als Vorlage für den Einsatz neuer Zahlungsarten verwendet werden soll. Sie simuliert einen Zahlungsdienst mit kreditkarten-spezifischen Zahlungsarten. In Bezug auf Bestellverwaltung beschreibt es die generell verfügbaren Transaktionen der Zahlungsabwicklung.

  • ISH Online Pay

    Eine Demo-Methode, die als Vorlage für den Einsatz neuer Zahlungsarten verwendet werden soll. Sie simuliert einen Zahlungsdienst, der die Zahlungsdaten vom Kunden entgegennimmt und eine Überweisung vom Bankkonto des Kunden auf das Bankkonto des Shops veranlasst.

  • ISH Fast Pay

    Eine Demo-Methode, die als Vorlage für den Einsatz neuer Zahlungsarten verwendet werden soll. Sie simuliert eine "Express-Zahlung", die den Kunden an einen externen Zahlungsdienst zur Authentifizierung und eigentlichen Zahlung weiterleitet.

Spezielle Zahlungsdienst-Adapter-Cartridges fügen dem Intershop Commerce Management-System neue Zahlungsarten hinzu. In der Organisationsverwaltung stellt der zentrale Systemadministrator neue Zahlungsarten für die Vertriebsorganisation zur Verfügung, die ihrerseits die neuen Zahlungsarten in ihren Channels aktivieren muss.

Um Zahlungsarten-Zuordnungen für eine Vertriebsorganisation zu verwalten:

  1. Klicken Sie in der Navigationsleiste auf Vertriebsorganisationen.

    Eine Liste aller Vertriebsorganisationen im System wird angezeigt.

  2. Klicken Sie auf den Namen der Vertriebsorganisation, die Sie bearbeiten wollen.

    Die Details der Vertriebsorganisation werden angezeigt.

  3. Öffnen Sie das Register Zahlung.

    Eine Liste der im System verfügbaren Zahlungsarten wird angezeigt.

  4. Klicken Sie auf das Kontrollkästchen der Zahlungsart, die Sie für die aktuelle Vertriebsorganisation aktivieren bzw. deaktivieren wollen.

  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Hinweis:

Informationen zum Einrichten zusätzlicher Zahlungsdienste finden Sie in der Dokumentation der jeweiligen Adapter-Cartridges.

Services verwalten

Intershop Commerce Management stellt Service-Schnittstellen bereit, um mittels spezieller Adapter-Cartridges externe Business-Services wie z. B. Zahlungs-, Web-Analyse-, E-Mail-, Bestellverarbeitungs-, PIM- oder ERP-Lösungen einzubinden.

Nach der Installation einer Service-Adapter-Cartridge werden die damit eingebundenen Dienste über die Organisationsverwaltung den Vertriebsorganisationen zur Verfügung gestellt.

Intershop Commerce Management enthält Integrationen der folgenden Services als Beispiel und Basis für projektspezifische Service-Integrationen:

  • Geschenkkarten-Service

  • E-Mail-Service

  • Captcha-Service

  • Adressprüfungs-Service

  • Bestand-Service

Um Service-Zuordnungen für eine Vertriebsorganisation zu verwalten:

  1. Klicken Sie in der Navigationsleiste auf Vertriebsorganisationen.

    Eine Liste aller Vertriebsorganisationen im System wird angezeigt.

  2. Klicken Sie auf den Namen der Vertriebsorganisation, die Sie bearbeiten wollen.

    Die Details der Vertriebsorganisation werden angezeigt.

  3. Öffnen Sie das Register Services.

    Die verfügbaren Servicegruppen werden angezeigt (Web-Analyse, Bestellungen, E-Mail usw.).

  4. Klicken Sie auf den Namen einer Service-Gruppe.

    Die verfügbaren Services der gewählten Service-Gruppe werden angezeigt.

  5. Klicken Sie auf das Kontrollkästchen der Services, die Sie für die aktuelle Vertriebsorganisation aktivieren bzw. deaktivieren wollen.

  6. Klicken Sie auf Aktivieren bzw. Deaktivieren.

    Die markierten Services werden sofort aktiviert bzw. deaktiviert.

Hinweis:

Informationen zum Einrichten zusätzlicher Services finden Sie in der Dokumentation der jeweiligen Adapter-Cartridges.
