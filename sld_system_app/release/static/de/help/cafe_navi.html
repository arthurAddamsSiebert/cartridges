Die Organisationsverwaltung ist eine ISML-basierte Web-Front-Applikation. Sie läuft, wie jede Web-Front-Applikation, in einem normalen Webbrowser.

Zugriff auf die Organisationsverwaltung

Um auf die Organisationsverwaltung mit einem Webbrowser zuzugreifen:

  1. Geben Sie eine URL mit der folgenden Syntax ein:

    http://<SYSTEM.HOST>/INTERSHOP/web/WFS/SLDSystem

    , wobei <SYSTEM.HOST> für die IP-Adresse oder den Servernamen des Intershop Web Server steht.

  2. Melden Sie sich als Administrator der Operations-Site an.

    Verwenden Sie bei einem Standard-Intershop Commerce Management-System folgende Zugangsdaten:

    • Anmeldename: admin

    • Kennwort: !InterShop00!

    • Organisation: Operations

    Hinweis:

    Das Standardkennwort kann unmittelbar beim Einrichten des Intershop Commerce Management-Systems geändert worden sein. Wenden Sie sich in diesem Fall an diejenigen, die das System eingerichtet haben, um das neue Kennwort zu erhalten.

Achtung:

Intershop empfiehlt dringend, entweder das Kennwort des voreingestellten Administrationsbenutzers bei der ersten Anmeldung zu ändern oder neue Administrationsbenutzer mit speziellen Zugriffsrechten anzulegen und den voreingestellten Administrationsbenutzer zu löschen. Für weitere Informationen, siehe Administrationsbenutzer verwalten.

Navigation durch die Organisationsverwaltung

Nach der Anmeldung wird die Startseite der Organisationsverwaltung angezeigt.

Abbildung 1: Startseite der Organisationsverwaltung

Die Seite ist in zwei Bereiche geteilt: links eine Navigationsleiste und rechts der Hauptbereich, in dem Listen oder Dialoge der in der Navigationsleiste ausgewählten Module angezeigt werden.

Die Navigationsleiste enthält die folgenden Abschnitte:

  • Vertriebsorganisationen

    Zugriff auf Funktionen zum Anlegen und Verwalten von Vertriebsorganisationen. Siehe Verwaltung von Organisationen.

  • Benutzer

    Zugriff auf Funktionen zum Anlegen und Verwalten von Administrationsbenutzern. Siehe Administrationsbenutzer verwalten.

  • Klassifikationskataloge

    Zugriff auf Funktionen zum Anlegen und Verwalten von Klassifikationssystemen. Siehe Klassifikationskataloge verwalten.

  • Import und Export

    Zugriff auf Funktionen zum Import und Export von Klassifikationssystemen. Siehe Datenimport und -export verwalten.

  • Site-Management

    Zugriff auf Funktionen zur Steuerung site-spezifischer Einstellungen. Siehe Site-Einstellungen verwalten.

  • Datenreplizierung

    Zugriff auf Funktionen zur Steuerung und Überwachung von Replikationsprozessen. Siehe Replikationsprozesse verwalten.

Mit den Direktzugriff-Links oberhalb des Hauptbereiches kann unmittelbar auf folgende Funktionen zugegriffen werden:

  • Startseite

    Öffnet die Frontend-Startseite.

  • Profil

    Öffnet die Benutzerdetailansicht für den aktuellen Benutzer (admin) zum Bearbeiten von allgemeinen Angaben und Adressdaten.

  • Hilfe

    Offnet die Online-Hilfe.

  • Abmelden

    Meldet den Benutzer (admin) von der aktuellen Session ab und zeigt die Anmeldeseite an.
