Die Organisationsverwaltung bietet Zugriff auf Funktionen zum Anlegen und Verwalten einzelner Systemadministrations-Benutzer. Intershop Commerce Management unterstützt zwei Typen von Systemadministratoren, die sich durch unterschiedliche Zugriffsrechte auszeichnen:

  • Administrator

    Verantwortlich für grundlegende administrative Aufgaben in der Geschäftsorganisation, z. B. die Verwaltung von Vertriebsorganisationen, Klassifikationssystemen oder Applikationseinstellungen und die Steuerung von Replikationsaufgaben.

  • SMC-Administrator

    Verantwortlich für grundlegende Aufgaben in der Systemadministration für Intershop Commerce Management-7-Cluster, z. B. das Verwalten von Jobs oder Logging-Optionen oder das Überwachen der Applikationsfunktionalität über die System-Management.

Administrationsbenutzer anzeigen

Um eine Liste aller Administrationsbenutzer im Intershop Commerce Management-System anzuzeigen:

  1. Klicken Sie in der Navigationsleiste auf Benutzer.

    Eine Liste aller Administrationsbenutzer wird angezeigt.

    Hinweis:

    Die Liste sollte mindestens den Standard-Systemadministrator anzeigen, der bei der Initialisierung des Systems immer angelegt wird.

Administrationsbenutzer anlegen

Um einen neuen Administrationsbenutzer anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Benutzer.

    Eine Liste aller Administrationsbenutzer wird angezeigt.

  2. Klicken Sie auf Neu.

    Der Dialog Neuen Benutzer anlegen wird angezeigt.

  3. Geben Sie die notwendigen Daten an.

    Erforderliche Angaben sind Vorname, Nachname, E-Mail-Adresse, Benutzer-ID und Anmeldename. Die anderen Angaben sind optional.

  4. Klicken Sie auf Übernehmen.

    Der Benutzer wird angelegt, und die anderen Register werden aktiviert.

    Sie können jetzt alle anderen Informationen bearbeiten. Klicken Sie auf Zurück zur Liste, um zurück zur Benutzerliste zu gelangen.

Hinweis:

Beim Anlegen des neuen Benutzers erzeugt Intershop Commerce Management ein Einmal-Kennwort und sendet dieses an die angegebene E-Mail-Adresse. Wenn der neue Benutzer sich zum ersten Mal anmeldet, setzt das System dieses Kennwort automatisch auf "abgelaufen" und fordert den Benutzer auf, ein neues Kennwort anzugeben.

Administrationsbenutzer-Daten bearbeiten

Um Daten eines Administrationsbenutzers zu bearbeiten:

  1. Klicken Sie in der Navigationsleiste auf Benutzer.

    Eine Liste aller Administrationsbenutzer wird angezeigt.

  2. Klicken Sie auf den Namen oder die ID des Benutzers, den Sie bearbeiten wollen.

    Die Detailansicht des Benutzers wird angezeigt.

  3. Bearbeiten Sie die Informationen wie erforderlich.

  4. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Administrationsbenutzer löschen

Um einen oder mehrere Administrationsbenutzer zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Benutzer.

    Eine Liste aller Administrationsbenutzer wird angezeigt.

  2. Markieren Sie die Kontrollkästchen der Benutzer, die entfernt werden sollen.

  3. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Die markierten Benutzer werden aus dem System entfernt.

Hinweis:

Beim nächsten Ablauf des Jobs "DeleteDomainReferences" werden Objekte komplett entfernt und deren IDs wieder verfügbar gemacht.

Administrationsbenutzer-Kennwort ändern

Um das Kennwort eines Administrationsbenutzers zu ändern:

  1. Klicken Sie in der Navigationsleiste auf Benutzer.

    Eine Liste aller Administrationsbenutzer wird angezeigt.

  2. Klicken Sie auf den Namen oder die ID des Benutzers, den Sie bearbeiten wollen.

    Die Detailansicht des Benutzers wird angezeigt.

  3. Geben Sie eine gültige E-Mail-Adresse an.

  4. Klicken Sie auf Neues Kennwort.

    Intershop Commerce Management erzeugt sofort ein Einmal-Kennwort und sendet dieses an die angegebene E-Mail-Adresse.

    Wenn der Benutzer sich das nächste Mal anmeldet, setzt das System dieses Kennwort automatisch auf "abgelaufen" und fordert den Benutzer auf, ein neues Kennwort anzugeben.

Achtung:

Intershop empfiehlt dringend, entweder das Kennwort des voreingestellten Administrationsbenutzers bei der ersten Anmeldung zu ändern oder neue Administrationsbenutzer mit speziellen Zugriffsrechten anzulegen und den voreingestellten Administrationsbenutzer zu löschen.

Deaktivierte Administrationsbenutzer entsperren

Nach sechs gescheiterten Anmeldeversuchen sperrt Intershop Commerce Management Organisationsverwaltung-Benutzerkonten für 30 Minuten sperren.

Um ein gesperrtes Benutzerkonto manuell wieder zu aktivieren:

  1. Klicken Sie in der Navigationsleiste auf Benutzer.

    Eine Liste aller Administrationsbenutzer wird angezeigt. Bei gesperrten Benutzerkonten wird der Status "Vorübergehend deaktiviert" angezeigt.

  2. Wählen Sie den wieder zu aktivierenden Benutzer aus.

    Markieren Sie das Kontrollkästchen neben dem Benutzer, den Sie entsperren wollen.

  3. Klicken Sie auf Konto reaktivieren und bestätigen Sie die Operation mit OK.

    Das ausgewählte Benutzerkonto wird unmittelbar entsperrt, und der entsprechende Benutzer kann sich wieder am System anmelden.

Administrationsbenutzer suchen

Um Administrationsbenutzer zu suchen:

  1. Klicken Sie in der Navigationsleiste auf Benutzer.

    Eine Liste aller Administrationsbenutzer wird angezeigt.

  2. Tragen Sie den Namen oder die ID des Benutzers in das Suchdialogfeld ein.

    Aktivieren Sie die Erweiterte Suche, um auf Grundlage weiterer Benutzerangaben eine detailliertere Suche durchzuführen.

  3. Klicken Sie auf Suchen.

    Eine Liste aller Benutzer, die den Suchkriterien entsprechen, wird angezeigt.

Zugriffsrechte von Administrationsbenutzern verwalten

Um die Zugriffsrechte für Administrationsbenutzer zu verwalten:

  1. Klicken Sie in der Navigationsleiste auf Benutzer.

    Eine Liste aller Administrationsbenutzer wird angezeigt.

  2. Klicken Sie auf den Namen oder die ID des Benutzers, den Sie bearbeiten wollen.

    Die Detailansicht des Benutzers wird angezeigt.

  3. Öffnen Sie das Register Zugriffsrechte.

    Die verfügbaren Zugriffsrechte werden angezeigt.

    Tabelle 3: Zugriffsrechte von Administrationsbenutzern
    Zugriffsrecht Beschreibung
    Administrator Verantwortlich für grundlegende administrative Aufgaben in der Geschäftsorganisation, z. B. die Verwaltung von Vertriebsorganisationen, Klassifikationssystemen oder Applikationseinstellungen und die Steuerung von Replikationsaufgaben.
    SMC-Administrator Verantwortlich für grundlegende Aufgaben in der Systemadministration für Intershop Commerce Management-7-Cluster, z. B. das Verwalten von Jobs oder Logging-Optionen oder das Überwachen der Applikationsfunktionalität über die System-Management.

  4. Markieren Sie die entsprechenden Kontrollkästchen, um die gewünschten Zugriffsrechte zuzuordnen.

  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Adressen von Administrationsbenutzern verwalten

Um die Adressen für Administrationsbenutzer zu verwalten:

  1. Klicken Sie in der Navigationsleiste auf Benutzer.

    Eine Liste aller Administrationsbenutzer wird angezeigt.

  2. Klicken Sie auf den Namen oder die ID des Benutzers, den Sie bearbeiten wollen.

    Die Detailansicht des Benutzers wird angezeigt.

  3. Öffnen Sie das Register Adressen.

    Eine Liste der Adressen wird angezeigt.

  4. Klicken Sie auf Neu, um eine neue Adresse anzulegen.

    Tragen Sie die notwendigen Daten ein und klicken Sie auf Übernehmen, um die Adressdaten zu speichern.

  5. Um vorhandene Adressen zu bearbeiten, klicken Sie auf den Namen der Adresse, die Sie bearbeiten wollen.

    Bearbeiten Sie alle Daten wie erforderlich und klicken Sie auf Übernehmen, um Ihre Einstellungen zu speichern.

  6. Um eine Adresse zu löschen, markieren Sie das Kontrollkästchen neben der betreffenden Adresse, klicken dann auf Löschen und bestätigen die Operation.

    Die markierten Adressen werden gelöscht und von der Liste entfernt.
