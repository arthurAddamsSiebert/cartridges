Im Zusammenhang mit Online Shops gibt es viele Daten-Import- und -Export-Prozesse. Produkt- und Preis-Daten müssen importiert, Bestell-Daten müssen exportiert werden. Die Quelle von Produkt- und Preis-Daten sind häufig Dateien von anderen Servern. Ebenso ist das Ziel für Bestell-Daten oft ein anderer Server. Das Transport Framework wird benutzt, um Dateien von einem Ort des Netzwerks zu einem anderen zu befördern.

Funktionalität

Das Transport Framework erlaubt Datei-Transporte via SFTP, FTP, HTTP oder EMail.

Für SFTP und FTP sind Push und Pull möglich. Aufgrund der Beschaffenheit von HTTP und E-Mail können Dateien per HTTP nur gelesen werden und nur versendet werden per E-Mail.

Voraussetzungen

  • Melden Sie sich in der Organisationsverwaltung an.

    Organisation = Operations

  • Zur Benutzung des Transport Frameworks benötigt der Nutzer die Zugriffsrechte eines Transport Managers , der verantwortlich ist für das Anlegen, Bearbeiten und Löschen von Transportprozess-Konfigurationen.

    Siehe Zugriffsrechte von Administrationsbenutzern verwalten für weitere Informationen, wie das erreicht wird.

Konfigurationsliste

  1. Melden Sie sich in der Organisationsverwaltung (Operations) an.

    Stellen Sie sicher, dass Sie über die Zugriffsrechte verfügen (siehe Voraussetzungen).

  2. Wählen Sie im Menü (links) Transport Konfiguration.

    Die Transport Konfiguration Liste wird angezeigt.

  3. Wählen Sie die Domain mittels der Dropdown-Liste und klicken Sie Übernehmen.

    Bereits vorhandene Konfigurationen werden angezeigt.

Neue Konfiguration anlegen

  1. Navigieren Sie zur Konfigurationsliste der gewünschten Domain.

  2. Geben Sie einen Name.

  3. Wählen Sie einen Typ.

    EMAIL, HTTP, FTP, SFTP

  4. Klicken Sie Hinzufügen.

    Die Detailseite der Konfiguration wird angezeigt.

Konfiguration bearbeiten

  1. Navigieren Sie zur Detailseite der Konfiguration.

    Abhängig vom gewählten Typ der Konfiguration müssen verschiedene Parameter angegeben werden.

  2. Geben Sie die notwendigen Parameter an.

    Jedes Eingabefeld bietet entsprechende Hilfetexte.

  3. Klicken Sie Übernehmen.

Hinweis:

Beachten Sie, dass die Eingabefelder nur über rudimentäre Validierung verfügen. Z. B.: wird eine URL benötigt, wird geprüft, ob der String sysntaktisch korrekt ist, jedoch nicht, ob das Ziel tatsächlich existiert.

Konfiguration löschen

  1. Navigieren Sie zur Konfigurationsliste der gewünschten Domain.

  2. Markieren Sie das Kontrollkästchen der entsprechenden Konfiguration.

    Mehrfachauswahl von Konfigurationen ist möglich.

  3. Klicken Sie Löschen.

    Bestätigen Sie das Löschen OK.

Transport Jobs ausführen

Eine Transport-Konfiguration kann mittels eines Jobs in der System-Management-Applikation ausgeführt werden.

Für allgemeine Informationen zur Zeitplanung von Jobs, siehe Kapitel Schedule Management in der Onlinehilfe der System-Management-Applikation.

Für eine Schritt-für-Schritt-Anleitung für die Ausführung einer Transport-Konfiguration, siehe Guide - Transport Framework in der Intershop Knowledge Base.
