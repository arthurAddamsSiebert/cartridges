Klassifikationskataloge verwalten

Klassifikationskataloge definieren einen Satz von Katalogkategorien, denen Produkte zugeordnet werden können. Intershop Commerce Management bietet direkte Unterstützung für Klassifikationskataloge nach eCl@ss- und UN/SPSC-Standard. Außerdem können Sie eigene Klassifikationssysteme definieren sowie neue oder andere Versionen von Standardklassifikationssystemen importieren.

Die Organisationsverwaltung unterstützt die folgenden klassifikationskatalog-relevanten Aufgaben:

  • Klassifikationskataloge, einschließlich der vordefinierten eCl@ss- und UN/SPSC-Kataloge, pflegen,

  • neue Klassifikationskataloge anlegen und Klassifikationssysteme importieren (siehe Datenimport und -export verwalten),

  • Klassifikationskataloge online oder offline setzen.

Hinweis:

Klassifikationskataloge, die in der Organisationsverwaltung erstellt und online gestellt wurden, sind für alle Vertriebsorganisationen global verfügbar. Die Vertriebsorganisationen wiederum schalten diese dann für ihre Channels frei.

Dieser Abschnitt erklärt das Anlegen, Bearbeiten, Aktivieren und Löschen von globalen Klassifikationskatalogen. Andere Aufgaben, wie z. B. das Verwalten von Katalogkategorien und Produkten oder von Klassifikationsattributen, werden von der Vertriebsorganisation im Commerce Management ausgeführt..

Klassifikationskatalog anlegen

Um einen Klassifikationskatalog anzulegen:

  1. Öffnen Sie das Modul Klassifikationskataloge.

    Klicken Sie in der Navigationsleiste auf Klassifikationskataloge. Eine Liste aller globalen Klassifikationskataloge im System wird angezeigt.

  2. Klicken Sie auf Neu.

    Der Dialog Klassifikationskatalog wird angezeigt.

  3. Wählen Sie aus der Dropdown-Liste die Locale (Sprachumgebung) aus, für die der Katalog angelegt werden soll.

    Sie können für jede vom System unterstützte Sprache einen Katalog anlegen. Sie sollten jedoch immer zuerst die Standard-Locale des Systems auswählen. Auf diese Locale wird zurückgegriffen, wenn eine Locale, die ein Benutzer zum Katalogdurchsuchen ausgewählt hat, vom System nicht unterstützt wird.

  4. Geben Sie einen Namen, eine Katalog-ID und eine Beschreibung in die entsprechenden Felder ein.

    Anzeigename und Katalog-ID sind obligatorisch.

    Hinweis:

    Der Name kann bis zu 256 Zeichen lang sein und darf keine Leerzeichen enthalten; die ID kann bis zu 28 Zeichen lang sein und darf nur aus den alphanumerischen Zeichen a-z und 0-9 bestehen.

  5. Geben Sie den Katalogtyp an.

    Wählen Sie aus der Dropdown-Liste Custom, UN/SPSC oder eCl@ss aus.

  6. Klicken Sie auf Übernehmen.

    Der neue Katalog wird angelegt.

    Der Katalog kann bearbeitet werden, wenn Sie in der Katalog-Listenansicht auf Bearbeiten klicken.

Klassifikationskatalog bearbeiten

Um einen Klassifikationskatalog zu bearbeiten:

  1. Öffnen Sie das Modul Klassifikationskataloge.

    Klicken Sie in der Navigationsleiste auf Klassifikationskataloge. Eine Liste aller globalen Klassifikationskataloge im System wird angezeigt.

  2. Klicken Sie auf Bearbeiten neben dem Katalog, den Sie bearbeiten wollen.

    Bearbeiten Sie die Details wie erforderlich.

Klassifikationskatalog online stellen

Kataloge können online oder offline sein. Offline-Kataloge sind für keine Vertriebsorganisation verfügbar. Um einen Katalog online oder offline zu setzen:

  1. Öffnen Sie das Modul Klassifikationskataloge.

    Klicken Sie in der Navigationsleiste auf Klassifikationskataloge. Eine Liste aller globalen Klassifikationskataloge im System wird angezeigt.

  2. Klicken Sie auf Bearbeiten neben dem Katalog, den Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird angezeigt.

  3. Öffnen Sie das Register Online/Offline.

  4. Setzen Sie im Feld Status "Online" oder "Offline" als Wert.

  5. Geben Sie den Gültigkeitszeitraum für den ausgewählten Status an.

    Der Gültigkeitszeitraum legt fest, wann der gewählte Status auf den Katalog angewendet wird. Wählen Sie eine der zwei Optionen "Immer" oder "Beschränkter Zeitraum". Wenn Sie einen beschränkten Zeitraum wählen, müssen Sie mindestens Startzeit und -datum angeben.

Klassifikationskatalog löschen

Um einen Klassifikationskatalog zu löschen:

  1. Öffnen Sie das Modul Klassifikationskataloge.

    Klicken Sie in der Navigationsleiste auf Klassifikationskataloge. Eine Liste aller globalen Klassifikationskataloge im System wird angezeigt.

  2. Klicken Sie auf Löschen neben dem Katalog, den Sie löschen wollen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  3. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Der Katalog wird aus dem System entfernt.
