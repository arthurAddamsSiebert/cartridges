License Auditing

Intershop is authorized to request license reports from Intershop Commerce Management customers. Use the License Report module to generate a license report and to send it to Intershop, according to your purchasing contract and license agreement.

Caution:

With multi-core technology, each CPU core is considered a CPU for licensing purposes.

To generate a license report:

  1. In the System Management navigation bar, select License Report.

    The License Report module is displayed.

    Figure 13: Generating a license report

  2. Specify a start and end date, and click Generate to start the report generation.

    The license report contains the contents of your license.xml file, lists all installed cartridges and provides general cluster information.

  3. Specify the e-mail addresses of the report's recipient and sender.

    Your purchasing contract or license agreement should specify an e-mail address where generated license reports are to be sent, for example license@intershop.com.

    Figure 14: Submitting the license report

  4. Enter the required e-mail addresses and click "Send As E-Mail" to send the license report to the specified e-mail addresses.

    Alternatively, click "Download Report" to download the report as encrypted XML file.
