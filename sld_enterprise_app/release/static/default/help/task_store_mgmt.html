Note:

The tasks described in this section are performed in the context of a sales channel.

Viewing Stores

To view the stores defined for a sales channel:

  1. Select the channel from the context selection box.

    This opens the navigation bar of the channel.

  2. In the navigation bar, select Stores.

    This displays the overview page.

  3. Select Stores.

    This displays a list view of all stores defined for the channel.

Creating a Store

To create a new store:

  1. Select the channel from the context selection box, then select Stores | Stores.

    This displays a list view of all stores defined for the channel.

  2. Click New.

    The General tab of the Store detail view is displayed.

  3. Specify the mandatory information.

    Mandatory information includes store ID, country and city, ZIP/postal code, street and number, as well as state. Select the Active check box to activate the store.

  4. Click Apply.

    This creates the store, and the Attributes tab becomes active.

  5. Optionally, define attributes as required, then click Apply.

    Clicking Back to List returns you to the store list.

Editing a Store

To edit the details of a store:

  1. Select the channel from the context selection box, then select Stores | Stores.

    This displays a list view of all stores defined for the channel.

  2. Click the name of the store you intend to edit.

    The General tab of the Store detail view is displayed.

  3. Edit the information as required.

    On the Attributes tab, you can manage custom store attributes.

  4. Click Apply to save your settings.

    Otherwise, click Reset to discard your changes.

Deleting a Store

To delete a store for a sales channel:

  1. Select the channel from the context selection box, then select Stores | Stores.

    This displays a list view of all stores defined for the channel.

  2. Select the checkbox for the store to be deleted.

  3. Click Delete, and confirm the operation.

    This removes the selected store from the sales channel.
