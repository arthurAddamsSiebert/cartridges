Partner Branding Management

  • Uninstall Branding Package
  • Remove Branding Package
Note:

The tasks described in this section are performed in the context of a partner channel.

Upload Branding Package

To upload a new branding package for a partner:

  1. Open the partner manager.

    Select the partner channel from the context selection box, then select Partners. This displays the partner list.

  2. Click the name or ID of the partner.

    This displays the partner detail view.

  3. Open the Branding tab.

    This opens a list of all branding packages that have been uploaded.

  4. Click Upload.

    The New Branding dialog is displayed.

  5. Provide a name for the branding package, a description, and locate the branding archive file.

  6. Click Apply.

    The new branding package is uploaded and can now be installed.

Install Branding Package

To install a branding package for a partner:

  1. Open the partner manager.

    Select the partner channel from the context selection box, then select Partners. This displays the partner list.

  2. Click the name or ID of the partner.

    This displays the partner detail view.

  3. Open the Branding tab.

    This opens a list of all branding packages that have been uploaded.

  4. Click Install in the row of the branding package you intend to install.

    A list of all existing installations of this branding package is displayed.

  5. In the New Installation field, select the locale for the installation and click Install.

    Select one of the available locales for the branding package from the drop-down list and click Install.

Uninstall Branding Package

To uninstall a branding package for a partner:

  1. Open the partner manager.

    Select the partner channel from the context selection box, then select Partners. This displays the partner list.

  2. Click the name or ID of the partner.

    This displays the partner detail view.

  3. Open the Branding tab.

    This opens a list of all branding packages that have been uploaded.

  4. Click Install in the row of the branding package you intend to uninstall.

    A list of all existing installations for this branding package is displayed.

  5. Select the installation you intend to remove.

    Check the box in the row of the branding package installation you intend to delete.

  6. Click Delete.

    The installation is removed.

Remove Branding Package

To remove a branding package from a partner:

  1. Open the partner manager.

    Select the partner channel from the context selection box, then select Partners. This displays the partner list.

  2. Click the name or ID of the partner.

    This displays the partner detail view.

  3. Open the Branding tab.

    This opens a list of all branding packages that have been uploaded.

  4. Select the checkbox for the branding package to be deleted.

  5. Click Delete and confirm the deletion.

    This removes the branding package from the system.
