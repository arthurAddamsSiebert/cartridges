Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Der Import und Export von URLs wird auf der Seite Import & Export durchgeführt, welche über die Übersichtsseite der URL-Verwaltung verfügbar ist.

Upload von Importdateien

Bevor Sie Kurz-URLs in Intershop Commerce Management importieren können, müssen Sie eine entsprechende Datei hochladen. Um eine Datei für den Import hochzuladen:

  1. Gehen Sie zur Seite Kurz-URL Import & Export.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie dann Import & Export.

  2. Klicken Sie auf Upload in der Zeile Import- und Exportdateien.

    Eine Liste der bereits für den Import geladenen Dateien wird angezeigt.

  3. Geben Sie im Feld Datei laden den Datei-Pfad an oder klicken Sie auf Durchsuchen, um die Datei zu suchen.

  4. Klicken Sie auf Upload, um die Datei hochzuladen.

    Die hochgeladene Datei wird in der Liste angezeigt. Sie kann jetzt importiert werden.

Tip:

Um Systemressourcen wieder freizugeben, empfiehlt Intershop Ihnen, die hochgeladenen Importdateien nach dem erfolgtem Importvorgang zu löschen.

Download von Export-Dateien

Wenn Daten erst in eine Datei exportiert wurden, kann diese aus dem System heruntergeladen werden. Gehen Sie wie folgt vor:

  1. Gehen Sie zur Seite Kurz-URL Import & Export.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie dann Import & Export.

  2. Klicken Sie in der Zeile Import- und Exportdateien auf Download.

    Eine Liste der bereits exportierten Dateien wird angezeigt.

  3. Klicken Sie auf einen Dateinamen, um die entsprechende Datei zu speichern.

    Der Dialog Download wird geöffnet und fordert Sie auf, die Datei zu speichern.

Tip:

Nach erfolgtem Download empfiehlt Intershop die Exportdateien zu löschen, um Systemressourcen wieder freizugeben.

URLs exportieren

Sie können Storefront-URLs für Kategorien und Produkte dieses Channels im CSV-Format exportieren. Um dies zu tun:

  1. Gehen Sie zur Seite Kurz-URL Import & Export.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie dann Import & Export.

  2. Klicken Sie in der Zeile URL-Export auf Export.

    Eine Liste der Kataloge im Channel wird angezeigt.

  3. Markieren Sie die zu exportierenden Kataloge, und klicken Sie auf Weiter.

    Alternativ können Sie auch auf einen Katalognamen klicken, um den Kategoriebaum zu öffnen und eine Kategorie zum Export wählen.

  4. Konfigurieren Sie den Exportbereich und geben Sie einen Namen für die Exportdatei an.

    Wählen Sie, ob Sie die URLs von zugeordneten Produkten und die gewählten Kategorien exportieren wollen.

  5. Klicken Sie auf Export.

    Der Exportvorgang wird sofort gestartet. Der Import- und Exportstatus wird angezeigt.

  6. Laden Sie die Exportdatei herunter.

    Für weitere Informationen, siehe Download von Export-Dateien .

Export von Kurz-URLs (CSV)

Sie können bestehende Kurz-URLs ins CSV-Format exportieren. Um dies zu tun:

  1. Gehen Sie zur Seite Kurz-URL Import & Export.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie dann Import & Export.

  2. Klicken Sie in der Zeile URLs (CSV) auf Export.

    Eine Liste der Kurz-URLs im Management-Kontext wird angezeigt.

  3. Markieren Sie die zu exportierenden Kurz-URLs, und klicken Sie auf Export.

  4. Geben Sie einen Namen für die Exportdatei an, und klicken Sie erneut auf Export.

    Der Exportvorgang wird sofort gestartet. Der Import- und Exportstatus wird angezeigt.

  5. Laden Sie die Exportdatei herunter.

    Für weitere Informationen, siehe Download von Export-Dateien .

Import von Kurz-URLs

Kurz-URLs können aus CSV- oder XML-Dateien importiert werden. Gehen Sie wie folgt vor:

  1. Gehen Sie zur Seite Kurz-URL Import & Export.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie dann Import & Export.

  2. Laden Sie die zu importierende Datei hoch.

    Für weitere Informationen, siehe Upload von Importdateien .

  3. Auf der Seite Import & Export klicken Sie auf eine Import-Schaltfäche je nach hochgeladenem Dateiformat.

    Eine Liste der für den Import verfügbaren Formate wird angezeigt.

  4. Markieren Sie die zu importierende Datei, und klicken Sie auf Weiter.

    Wenn Sie CSV-Dateien importieren wollen, bestimmen Sie folgende zusätzliche Import-Parameter:

    • Zuordnungsvorlage

    • Feldtrenner

    • Formatzeile

    • Benutzerdefinierte Formatzeile

    Die Importdatei wird jetzt validiert. Bei Erfolg wird die vom Parser erkannte Anzahl der URLs angezeigt. Bei Fehlern wird eine herunterladbare Logdatei mit Fehlermeldungen erzeugt.

  5. Klicken Sie auf Weiter und wählen Sie den Importmodus.

    Für den Import von URLs stehen die Modi Aktualisieren und Ersetzen zur Verfügung. Für weitere Informationen zu Importmodi, siehe Standard-Importmodi.

  6. Klicken Sie auf Import.

    Der Importvorgang wird gestartet und Sie werden auf die Seite Import & Export zurückgeleitet, wo Sie den Status der Importvorgänge einsehen können.
