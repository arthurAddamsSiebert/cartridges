Content-Redakteure können im Commerce Management Includes bearbeiten. Für Informationen zu Includes und anderen Content-Management-Konzepten, siehe Content-Management: Begriffe.

Hinweis:

Includes sind Entwicklungsartefakte, d.h. Benutzer des Commerce Management können Includes nur konfigurieren und ihnen Komponenten zuordnen.

Includes Anzeigen

Um alle verfügbaren Includes für einen bestimmten Kontext anzuzeigen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox.

    Die Navigationsleiste für den Managementkontext wird geöffnet.

  2. Klicken Sie in der Navigationsleiste auf Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  3. Wählen Sie den Applikationstyp aus (optional).

    Im Channel- oder Organisationskontext zeigt die Liste alle Includes aller im jeweiligen Kontext verfügbaren Applikationstypen an. Mit der Auswahl eines Applikationstyps legen Sie das Content-Modell, mit dem Sie arbeiten werden, fest (siehe Was ist eine Applikation?) und filtern die Includes entsprechend.

    Abbildung 41: Auswahl des Content-Modells
    Tip:

    Diese Einstellung ist während Ihrer gesamten Commerce Management-Sitzung aktiv. Um die Auswahl aufzuheben, wählen Sie einen anderen Applikationstyp oder "Alle" aus.

Includes suchen

Einfache Include-Suche

In der einfachen Suche lassen sich Includes nach Namen oder ID suchen. Falls erforderlich, können Sie die Suche auf Includes einer bestimmten Gruppe oder eines bestimmten Typs oder mit bestimmtem Label eingrenzen.

Um die einfache Suche zu verwenden:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Geben Sie den Namen des Includes oder die ID als Suchbegriff ein.

    Falls erforderlich, wählen Sie einen Seitentyp oder ein Label aus, um die Suche einzugrenzen.

  3. Klicken Sie auf Suchen.

    Die Suchergebnisse werden in der Listenansicht angezeigt.

Tip:

Standardmäßig wird die einfache Suche angezeigt. Wenn Sie zuvor die erweiterte Suche verwendet haben, klicken Sie auf Einfache Suche.

Erweiterte Include-Suche

Mit der erweiterten Suche können Sie durch die Angabe exakter Suchkriterien gezielter suchen. Um die erweiterte Suchmaske zu verwenden:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt. Standardmäßig wird die einfache Suche angezeigt.

  2. Klicken Sie auf Erweiterte Suche.

  3. Geben Sie Ihre Suchkriterien in den entsprechenden Feldern an.

    Die Felder sind selbsterklärend.

  4. Klicken Sie auf Suchen.

    Die Suchergebnisse werden in der Listenansicht angezeigt.

Includes sperren

Bevor Sie ein Include bearbeiten, sollten Sie sie sperren. Die Sperre verhindert, dass das Include zur gleichen Zeit von anderen Benutzern bearbeitet werden kann. Die Sperre kann manuell gelöst werden. Nach Ablauf eines festgelegten Intervalls wird die Produktsperre automatisch aufgehoben. Information zum Konfigurieren der Includesperre, siehe Zeitraum für Content-Sperre festlegen.

Tip:

Beim Bearbeiten eines Includes wird die Seite automatisch für Sie gesperrt, falls sie nicht schon vorher von einem anderen Benutzer gesperrt wurde.

Includes können auf verschiedene Weise gesperrt werden: entweder direkt über der Include-Liste oder über die Include-Detailansicht.

Includes in der Listenansicht sperren

Um Includes über die Listenansicht zu sperren:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf das Symbol in der Zeile des Includes, das Sie bearbeiten wollen.

    Falls das Include gegenwärtig von einem anderen Benutzer bearbeitet wird, erscheint eine Fehlermeldung.

Hinweis:

Klicken Sie auf das Symbol , um die Include-Sperre aufzuheben, wenn Sie mit der Bearbeitung fertig sind.

Includes in der Detailansicht sperren

Um ein Include über die Include-Detailansicht zu sperren:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Klicken Sie auf das Symbol oben im Dialog.

    Falls das Include gegenwärtig von einem anderen Benutzer bearbeitet wird, erscheint eine Fehlermeldung.

Hinweis:

Klicken Sie auf das Symbol , um die Include-Sperre aufzuheben, wenn Sie mit der Bearbeitung fertig sind.

Includes bearbeiten

Allgemeine Optionen zur Bearbeitung von Includes sind das Bearbeiten von Include-Content und das Festlegen verfügbarer Einstellungen der Includes. Vor dem Bearbeiten müssen sie gesperrt werden, siehe Includes sperren.

Include-Content bearbeiten

Um den Inhalt von Includes zu bearbeiten:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

    Tip:

    Das Register Content zeigt eine "Komfort"-Ansicht für den Content der Komponente mit der höchsten Priorität, siehe Komponentenpriorität festlegen.

  3. Legen Sie die erforderlichen Parameter-Werte fest und bearbeiten Sie den Content wie erforderlich.

    Die verfügbaren Parameter oder Content-Elemente hängen vom entsprechenden Komponententyp ab.

  4. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Include-Eigenschaften festlegen

Um die Eigenschaften eines Includes festzulegen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Wechseln Sie zum Register Eigenschaften.

    Das Register Eigenschaften ermöglicht die Bearbeitung des Anzeigennamens und der Beschreibung einer gewählten Locale.

  4. Bearbeiten Sie Anzeigennamen und Beschreibung wie erforderlich.

  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Komponenten zu Includes hinzufügen

Das Register Komponenten der Detailansicht ermöglicht das explizite Zuordnen einer Komponente zu einem Include.

Hinweis:

Nur publizierte Komponenten können online sichtbar sein, unabhängig von ihren Include-Zuordnungen.

Verfügbare Komponenten beschränken

Um die Komponenten für mögliche Zuordnungen zu beschränken:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen oder die ID des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Beschränkungen.

    Die Liste mit Komponenten-Templates wird gezeigt, auf deren Basis Sie dem aktuellen Include Komponenten zuordnen können.

  4. Wählen Sie die Komponenten-Templates, die Sie zulassen wollen.

    Markieren Sie das/die Kontrollkästchen bzw. entfernen Sie die Markierung der entsprechenden Komponenten-Templates.

  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Hinweis:

Ein ausgewähltes Kontrollkästchen macht die Komponenten aufgrund des entsprechenden Komponenten-Templates für die Zuordnung zum aktuellen Include verfügbar. Ein unmarkiertes Kontrollkästchen beschränkt jedoch die Zuordnung von Komponenten zum aktuellen Include aufgrund des zugehörigen Komponenten-Templates.

Bestehende Komponenten zuordnen

Um eine bestehende Komponente in ein Include einzufügen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Komponenten.

    Die dem Include zugeordneten Komponenten werden in einer Listenansicht gezeigt.

  4. Klicken Sie auf Zuordnen.

    Der Dialog Komponente zuordnen wird angezeigt. Unterhalb des Abschnitts Priorität und Sichtbarkeit listet der Dialog die verfügbaren Komponenten auf. Falls erforderlich, können Sie die Liste durch Angabe eines Komponentennamens oder -typs eingrenzen.

  5. Legen Sie den Sichtbarkeitszeitraum und die Priorität fest.

  6. Wählen Sie die Komponten aus, die dem Include zugeordnet werden sollen.

  7. Klicken Sie auf Zuordnen.

    Die Komponenten werden dem aktuellen Include zugeordnet.

Neue Komponenten zuordnen

Um eine neue Komponente anzulegen und sie direkt einem Include zuzuordnen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Komponenten.

    Die dem Include zugeordneten Komponenten werden in einer Listenansicht gezeigt.

  4. Klicken Sie Neu.

    Der Dialog Neue Komponente wird angezeigt.

  5. Bearbeiten Sie die Details der Komponente wie erforderlich.

    Geben Sie einen Namen an, bearbeiten Sie die ID, falls erforderlich, wählen Sie ein Komponenten-Template aus, definieren Sie den Publikationszeitraum und legen Sie den Sichtbarkeitszeitraum und die Priorität fest.

  6. Klicken Sie Übernehmen.

    Die neue Komponente wird angelegt und in der Detailansicht zur Bearbeitung geöffnet, siehe Komponenten bearbeiten.

    Um zur Include-Detailansicht zurückzukehren, klicken Sie auf Zurück zur Liste.

Komponenten verschieben

Zugeordnete Komponenten können von einem Include zu einem anderen verschoben werden. Um dies zu tun:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Komponenten.

    Die dem Include zugeordneten Komponenten werden in einer Listenansicht gezeigt.

  4. Markieren Sie die Zuordnung, die Sie verschieben wollen.

    Markieren Sie das Kontrollkästchen der entsprechenden Komponentenzuordnung.

  5. Klicken Sie auf Auswahl bearbeiten.

    Die verfügbaren Stapelprozesse werden angezeigt.

  6. Wählen Sie Ausgewählte Komponenten zu Includes zuordnen oder Ausgewählte Komponenten zu Komponenten verschieben und klicken Sie auf Weiter.

    Der Assistent zum Zuordnen der aktuellen Komponente zu einem anderen Include wird geöffnet.

    • Mit "Zuordnen" ordnen Sie die Komponente dem neuen Include zu und die ursprüngliche Zuordnung wird beibehalten.

    • Mit "Verschieben" entfernen Sie die Zuordnung der Komponente zum ursprünglichen Include und die Komponente wird nur dem neuen Include zugeordnet.

  7. Markieren Sie das gewünschte Include und klicken Sie auf Weiter.

  8. Legen Sie den Sichtbarkeitszeitraum und die Priorität fest und klicken Sie auf Fertigstellen.

    Die Komponente wird dem neuen Ziel-Include zugeordnet und die Zuordnung zum vorherigen Include wird aufgehoben.

Komponentenpriorität festlegen

Falls einem Include mehrere Komponenten mit überlappenden Sichtbarkeitszeiträumen zugeordnet sind, bestimmt die Priorität die Reihenfolge, in der die online gesetzten Komponenten tatsächlich angezeigt wird. Um die Priorität von zugeordneten Komponenten festzulegen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Komponenten.

    Die dem Include zugeordneten Komponenten werden in einer Listenansicht gezeigt.

  4. Benutzen Sie in der Komponenten-Liste die Pfeile in der Spalte Priorität, um die Priorität der Komponenten zu ändern.

    Durch Klick auf eine Pfeil-Schaltfläche wird die entsprechende Komponente nach oben oder unten verschoben, wodurch unmittelbar die Anzeigepriorität geändert wird.

Komponenten-Sichtbarkeit festlegen

Die Sichtbarkeit bezieht sich auf die Verfügbarkeit der Komponente in einem bestimmten Kontext, d.h. innerhalb eines Includes. Die Sichtbarkeit legt die Gültigkeitsdauer einer Include-Komponenten-Zuordnung fest. Falls einem Include mehrere Komponenten mit überlappenden Sichtbarkeitszeiträumen zugeordnet sind, bestimmt die Priorität die Reihenfolge, in der die online gesetzten Komponenten tatsächlich angezeigt wird. Damit Komponenten in der Storefront sichtbar werden, müssen die Publikationsdauer sowie der Sichtbarkeitszeitraum und die Priorität übereinstimmen.

Um die Sichtbarkeit von zugeordneten Komponenten festzulegen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Komponenten.

    Die dem Include zugeordneten Komponenten werden in einer Listenansicht gezeigt.

  4. Klicken Sie auf den Sichtbarkeitszeitraum der entsprechenden Komponente.

    Das Dialogfeld Priorität und Sichtbarkeit wird geöffnet.

  5. Bearbeiten Sie den Sichtbarkeitszeitraum und die Priorität wie gewünscht.

  6. Klicken Sie auf OK, um Änderungen zu speichern.

    Anderenfalls klicken Sie auf Abbrechen, um die Änderungen zu verwerfen.

Zuordnungen von Komponenten zu Includes aufheben

Um die Zuordnung einer Komponente zu einem Include aufzuheben:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Komponenten.

    Die dem Include zugeordneten Komponenten werden in einer Listenansicht gezeigt.

  4. Markieren Sie Zuordnung, die Sie aufheben wollen.

    Markieren Sie das Kontrollkästchen der entsprechenden Komponentenzuordnung.

  5. Klicken Sie auf Aufheben und bestätigen Sie die Operation.

    Die Zuordnung der Komponente zum aktuellen Include wird aufgehoben.

Include-Label verwalten

Label ermöglichen die Definition von Gruppen abhängiger Objekte, um dann bestimmte Aktionen mit diesen Objekten auszuführen. Informationen zum Erzeugen von Labeln, siehe Label anlegen. Für weitere Informationen zur Ausführung label-basierter Aktionen, siehe Aktionen für Objekte mit Labeln durchführen.

Label zu Includes hinzufügen

Um einem Include ein Label zuzuordnen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Label.

    Eine Liste der Label, die diesem Include zugeordnet sind, wird angezeigt.

  4. Klicken Sie auf Neu, um eine neue Label-Zuordnung zu definieren.

    Eine Liste der verfügbaren Label wird angezeigt.

  5. Markieren Sie das gewünschte Label und klicken Sie auf OK.

    Das ausgewählte Label wird dem aktuellen Include zugeordnet.

Label von Includes entfernen

Um die Zuordnung eines Labels zu einem Include aufzuheben:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Namen des Includes, das Sie bearbeiten wollen.

    Das Register Content der Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Label.

    Eine Liste der Label, die diesem Include zugeordnet sind, wird angezeigt.

  4. Markieren Sie das Kontrollkästchen der Labelzuordnung, die Sie aufheben wollen.

  5. Klicken Sie auf Aufheben und bestätigen Sie die Operation.

    Die Zuordnung des Labels zum aktuellen Include wird entfernt.

Includes in Stapelprozessen bearbeiten

Bestimmte Operationen in der Include-Verwaltung können mittels Stapelprozessen mit mehreren Includes gleichzeitig ausgeführt werden. Operationen, die als Stapelprozesse ausgeführt werden können, sind:

  • Eine Komponente zu ausgewählten Includes zuordnen

  • Ausgewählten Includes ein Label zuordnen

  • Markierte Includes löschen

Weitere Informationen zum Bearbeiten von Includes per Stapelprozess, siehe Objekte in Stapelprozessen bearbeiten.

Includes publizieren

In einer Umgebung der Datenreplizierung müssen Includes, die im Bearbeitungssystem erzeugt werden, im Zielsystem veröffentlicht werden. Dies geschieht entweder automatisch im Kontext von regulärer Datenreplizierung (siehe Verwaltung von Replikationsaufgaben) oder manuell, wie unten beschrieben.

Um Includes manuell zu publizieren, gehen Sie wie folgt vor:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Content | Includes.

    Die Include-Listenansicht wird angezeigt.

  2. Markieren Sie die Kontrollkästchen der zu publizierenden Includes in der Listenansicht.

    Alternativ können Sie eine Suche ausführen und auf Alles auswählen klicken, um das Suchergebnis auszuwählen.

  3. Klicken Sie Publizieren im Zielsystem.

    Die Liste konfigurierter Zielsysteme wird angezeigt.

  4. Wählen Sie das gewünschte Zielsystem und die Lösung für einen Sperrkonflikt.

    Für Informationen zu den Sperrkonflikt-Lösungen, siehe Objektsperren.

  5. Klicken Sie auf Fertigstellen.

    Die gewählten Includes werden an das gewählte Zielsystem repliziert.
