Produkte finden

Kataloge durchsuchen

Eine Möglichkeit, Produkte aufzufinden, ist das Durchsuchen von Katalogen. Produkte sind Katalogkategorien zugewiesen.

Hinweis:

Wenn Sie einen Katalog nach einem Produkt durchsuchen, sehen Sie nur die Produkte, die vorher dem Katalog zugewiesen wurden. Um Produkte zu finden, die keinem Katalog zugewiesen sind, benutzen Sie Einfache Suche, Erweiterte Suche oder Klassifikationssuche.

Um Kataloge nach Produkten zu durchsuchen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf einen Katalog und durchsuchen Sie seine Kategorien nach Produkten.

Einfache Suche

In der einfachen Suche können Sie mit einem einzelnen Suchwort nach einem Produkt suchen. Hierbei werden Produktnamen und -IDs in jedem Katalog nach dem Suchwort durchsucht. Die Ergebnisse werden in einer Liste angezeigt.

Standardmäßig wird die einfache Suche angezeigt. Wenn Sie zuvor die erweiterte Suche verwendet haben, klicken Sie auf Einfache Suche.

Tip:

Um eine Liste aller Produkte im aktuellen Produktbestand zu erhalten, lassen Sie das Suchfeld frei und klicken auf Suchen.

Einen Überblick über gängige Suchoptionen finden Sie unter Allgemeine Suchoptionen.

Erweiterte Suche

Mit der erweiterten Suche können Sie durch die Angabe exakter Suchkriterien gezielter suchen. Sie können beispielsweise nach einem bestimmten Produktnamen, einer Produkt-ID, einem Herstellernamen oder einer Preisspanne suchen.

Um die erweiterte Suche zu verwenden:

  1. Klicken Sie auf Erweiterte Suche.

    Standardmäßig wird die einfache Suche angezeigt.

  2. Geben Sie Ihre Suchkriterien in den entsprechenden Feldern an.

    Die Suchkriterien sind wie folgt gruppiert:

    • allgemeine Produktattribute: ID, Typ, Hersteller, Preis u.a.

      Abbildung 24: Erweiterte Produktsuche: Allgemeine Produktattribute

    • Produktstatus: Online-Status, Verfügbarkeit, Genehmigungsstatus u.a.

      Abbildung 25: Erweiterte Produktsuche: Produktstatus

    • Produktlebenszyklus: Letzte Bestellmöglichkeit und End-of-Life-Datum

      Abbildung 26: Erweiterte Produktsuche: Produktlebenszyklus

    • benutzerdefinierte Attribute

      Abbildung 27: Erweiterte Produktsuche: Benutzerdefinierte Attribute

    • Seiten/Includes: Seiten- und Include-Zuordnungen

      Abbildung 28: Erweiterte Produktsuche: Seiten/Includes

    • Suchergebnisanzeige: Festlegung der Suchergebnis-Sortierreihenfolge

      Abbildung 29: Erweiterte Produktsuche: Suchergebnisanzeige

  3. Klicken Sie auf Suchen.

    Die Suchergebnisse werden in einer Liste angezeigt.

Einen Überblick über gängige Suchoptionen finden Sie unter Allgemeine Suchoptionen.

Klassifikationssuche

Die Klassifikationssuche ist eine besondere Form der erweiterten Suche. Sie dient in erster Linie dazu, mit bestimmten Klassifikationsattributen als Suchkriterien zu suchen, siehe Klassifikationsattribute.

Um die Klassifikationssuche zu verwenden:

  1. Klicken Sie auf Erweiterte Suche.

  2. Klicken Sie auf Klassifikationssuche.

  3. Wählen Sie einen Klassifikationskatalog aus.

    Wählen Sie den gewünschten Klassifikationskatalog aus der Dropdown-Liste aus und klicken Sie auf Übernehmen.

  4. Klicken Sie im Abschnitt Klassifikationsattribute auf Hinzufügen.

    Die Klassifikationskategorien werden angezeigt.

  5. Wählen Sie die Klassifikationskategorie aus, der das gewünschte Klassifikationsattribut zugeordnet ist.

    Sie können entweder durch die Klassifikationsbaumansicht navigieren, die gewünschte Kategorie markieren und auf Übernehmen klicken, oder Sie können direkt die entsprechende Klassifikationsnummer angeben und auf OK klicken. Damit werden die Attribute, die der gewählten Kategorie zugeordnet sind, unter dem Klassifikationsbaum angezeigt.

  6. Wählen Sie das gewünschte Klassifikationsattribut aus.

    Markieren Sie dazu das entsprechende Kontrollkästchen, und klicken Sie dann im Abschnitt Attribute auf OK. Die Klassifikationssuche wird wieder angezeigt.

  7. Geben Sie, falls erforderlich, weitere Suchkriterien an.

    Wählen Sie aus, ob Sie nach den Produkten suchen wollen, für die das ausgewählte Attribut definiert ist, oder nach den Produkten, für das nicht der Fall ist. Sie können darüber hinaus die Suche auf die aktuelle Klassifikationskategorie beschränken und die Sortierreihenfolge des Suchergebnisses bestimmen.

  8. Klicken Sie auf Suchen.

    Die Suchergebnisse werden in einer Liste angezeigt.

Um einen neues Produkt anzulegen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Produkte.

    Die Produkt-Listenansicht wird angezeigt.

  2. Klicken Sie Neu.

    Das allgemeine Register des Dialogs Standardprodukt wird angezeigt.

  3. Geben Sie im allgemeinen Register die Details wie erforderlich an.

    Erforderliche Angaben sind Name und ID.

  4. Klicken Sie Übernehmen.

    Das neue Produkt wird angelegt und die anderen Register aktiviert.

  5. Tragen Sie unter jedem Register die entsprechenden Angaben zum neuen Produkt ein.

    Weitere Informationen zu den Produktbearbeitungsoptionen finden Sie in den Abschnitten unten.

Produkte sperren

Bevor Sie ein Produkt bearbeiten, müssen Sie es sperren. Die Sperre verhindert, dass das Produkt zur gleichen Zeit von anderen Benutzern bearbeitet werden kann. Die Produktsperre kann manuell gelöst werden. Nach Ablauf eines festgelegten Intervalls wird die Produktsperre automatisch aufgehoben. Siehe Produktsperren verwalten für Informationen dazu, wie man das Intervall für die Produktsperre konfiguriert.

Produkte können auf verschiedene Weise gesperrt werden: entweder direkt über der Produktliste oder über die Produkt-Detailansicht.

Produkte in der Listenansicht sperren

Um Produkte über die Produkt-Listenansicht zu sperren:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Produkte.

    Die Produkt-Listenansicht wird angezeigt.

  2. Klicken Sie auf das Symbol in der Zeile des/der Produkts/e, das/die Sie bearbeiten wollen.

    Wenn der Dialog Produkt geöffnet wird, werden automatisch alle Bearbeitungsoptionen aktiviert. Falls das Produkt gegenwärtig von einem anderen Benutzer bearbeitet wird, erscheint eine Fehlermeldung.

Hinweis:

Klicken Sie auf das Symbol um die Produktsperre aufzuheben, wenn Sie mit der Bearbeitung fertig sind.

Produkte in der Detailansicht sperren

Um Produkte über die Produkt-Detailansicht zu sperren:

  1. Suchen Sie das Produkt, das Sie bearbeiten wollen.

    Für weitere Informationen, siehe Produkte finden.

  2. Klicken Sie auf den Namen des Produkts, das Sie bearbeiten wollen.

    Das allgemeine Register des Dialogs Standardprodukt wird angezeigt.

  3. Klicken Sie auf das Symbol oben im Dialog oder auf Sperren unten im Dialog.

    Dadurch werden Schaltflächen wie Speichern oder Übernehmen aktiviert, die zum Speichern von Produktdaten verwendet werden. Falls das Produkt gegenwärtig von einem anderen Benutzer bearbeitet wird, erscheint eine Fehlermeldung.

Hinweis:

Klicken Sie auf das Symbol oder auf Entsperren, um die Produktsperre zu lösen, wenn Sie mit Ihrer Bearbeitung fertig sind.

Produkt-Onlinestatus einstellen

Produkte können online oder offline sein. Ein Offline-Produkt ist außer für Katalogverwalter für keinen Benutzer sichtbar. Der Produktstatus kann entweder direkt über die Produkt-Listenansicht oder über den Dialog Produkt gesetzt werden (letzteres ermöglicht es, eine Zeit für online/offline festzulegen).

Hinweis:

Der Status eines Variationsprodukts kann sich auch ändern, wenn Änderungen an einem verbundenen Produkt vorgenommen werden. Weitere Informationen über Variationsprodukte, siehe Produktvariationen verwalten.

Onlinestatus in der Listenansicht einstellen

Um ein Produkt über die Produkt-Listenansicht online/offline zu setzen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Produkte.

    Die Produkt-Listenansicht wird angezeigt.

  2. Klicken Sie auf das Symbol in der Zeile des/der Produkts/e, das/die Sie offline setzen wollen.

    Der Onlinestatus wird sofort auf offline gesetzt ohne Zeitrestriktionen (Gültigkeit = für immer).

Tip:

Um das Produkt wieder online zu setzen, klicken Sie Symbol.

Onlinestatus in der Detailansicht einstellen

Um den Produktstatus mit der Gültigkeit in der Produkt-Detailansicht einzustellen:

  1. Suchen Sie das Produkt, das Sie online oder offline setzen wollen.

    Für weitere Informationen, siehe Produkte finden.

  2. Klicken Sie auf den Namen des Produkts.

    Das allgemeine Register des Dialogs Standardprodukt wird angezeigt.

  3. Öffnen Sie das Register Online/Offline.

  4. Setzen Sie im Feld Status "Online" oder "Offline" als Wert.

  5. Geben Sie die Gültigkeit des ausgewählten Status an.

    Die Gültigkeit legt fest, wann der gewählte Status auf das Produkt angewendet wird. Wählen Sie eine der zwei Optionen "Beschränkter Zeitraum" oder "Immer". Wenn Sie einen beschränkten Zeitraum wählen, müssen Sie eine Start- und eine Endzeit mit genauem Datum und genauer Zeit in den entsprechenden Feldern angeben.

Produktgenehmigungen verwalten

Wie bereits erwähnt in Produktgenehmigung erwähnt, betrifft die Produktgenehmigung zwei Zugriffsrechte bzw. Rollen: Produktgenehmiger und Katalog- und Produktverwalter. Produktgenehmiger sind verantwortlich für das Prüfen und Genehmigen der Produktdaten, Katalog- und Produktverwalter bearbeiten die Daten und fordern die Genehmigung an. Beide Rollen können (je nach aktuellem Genehmigungsstatus) Produkte in den Status "Bearbeiten" setzen, falls sie fehlende oder falsche Daten feststellen.

Tip:

Sie können die Statistiken der Produktübersicht oder eine Produkt-Vollständigkeitsprüfung verwenden, um fehlende Daten zu identifizieren. Für weitere Informationen, siehe Produkt-Vollständigkeit.

Für das Arbeiten mit Produktgenehmigungsstatus muss die Produktgenehmigung zunächst aktiviert werden. Für weitere Informationen, siehe Produktgenehmigung aktivieren.

Der Produktgenehmigungsstatus kann auf verschiedene Weise geändert werden: direkt über die Produkt-Listenansicht oder über die Produkt-Detailansicht. Die Schritte unterscheiden sich.

Änderung von Produktdaten anfordern

Je nach aktuellem Genehmigungsstatus können Produktgenehmiger und Katalog- und Produktverwalter eine Änderung der Produktdaten anfordern, falls sie fehlende oder falsche Daten feststellen, indem sie den Genehmigungsstatus auf "Bearbeiten" setzen.

Änderung von Produktdaten in der Detailansicht anfordern

Um den Genehmigungsstatus "Bearbeiten" für ein einzelnes Produkt in der Produkt-Detailansicht einzustellen:

  1. Suchen Sie das Produkt, das geändert werden soll.

    Tip:

    Sie können die Produkt-Vollständigkeitsprüfung verwenden, um das Produkt zu identifizieren. Für weitere Informationen, siehe Produkt-Vollständigkeitsprüfung.

  2. Klicken Sie auf den Namen des Produkts, das Sie bearbeiten wollen.

    Das Register Allgemein der Produkt-Detailansicht wird geöffnet, das den aktuellen Genehmigungsstatus rechts oben anzeigt.

  3. Klicken Sie auf Zu bearbeiten.

    Ein Dialog zur Angabe eines Grundes wird geöffnet.

  4. Geben Sie einen Grund an und klicken Sie auf OK.

    Dadurch erhält das aktuelle Produkt den Genehmigungsstatus "Bearbeiten".

Änderung von Produktdaten per Stapelverarbeitungsprozess anfordern

Um den Genehmigungsstatus "Bearbeiten" in der Produkt-Listenansicht für mehrere Produkte per Stapelprozess einzustellen, siehe Produkte in Stapelprozessen bearbeiten:

  1. Suchen Sie die Produkte, die geändert werden sollen.

    Markieren Sie die Kontrollkästchen der gewünschten Produkte in der Listenansicht, und klicken Sie auf Auswahl bearbeiten. Alternativ können Sie eine erweiterte Suche durchführen (siehe Erweiterte Suche) und auf Alles bearbeiten klicken, um das gesamte Suchergebnis zu bearbeiten.

    Tip:

    Sie können die Statistiken der Produktübersicht verwenden, um die Produkte zu identifizieren. Für weitere Informationen siehe Produktübersicht.

  2. Wählen Sie die Aufgabe Produktgenehmigungsstatus aktualisieren und klicken Sie auf Weiter.

    Der Assistent zum Aktualisieren des Produktgenehmigungsstatus wird gestartet.

  3. Wählen Sie den aktuellen Genehmigungsstatus oder Vollständigkeitsstatus aus, legen Sie "Bearbeiten" als neuen Genehmigungsstatus fest, und klicken Sie auf Weiter.

    Der Bestätigungsdialog wird angezeigt.

  4. Prüfen Sie die anzuwendenden Änderungen und klicken Sie auf Fertigstellen.

    Dadurch erhalten die ausgewählten Produkte den Genehmigungsstatus "Bearbeiten".

Genehmigung von Produktdaten anfordern

Nach erfolgter Änderung der Produktdaten können Katalog- und Produktverwalter die Freigabe der Daten anfordern, indem sie den Genehmigungsstatus auf "Prüfen" setzen.

Genehmigung von Produktdaten über die Detailansicht anfordern

Um den Genehmigungsstatus "Prüfen" für ein einzelnes Produkt in der Produkt-Detailansicht einzustellen:

  1. Suchen Sie das Produkt, das geprüft werden soll.

    Tip:

    Sie können die Suche verwenden, um das Produkt zu identifizieren.

  2. Klicken Sie auf den Namen des Produkts, das Sie bearbeiten wollen.

    Das Register Allgemein der Produkt-Detailansicht wird geöffnet, das den aktuellen Genehmigungsstatus rechts oben anzeigt.

  3. Klicken Sie auf Zu prüfen.

    Dadurch erhält das aktuelle Produkt den Genehmigungsstatus "Prüfen".

Genehmigung von Produktdaten per Stapelverarbeitungsprozess anfordern

Um den Genehmigungsstatus "Prüfen" in der Produkt-Listenansicht für mehrere Produkte per Stapelprozess einzustellen, siehe Produkte in Stapelprozessen bearbeiten:

  1. Suchen Sie die Produkte, die geprüft werden sollen.

    Markieren Sie die Kontrollkästchen der gewünschten Produkte in der Listenansicht, und klicken Sie auf Auswahl bearbeiten. Alternativ können Sie eine erweiterte Suche durchführen (siehe Erweiterte Suche) und auf Alles bearbeiten klicken, um das gesamte Suchergebnis zu bearbeiten.

  2. Wählen Sie die Aufgabe Produktgenehmigungsstatus aktualisieren und klicken Sie auf Weiter.

    Der Assistent zum Aktualisieren des Produktgenehmigungsstatus wird gestartet.

  3. Wählen Sie den aktuellen Genehmigungsstatus oder Vollständigkeitsstatus aus, legen Sie "Prüfen" als neuen Genehmigungsstatus fest, und klicken Sie auf Weiter.

    Der Bestätigungsdialog wird angezeigt.

  4. Prüfen Sie die anzuwendenden Änderungen und klicken Sie auf Fertigstellen.

    Dadurch erhalten die ausgewählten Produkte den Genehmigungsstatus "Prüfen".

Produkte bearbeiten

Um ein bestehendes Produkt zu bearbeiten:

  1. Suchen Sie das Produkt, das Sie bearbeiten wollen.

    Für weitere Informationen, siehe Produkte finden.

  2. Klicken Sie auf den Namen des Produkts, das Sie bearbeiten wollen.

    Das allgemeine Register des Dialogs Standardprodukt wird angezeigt.

  3. Sperren Sie das Produkt.

    Für weitere Informationen, siehe Produkte sperren.

  4. Führen Sie unter jedem Register die erforderlichen Änderungen an den Einträgen durch.

    Hinweis:

    Wird ein freigegebenes Master-Produkt bearbeitet, erscheint zusätzlich das Register Änderungen, das die vorgenommen Änderungen gegenüber dem Original anzeigt und es erlaubt, sie zu löschen.

Produkte werden kopiert

Sie können von einem Produkt eine Kopie im gleichen Produktbestand erzeugen. Dem Namen der Kopie ist "Copy of" vorangestellt (in lokalisierten Systemen werden entsprechende Übersetzungen verwendet). Das System generiert eine UUID als SKU für das neue Produkt.

Das neue Produkt ist offline. Ist das Originalprodukt von einem anderen Produkt durch Syndizierung abgeleitet, wird die Referenz zu diesem Ausgangsprodukt beibehalten.

Folgende Attribute werden vom Originalprodukt auf die Kopie übertragen:

  • Allgemeine Attribute (Beschreibung, Preise usw.) und benutzerdefinierte Attribute (einschließlich Verfügbarkeit)

  • Anlagen, Produktverweise, Listenpreise und Staffelpreise

  • Content-Beziehungen

  • Zuordnungen zu Kategorien und Klassifikationen

Ist das Originalprodukt ein Variationsprodukt, wird die Kopie nicht mit dem Variations-Stammprodukt verbunden. Ist das Originalprodukt dagegen ein Variations-Stammprodukt, werden alle dem Original zugewiesenen Variationsprodukte kopiert und auch der Kopie zugewiesen.

Hinweis:

Beim Kopieren eines Variations-Stammprodukts werden automatisch Kopien aller dazugehörigen Variationsprodukte erzeugt.

Ist das Originalprodukt ein gebündeltes Produkt (d.h. Teil eines Produktbündels) oder Teil eines Retail-Sets, wird die Kopie nicht mit in das Produktbündel oder das Retail-Set eingefügt. Ist das Originalprodukt dagegen ein Produktbündel oder Retail-Set, wird auch die Kopie ein Produktbündel bzw. Retail-Set. Die Produkte, die zum Original-Produktbündel oder -Retail-Set gehören, werden auch Teil der jeweiligen Kopien.

Um ein Produkt zu kopieren:

  1. Suchen Sie das Produkt, das Sie kopieren wollen.

    Für weitere Informationen, siehe Produkte finden.

  2. Klicken Sie auf den Namen des Produkts.

    Das allgemeine Register des Dialogs Standardprodukt wird angezeigt.

  3. Klicken Sie auf Kopieren.

    Das neue Produkt wird in der Produkt-Listenansicht angezeigt.

Hinweis:

Wenn Sie ein freigegebenes (vom übergeordneten Repository) Variationsprodukt offline setzen ist der Master davon nicht betroffen. Ebenso wird kein neues Standard-Variationsprodukt festgelegt.

Kategorie-/Produktverweise verwalten

Kategorie- und Produktverweise verbinden das aktuelle Produkt mit bestimmten Katalogkategorien oder anderen Produkten im Produktbestand. Kategorie- und Produktverweise können verschiedene Beziehungen beschreiben und für verschiedene Zwecke verwendet werden. Diese Unterschiede werden durch die verschiedenen Verweistypen deutlich. Weitere Informationen zu Verweistypen, siehe Kategorie- und Produktverweise.

Kategorieverweise und Produktverweise werden mit dem Register Verweise verwaltet. Dieser Tab listet sowohl "ausgehende" als auch "eingehende" Links auf. Ausgehende Links definieren ein anderes Produkt oder eine Kategorie als das "Verweisziel" ("<Verweistyp> für"), wohingegen eingehende Links legen das aktuelle Produkt als "Verweisziel" eines anderen Produktes oder einer Kategorie fest ("ist <Verweistyp> von").

Die Verweistypauswahl ist in zwei Spalten unterteilt: die linke zeigt die ausgehenden Links, die rechte zeigt die eingehenden Links. Verweistyp auswählen zeigt die dazugehörige Linkliste, die die tatsächlichen Links anzeigt. Multiple Verweislisten sind nach Verweistypen gruppiert, wo ausgehende Links über den eingehenden Links desselben Typs stehen.

Ausgehende Kategorie-/Produktverweise können manuell sortiert werden.

  3. Markieren Sie im Register Allgemein das Kontrollkästchen Geschenkverpackung.

  4. Öffnen Sie das Register Verweise.

    Es werden bereits definierte Kategorie- und Produktverweise angezeigt.

  5. Klicken Sie im Abschnitt Produktverweise auf Neu.

    Der Dialog Verweistyp wird angezeigt.

  6. Markieren Sie den gewünschten Verweistyp (Geschenkverpackung|Geschenknachricht) und klicken Sie auf Weiter.

    Die Produkt-Listenansicht wird angezeigt.

  7. Ermitteln Sie das Geschenk-Produkt, welches als Verweisziel gelten soll, wählen Sie die entsprechenden Kontrollkästchen und klicken Sie Fertigstellen.

    Das gewählte Geschenk-Produkt wird nun im Abschnitt Produktverweise im Register Verweise des Standard-Produktdialogs angezeigt.

    Hinweis:

    Wenn in den Warenkorb-Einstellungen der Applikation aktiviert (siehe Warenkorb-Einstellungen verwalten), wird die gewählte Geschenkoption in der Storefront angeboten.

Um Label für ein einzelnes Produkt zu verwalten:

  3. Geben Sie im allgemeinen Register das Label an.

    Geben Sie die ersten Buchstaben im Feld Label ein. Existierende Label mit denselben Anfangsbuchstaben werden dadurch angezeigt. Sie können eines der angezeigten Label auswählen oder den Namen eines neuen Labels zu Ende eingeben; durch Enter bestätigen Sie die Eingabe.

  4. Um die Zuordnung eines Labels aufzuheben, klicken Sie beim Label.

Klassifikationen verwalten

Produkte können in Klassifikationskataloge eingeordnet sein. Intershop Commerce Management bietet direkte Unterstützung für Klassifikationskataloge nach eCl@ss- und UN/SPSC-Standard. Vertriebsorganisationen können zusätzlich benutzerdefinierte Klassifikationskataloge anlegen.

Um Klassifikationszuordnungen für Produkte zu verwalten:

  1. Suchen Sie das Produkt, dessen Klassifikationszuordnungen Sie verwalten wollen.

    Für weitere Informationen, siehe Produkte finden.

  2. Klicken Sie auf den Namen des Produkts.

    Das allgemeine Register des Dialogs Standardprodukt wird angezeigt.

  3. Öffnen Sie das Register Klassifikationen.

    Die Zuordnungen zu Kategorien von Standardklassifikationen (eCl@ss·und·UN/SPSC) und allen angelegten benutzerdefinierten Klassifikationen werden angezeigt.

  4. Um eine Kategoriezuordnung anzulegen, klicken Sie auf Zuordnen.

    Klicken Sie auf Aufheben, um eine Zuordnung aufzuheben. Klicken Sie auf die Klassifikationskategorie, um die Einstellungen für die Zuordnung zu bearbeiten.

  5. Geben Sie einen gültigen Klassifikations-Code an. Alternativ können Sie eine Kategorie aus der Dropdown-Liste auswählen und auf Auswählen klicken. Dadurch werden alle Unterkategorien der aktuellen Kategorie angezeigt. Öffnen Sie weiter alle Unterkategorien bis zur untersten Ebene.

    Eine Liste der Klassifikationsattribute wird angezeigt.

  6. Geben Sie Werte für die Klassifikationsattribute an.

    Hinweis:

    Klassifikationsattribute definieren bestimmte Attribute, die Produkte in einer bestimmten Kategorie haben müssen. Nur solche Klassifikationsattribute, für die Werte festgelegt sind, werden auf das Produkt übertragen.

Preislisten verwalten

Preislisten ermöglichen die Festlegung von gestaffelten, zielgruppenspezifischen Preisen für bestimmte Produkte. Darüber hinaus ermöglichen die Standardpreisliste und die Einstandspreisliste die einfache Verwaltung von Listenpreisen und Einstandspreisen für Produkte.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Preislisten anlegen

Um eine Spezialpreisliste zu definieren:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Preislisten.

    Die Standardpreisliste, die Einstandspreisliste und die Spezialpreislisten, die für die Channel-Produkte definiert sind, werden angezeigt.

  2. Klicken Sie auf Neu, um eine neue Preisliste anzulegen.

    Dadurch wird der neue Preislisten-Dialog geöffnet.

  3. Definieren Sie Namen und ID für die Preisliste, geben Gültigkeitsdaten an, falls erforderlich, und klicken Sie auf Übernehmen.

    Werden keine Gültigkeitsdaten angegeben, gilt die Preisliste für einen unbegrenzten Zeitraum.

    Tip:

    Neue Preislisten sind standardmäßig aktiviert.

  4. Öffnen Sie das Register Preise, um alle Produkte anzuzeigen, die Preise für diese Preisliste definiert haben.

    Für weitere Informationen zum Definieren von Preisen in Spezialpreislisten, siehe Staffelpreise verwalten.

  5. Öffnen Sie das Register Zielgruppe, um festzulegen, für welche Kundensegmente und Kunden diese Preislisten verfügbar sein sollen.

    Standardmäßig ist eine Preisliste für alle Kunden verfügbar.

    Hinweis:

    Um Kundensegmente als Zielgruppe zuordnen zu können, muss mindestens ein Segmentierungsdienst für den Channel konfiguriert sein, siehe Kundensegmente.

Gültigkeit von Preislisten ändern

Um den Gültigkeitszeitraum einer Preisliste zu ändern:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Preislisten.

    Die Standardpreisliste, die Einstandspreisliste und die Spezialpreislisten, die für die Channel-Produkte definiert sind, werden angezeigt.

  2. Klicken Sie auf den Namen der Preisliste, die Sie bearbeiten wollen.

    Die Detailansicht der Preisliste wird geöffnet.

  3. Wechseln sie zum Register Allgemein.

    Die allgemeinen Preisdaten werden angezeigt.

  4. Bearbeiten Sie die Details wie erforderlich.

    Sie können Start- und Endpunkt des Gültigkeitszeitraumes angeben als auch die Preisliste deaktivieren.

  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Kunden zu Preislisten zuordnen

Sie können festlegen, für welche Zielgruppe d.h. Kunden oder Kundensegmente, die Preisliste gilt. Um dies zu tun:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Preislisten.

    Die Standardpreisliste, die Einstandspreisliste und die Spezialpreislisten, die für die Channel-Produkte definiert sind, werden angezeigt.

  2. Klicken Sie auf den Namen der Preisliste, die Sie bearbeiten wollen.

    Die Detailansicht der Preisliste wird geöffnet.

  3. Öffnen Sie das Register Zielgruppe.

    Dieses Register ist in zwei Abschnitte unterteilt:

    • Kundensegmente: Zeigt alle Kundensegmente, denen die Preisliste zugeordnet ist.

      Hinweis:

      Um Kundensegmente als Zielgruppe zuordnen zu können, muss mindestens ein Segmentierungsdienst für den Channel konfiguriert sein, siehe Kundensegmente.

    • Kunden: Zeigt alle Kunden, denen die Preisliste zugeordnet ist.

  4. Klicken Sie im entsprechenden Abschnitt (Kundensegmente/Kunden) auf Zuordnen.

    Die verfügbaren Kundensegmente bzw. Kunden werden angezeigt.

  5. Wählen Sie das gewünschte Kundensegment oder den Kunden und klicken Sie auf Ok.

    Das gewählte Kundensegment oder Kunde ist der Preisliste zugeordnet. Sie werden auf das Zielgruppenregister zurückgeleitet, welches die aktualisierte Kundensegment-/Kundenliste anzeigt.

Tip:

Sie können Preislisten zu Kunden oder Kundensegmenten zuordnen, auch während Kunden- oder Kundensegmentdetails bearbeitet werden, siehe Preisliste für Kunden verwalten oder Preislisten für Kundensegmente verwalten.

Zuordnung von Kunden zu Preislisten aufheben

Um die Zuordnung einer Preisliste zu einem Kundensegment oder Kunden aufzuheben:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Preislisten.

    Die Standardpreisliste, die Einstandspreisliste und die Spezialpreislisten, die für die Channel-Produkte definiert sind, werden angezeigt.

  2. Klicken Sie auf den Namen der Preisliste, die Sie bearbeiten wollen.

    Die Detailansicht der Preisliste wird geöffnet.

  3. Öffnen Sie das Register Zielgruppe.

    Dieses Register ist in zwei Abschnitte unterteilt:

    • Kundensegmente: Zeigt alle Kundensegmente, denen die Preisliste zugeordnet ist.

      Hinweis:

      In diesem Abschnitt werden nur Kundensegmente angezeigt, die von einem aktiven Segmentierungsdienst bereitgestellt wurden. Ist ein bestimmter Dienst deaktiviert, werden alle von ihm bereitgestellten Kundensegmente als Zielgruppen entfernt.

    • Kunden: Zeigt alle Kunden, denen die Preisliste zugeordnet ist.

  4. Wählen Sie die Zuordnung, die Sie löschen möchten.

    Markieren Sie die Kontrollkästchen der Kundensegmente/Kunden im entsprechenden Abschnitt (Kundensegment/Kunden), deren Zuordnung zur Preisliste Sie entfernen wollen.

  5. Klicken Sie auf Aufheben und dann auf Ok, um den Vorgang zu bestätigen.

    Die Zuordnung der ausgewählten Kundensegmente/Kunden zur Preisliste werden gelöscht.

Preislisten publizieren

In Datenreplizierungs-Umgebungen können Katalogverwalter Preislisten aus dem Bearbeitungssystem in das Zielsystem detailliert veröffentlichen. Dazu gehören zwei Schritte:

  • die (leere) Preisliste publizieren und

  • einzelne Preise dieser Preisliste publizieren.

Um eine Preisliste in das Zielsystem zu publizieren:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Preislisten.

    Die Standardpreisliste, die Einstandspreisliste und die Spezialpreislisten, die für die Channel-Produkte definiert sind, werden angezeigt.

  2. Wählen Sie die gewünschte Preisliste aus.

    Markieren Sie das Kontrollkästchen der Preisliste, die Sie im Zielsystem publizieren wollen.

  3. Klicken Sie Publizieren im Zielsystem.

    Die Liste konfigurierter Zielsysteme wird angezeigt.

  4. Wählen Sie ein Zielsystem aus und klicken Sie auf Fertigstellen.

    Die Preisliste wird im gewählten Zielsystem repliziert.

Hinweis:

Das Publizieren einer Preisliste im Zielsystem erfolgt nicht mit den bestimmten Preisen, wie sie im Bearbeitungssystem definiert wurden.

Um einzelne Preise einer Preisliste im Zielsystem zu publizieren:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Preislisten.

    Die Standardpreisliste, die Einstandspreisliste und die Spezialpreislisten, die für die Channel-Produkte definiert sind, werden angezeigt.

  2. Klicken Sie auf den Namen der gewünschten Preisliste.

    Die Detailansicht der Preisliste wird geöffnet.

  3. Öffnen Sie das Register Preise.

    Es werden alle Spezialpreise angezeigt, die in der aktuellen Preisliste definiert wurden.

  4. Wählen Sie die gewünschten Preise aus.

    Markieren Sie das/die Kontrollkästchen der einzelnen Preise, die sie im Zielsystem publizieren wollen.

  5. Klicken Sie Publizieren im Zielsystem.

    Die Liste konfigurierter Zielsysteme wird angezeigt.

  6. Wählen Sie ein Zielsystem aus und klicken Sie auf Fertigstellen.

    Die ausgewählten Preise werden in die entsprechende Preisliste im Zielsystem repliziert.

Hinweis:

Stellen Sie sicher, dass die Preisliste im Zielsystem publiziert wurde, bevor sie einzelne Preise publizieren.

Preisfindungsstrategie verwalten

Die Preisfindungsstrategie bestimmt, wie das System vorgehen soll, wenn in einer Situation mehr als eine Preisliste angewendet werden kann.

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Preislisten.

    Die Standardpreisliste, die Einstandspreisliste und die Spezialpreislisten, die für die Channel-Produkte definiert sind, werden angezeigt.

  2. Wählen Sie aus der Dropdown-Liste Preisfindung entweder "Günstigster Preis" oder "Preislisten-Rangfolge" aus.

    Weitere Informationen zu diesen Vorgehensweisen, siehe Spezialpreislisten und Staffelpreise.

  3. Wenn Sie "Preislisten-Rangfolge" ausgewählt haben, verwenden Sie die Pfeil-Schaltflächen, um die Rangfolge der Preislisten festzulegen.

    Ihre Änderungen werden sofort wirksam.

Produkt-Listenpreise und Einstandspreise verwalten

Sie können in jeder unterstützten Währung einen Listenpreis und einen Einstandspreis für jedes Produkt festlegen. Um einen Listen- oder Einstandspreis zu definieren:

  1. Suchen Sie das Produkt, für das Sie Listen- oder Einstandspreise verwalten wollen.

    Für weitere Informationen, siehe Produkte finden.

  2. Klicken Sie auf den Namen des Produkts.

    Das allgemeine Register des Dialogs Standardprodukt wird angezeigt.

  3. Öffnen Sie das Register Preise.

    Eine Liste der bereits definierten Listen- und Einstandspreise wird angezeigt.

  4. Wählen Sie die Steuerklasse aus.

    Vier Optionen sind verfügbar: NoTax,·ReducedTax,·FullTax,·oder·Nicht definiert. Diese Einstellung gilt für alle definierten Preise.

  5. Wählen Sie eine Währung aus, und geben Sie einen Preis an.

    Sie können für jede im System aktive Währung einen Listenpreis und einen Einstandspreis festlegen.

Tip:

Alternativ können Listenpreise und Einstandspreise im Preislisten-Manager verwaltet werden. Die Standardpreisliste enthält alle Produkte mit den für sie definierten Listenpreisen. Die Einstandspreisliste hingegen zeigt alle Produkte, für die ein Einstandspreis definiert ist.

Staffelpreise verwalten

Hinweis:

Bevor Sie Produktpreise verwalten, stellen Sie sicher, dass die entsprechenden Preise eingestellt wurden, siehe Preiseinstellungen verwalten. Stellen Sie außerdem sicher, dass mindestens eine Preisliste definiert ist, siehe Preislisten anlegen.

Sie können für jedes Produkt eine Staffelpreis-Definition für jede verfügbare Preisliste festlegen.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Um einen Staffelpreis zu definieren:

  1. Suchen Sie das Produkt, für das Sie Staffelpreise verwalten wollen.

    Für weitere Informationen, siehe Produkte finden.

  2. Klicken Sie auf den Namen des Produkts.

    Das allgemeine Register des Dialogs Standardprodukt wird angezeigt.

  3. Öffnen Sie das Register Preise.

    Eine Liste der bereits definierten Listen- und Einstandspreise wird angezeigt.

  4. Klicken Sie in der Zeile Spezialpreise auf Neu.

    Der Assistent zum Anlegen von Staffelpreisen wird geöffnet.

  5. Wählen Sie den Listenpreis aus, für den der Staffelpreis definiert werden soll, und klicken Sie auf Weiter.

  6. Wählen Sie eine Währung für den Staffelpreis aus und klicken Sie auf Weiter.

  7. Geben Sie, falls erforderlich, Gültig-von- und Gültig-bis-Daten an.

    Werden keine Gültigkeitsdaten angegeben, hängt die Gültigkeit des Staffelpreises von den für die Preisliste definierten Gültig-von- und Gültig-bis-Daten ab.

  8. Geben Sie eine Produktanzahl und einen festen Preis (z. B. 19,99) oder einen Rabatt (z. B. 10%) an, und klicken Sie auf Hinzufügen.

    Wiederholen Sie diesen Schritt falls erforderlich.

  9. Klicken Sie auf Fertigstellen.

    Hinweis:

    Bei der Festlegung von Staffelpreisen bietet Intershop 7 die Möglichkeit, mehrere Staffelpreise für die gleiche Menge eines Produkts in derselben Preisliste zu konfigurieren. Bitte beachten Sie, dass sich diese Preise in einem Zeitabschnitt nicht überschneiden dürfen, da ansonsten die Preissuche nicht korrekt funktioniert.
