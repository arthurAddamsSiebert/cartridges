Dieser Abschnitt beschreibt die Aufgaben zur Verwaltung von Produktbildern. Das Modul ist in vier Register aufgeteilt: Bild-Upload, Bildanzeigen, Bildtypen und Bildsets, welche im Folgenden beschrieben werden.

Tabelle 23: Register des Bilderverwaltungswerkzeuges
Register Beschreibung
Bild-Upload Wird verwendet, um die Bildverzeichnisstruktur zu erstellen und die Bilddateien in das Intershop Commerce Management-System zu laden.
Bildanzeigen Wird verwendet, um die in der Organisation vorhandenen Bildanzeigen zu verwalten.
Bildtypen Wird verwendet, um die in der Organisation vorhandenen Bildtypen zu verwalten.
Bildsets Wird verwendet, um die in der Organisation vorhandenen Bildsets zu verwalten.

Weitere Informationen zum Import/Export von Bild-Metadaten, siehe Bild-Metadaten exportieren/importieren.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext der Organisation oder eines Partner Channels ausgeführt.

Bilddateien hochladen

Bilder sowie anderer statischer Content sind locale-spezifisch. Das bedeutet, dass Sie vor Anlegen der Zielverzeichnisstruktur und Upload der Dateien eine Ziel-Locale bestimmen müssen.

Hinweis:

Die Verzeichnisstruktur und die eigentlichen Bilder können für jede Locale verschieden sein.

Bild-Verzeichnis anlegen

Um ein Upload-Verzeichnis für Bilder zu erzeugen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Wählen Sie die gewünschte Locale aus und klicken Sie auf Übernehmen.

    Die Bildverzeichnisstruktur und die Dateien im aktuellen Verzeichnis werden für die gewählte Locale angezeigt.

  3. Wählen Sie ein Stammverzeichnis für das neue Upload-Verzeichnis aus.

    Durchsuchen Sie die Verzeichnisstruktur und wählen Sie das Stammverzeichnis für das neue Bild-Verzeichnis durch Klick auf den Namen aus. Klicken Sie auf die Plus- bzw. Minussymbole, um die Liste der Unterverzeichnisse auf- bzw. zuzuklappen.

  4. Tragen Sie im Feld Neues Verzeichnis den Namen für das neu anzulegende Verzeichnis ein und klicken Sie Hinzufügen.

    Das neue Verzeichnis wird in der Baumansicht unter dem gewählten Stamm angezeigt.

Bild-Verzeichnis umbenennen

Um ein Bildverzeichnis umzubenennen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Wählen Sie die gewünschte Locale aus und klicken Sie auf Übernehmen.

    Die Bildverzeichnisstruktur und die Dateien im aktuellen Verzeichnis werden für die gewählte Locale angezeigt.

  3. Markieren Sie das Verzeichnis, das Sie umbenennen wollen, und klicken auf Umbenennen.

  4. Spezifizieren Sie den neuen Verzeichnisnamen und klicken Sie OK.

    Das Verzeichnis wird sofort umbenannt.

    Um den zuvor gespeicherten Zustand wiederherzustellen, klicken Sie auf Abbrechen.

Bild-Verzeichnis entfernen

Um ein Bildverzeichnis zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Wählen Sie die gewünschte Locale aus und klicken Sie auf Übernehmen.

    Die Bildverzeichnisstruktur und die Dateien im aktuellen Verzeichnis werden für die gewählte Locale angezeigt.

  3. Markieren Sie das Verzeichnis, das Sie entfernen wollen, und klicken auf Löschen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  4. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Das gewählte Bild-Verzeichnis wird gelöscht.

    Klicken Sie auf Abbrechen, um das Löschen abzubrechen.

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Wählen Sie die gewünschte Locale aus und klicken Sie auf Übernehmen.

    Die Bildverzeichnisstruktur und die Dateien im aktuellen Verzeichnis werden für die gewählte Locale angezeigt.

  3. Wählen Sie ein Stammverzeichnis für die zu ladende Datei aus.

    Durchsuchen Sie die Verzeichnisstruktur und wählen Sie das Stammverzeichnis für das neue Bild-Verzeichnis durch Klick auf den Namen aus. Klicken Sie auf die Plus- bzw. Minussymbole, um die Liste der Unterverzeichnisse auf- bzw. zuzuklappen.

  4. Geben Sie im Feld Datei hochladen den Datei-Pfad an oder klicken Sie auf Datei wählen, um die Datei zu suchen.

  5. Klicken Sie auf Upload, um die Datei hochzuladen.

    Die geladene Datei wird in der Liste der Dateien im aktuellen Upload-Verzeichnis angezeigt.

Bilddateien löschen

Um Bilddateien zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Wählen Sie die gewünschte Locale aus und klicken Sie auf Übernehmen.

    Die Bildverzeichnisstruktur und die Dateien im aktuellen Verzeichnis werden für die gewählte Locale angezeigt.

  3. Wählen Sie das Stammverzeichnis der zu löschenden Datei aus.

    Durchsuchen Sie die Verzeichnisstruktur und wählen Sie das Stammverzeichnis für das neue Bild-Verzeichnis durch Klick auf den Namen aus. Klicken Sie auf die Plus- bzw. Minussymbole, um die Liste der Unterverzeichnisse auf- bzw. zuzuklappen.

  4. Markieren Sie die Dateien, die Sie löschen wollen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  5. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Die ausgewählten Bilder werden entfernt.

    Klicken Sie auf Abbrechen, um das Löschen abzubrechen.

Bildanzeigen verwalten

Katalogverwalter können Bildanzeigen erstellen, bearbeiten oder löschen und Bildtypen zuordnen.

Bildanzeigen erstellen

Um eine neue Bildanzeige anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildanzeigen.

    Alle Bildanzeigen der Organisation werden angezeigt.

  3. Klicken Sie auf Neu.

    Eine leere Detailansicht der Bildanzeige wird geöffnet.

  4. Geben Sie einen Namen und eine ID für die neue Bildanzeige an.

    Optional können Sie das Kontrollkästchen Primäre Bildansicht markieren, um die neue Bildanzeige als Standard-Anzeige in der Organisation festzulegen.

  5. Klicken Sie auf Übernehmen.

    Die neue Bildanzeige wird erstellt und der Abschnitt Verwandte Bildtypen wird angezeigt.

    Sie können nun Bildtypen zuordnen. Klicken Sie auf Zurück zur Liste, um zurück zur Bildanzeigen-Liste zu gelangen.

Bildtypen zuordnen

Um Bildtypen zu einer Bildanzeige zuzuordnen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildanzeigen.

    Alle Bildanzeigen der Organisation werden angezeigt.

  3. Klicken Sie auf den Namen oder die ID der Bildanzeige, die Sie bearbeiten wollen.

    Die Detailansicht der Bildanzeige wird geöffnet.

  4. Klicken Sie im Abschnitt Verwandte Bildtypen auf Zuordnen.

    Eine Liste der verfügbaren Bildtypen wird angezeigt.

  5. Wählen Sie den/die gewünschten Bildtyp(en) aus und klicken Sie OK.

    Die gewählten Bildtypen werden der aktuellen Bildanzeige zugeordnet und Sie werden auf die Detailseite der Bildanzeige zurückgeleitet.

Zuordnung von Bildtypen aufheben

Um die Zuordnung einer Bildanzeige zu einem Bildtypen zu entfernen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildanzeigen.

    Alle Bildanzeigen der Organisation werden angezeigt.

  3. Klicken Sie auf den Namen oder die ID der Bildanzeige, die Sie bearbeiten wollen.

    Die Detailansicht der Bildanzeige wird geöffnet.

  4. Im Abschnitt Verwandte Bildtypen wählen Sie die Kontrollkästchen der Bildtypenzuordnungen, die entfernt werden sollen.

  5. Klicken Sie auf Aufheben und bestätigen Sie die Operation.

    Die Zuordnung des gewählten Bildtypen zur aktuellen Bildanzeige wird entfernt. Klicken Sie auf Zurück zur Liste, um zurück zur Bildanzeigen-Liste zu gelangen.

Bildanzeigen bearbeiten

Um eine bestehende Bildanzeige zu bearbeiten:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildanzeigen.

    Alle Bildanzeigen der Organisation werden angezeigt.

  3. Klicken Sie auf den Namen oder die ID der Bildanzeige, die Sie bearbeiten wollen.

    Die Detailansicht der Bildanzeige wird geöffnet.

  4. Bearbeiten Sie die Details wie erforderlich.

    Bearbeiten Sie die allgemeinen Angaben oder entfernen Sie die Zuordnung zu Bildtypen bzw. fügen Sie eine Zuordnung hinzu. Klicken Sie auf Zurück zur Liste, um zurück zur Bildanzeigen-Liste zu gelangen.

Bildanzeigen löschen

Um Bildanzeigen zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildanzeigen.

    Alle Bildanzeigen der Organisation werden angezeigt.

  3. Markieren Sie die Kontrollkästchen neben den Bildanzeigen, die Sie löschen wollen.

  4. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Die gewählten Bildanzeigen werden entfernt.

Bildtypen verwalten

Katalogverwalter können Bildtypem erstellen, bearbeiten oder löschen und Bildtypen zuordnen.

Bildtypen erstellen

Um einen neuen Bildtypen anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildtypen.

    Alle Bildtypen der Organisation werden angezeigt.

  3. Klicken Sie auf Neu.

    Eine leere Detailansicht des Bildtypen wird geöffnet.

  4. Geben Sie die Details zum Bildtypen wie erforderlich an.

    Die folgende Tabelle zeigt Details zu den Bildtypen.

    Tabelle 24: Details Bildtypen
    Parameter Status Beschreibung
    Name obligatorisch Legt den Anzeigennamen des neuen Bildtyps fest.
    ID obligatorisch Legt die systeminterne ID des neuen Bildtyps fest.
    Quelle obligatorisch Legt fest, ob die Bilder aus einem lokalen (Intershop Commerce Management-) Bildverzeichnis oder einem externen Server abgerufen werden (intern|extern).
    URL-Präfix obligatorisch Legt das URL-Präfix fest, welches für die komplette Auflösung des Speicherorts für die Bilddateien benötigt wird. Für interne (Intershop Commerce Management-) Speicherorte beginnt der Pfad mit <site>/:.
    Breite optional Legt die Breite eines Bildes fest, die vom aktuellen Typ abhängt. Dieser Parameter soll von den Content-Komponenten genutzt werden, die sich auf den aktuellen Bildtypen beziehen.
    Höhe optional Legt die Höhe eines Bildes fest, die vom aktuellen Typ abhängt. Dieser Parameter soll von den Content-Komponenten genutzt werden, die sich auf den aktuellen Bildtypen beziehen.
    Ersatz-Bild optional Legt ein Bild fest, das als Ersatz genutzt werden kann. Mit Klick auf Auswählen wird ein Dialog zum Durchsuchen der lokalen Bildverzeichnisse und zur Auswahl eines verfügbaren Bildes geöffnet.

  5. Klicken Sie auf Übernehmen.

    Der neue Bildtyp wird erstellt und die Abschnitte Verwandte Bildanzeigen und Benutzerdefinierte Attribute werden angezeigt.

    Sie können nun Bildanzeigen zuordnen. Klicken Sie auf Zurück zur Liste, um zurück zur Bildtypen-Liste zu gelangen.

Bildsets verwalten

Die Verwaltung von Bildsets besteht aus dem Erstellen, Bearbeiten und Löschen von Bildsets sowie dem Hinzufügen von Anzeigen-Typ-Beziehungen und Zuordnungen von Klassifikationskategorien zu Bildsets.

Bildsets erstellen

Um ein neues Bildset anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildsets.

    Alle Bildsets der Organisation werden angezeigt.

  3. Klicken Sie auf Neu.

    Eine leere Detailansicht des Bildsets wird geöffnet.

  4. Geben Sie einen Namen und eine ID für das neue Bildset an und klicken Sie auf Übernehmen.

    Ein neues Bildset wird angelegt und die Abschnitte Verwandte Bildtypen und -anzeigen und Klassifikationskategorien werden angezeigt.

    Sie können nun Bildtyp/-anzeigen-Beziehungen hinzufügen oder Klassifikationskategorien zuordnen. Klicken Sie auf Zurück zur Liste, um zurück zur Bildset-Liste zu gelangen.

Bildtyp/-anzeigen-Beziehung hinzufügen

Um Bildtyp/-anzeigen-Beziehungen zu einem Bildset hinzuzufügen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildsets.

    Alle Bildsets der Organisation werden angezeigt.

  3. Klicken Sie auf den Namen oder die ID des Bildsets, das Sie bearbeiten wollen.

    Die Detailansicht des Bildsets wird geöffnet.

  4. Klicken Sie im Abschnitt Verwandte Bildtypen und anzeigen auf Neu.

    Eine Liste der verfügbaren Bildanzeigen wird angezeigt.

  5. Wählen Sie die gewünschte Bildanzeige aus und klicken Sie Weiter.

    Eine Liste der verfügbaren Bildtypen wird angezeigt.

  6. Wählen Sie den/die gewünschten Bildtyp(en) aus und klicken Sie Fertigstellen.

    Die gewählten Bildtypen/-anzeigen-Beziehungen werden dem aktuellen Bildset hinzugefügt und Sie werden auf die Detailseite des Bildsets zurückgeführt.

    Tip:

    Mit der Auswahl mehrerer Bildtypen auf der zweiten Seite des Assistenten werden mehrere Bildtypen/-anzeigen-Beziehungen zum aktuellen Bildset hinzugefügt.

Bildtyp/-anzeigen-Beziehung obligatorisch machen

Um eine Bildtyp/-anzeigen-Beziehung für das Bildset festzulegen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildsets.

    Alle Bildsets der Organisation werden angezeigt.

  3. Klicken Sie auf den Namen oder die ID des Bildsets, das Sie bearbeiten wollen.

    Die Detailansicht des Bildsets wird geöffnet.

  4. Markieren Sie im Abschnitt Verwandte Bildtypen- und -anzeigen das Kontrollkästchen Benötigt, damit die Bildtypen/-anzeigen-Beziehung(en) als obligatorisch gilt.

  5. Klicken Sie auf Übernehmen.

    Die gewählten Bildtyp/-anzeigen-Beziehungen werden für das Bildset als benötigt angesehen. Klicken Sie auf Zurück zur Liste, um zurück zur Bildset-Liste zu gelangen.

Bildtyp/-anzeigen-Beziehung entfernen

Um eine Bildtyp/-anzeigen-Beziehung von einem Bildset zu entfernen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildsets.

    Alle Bildsets der Organisation werden angezeigt.

  3. Klicken Sie auf den Namen oder die ID des Bildsets, das Sie bearbeiten wollen.

    Die Detailansicht des Bildsets wird geöffnet.

  4. Wählen Sie im Abschnitt Verwandte Bildtypen und -anzeigen die Kontrollkästchen der Bildtypen/-anzeigen-Zuordnungen, die entfernt werden sollen.

  5. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Die gewählten Bildtyp/-anzeigen-Beziehungen werden vom aktuellen Bildset entfernt. Klicken Sie auf Zurück zur Liste, um zurück zur Bildset-Liste zu gelangen.

Bildtyp/-anzeigen-Beziehungen sortieren

Mit der Sortierung von Bildtypen-/anzeigen-Beziehungen wird die Reihenfolge bestimmt, in der die zugehörigen Bilder abgerufen werden, wenn das Bildset z. B. in Content-Komponenten verwendet wird. Um die Bildtyp/-anzeigen-Beziehungen zu sortieren:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildsets.

    Alle Bildsets der Organisation werden angezeigt.

  3. Markieren Sie die Kontrollkästchen für die Bildtyp/-anzeigen-Beziehungen, die Sie sortieren wollen.

  4. Verwenden Sie die Pfeiltasten, um die gewählten Bildtyp/-anzeigen-Beziehungen neu zu sortieren.

    Die neue Sortierung der Bildtyp/-anzeigen-Beziehungen wird sofort übernommen.

Klassifikationskategorien zuweisen

Um Klassifikationskategorien zu einem Bildset zuzuordnen:

  1. Klicken Sie in der Navigationsleiste auf Kataloge | Bildverwaltung.

    Das Register Bild-Upload wird geöffnet.

  2. Öffnen Sie das Register Bildsets.

    Alle Bildsets der Organisation werden angezeigt.

  3. Klicken Sie auf den Namen oder die ID des Bildsets, das Sie bearbeiten wollen.

    Die Detailansicht des Bildsets wird geöffnet.

  4. Klicken Sie im Abschnitt Klassifikationskategorien auf Hinzufügen.

    Eine Liste der verfügbaren Klassifikationskataloge wird angezeigt.

  5. Markieren Sie den gewünschten Klassifikationskatalog und klicken Sie auf Weiter.

    Eine Dropdown-Liste von Kategorien der ersten Ebene des gewählten Klassifikationskatalogs wird angezeigt.

  6. Wählen Sie die gewünschte Klassifikationskategorie aus der Dropdown-Liste und klicken Sie Auswählen.

    Eine Dropdown-Liste von Unterkategorien der nächsten Ebene des zuvor gewählten Klassifikationskatalogs wird angezeigt. Um weiter durch den Kategoriebaum zu navigieren, wählen Sie eine Unterkategorie und klicken Sie entsprechend auf Auswählen.

  7. Gehen Sie zur Klassifikationskategorie, die dem Bildset zugeordnet werden soll und klicken Sie auf Fertigstellen.

    Somit wird die ausgewählte Klassifikationskategorie dem aktuellen Bildset zugeordnet und Sie kommen zurück auf die Detailseite des Bildsets.

Zuordnung von Klassifikationskategorien aufheben

    Die Zuordnung des gewählten Klassifikationskategorien zum Bildset wird entfernt. Klicken Sie auf Zurück zur Liste, um zurück zur Bildset-Liste zu gelangen.
