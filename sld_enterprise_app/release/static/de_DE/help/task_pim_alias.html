In diesem Abschnitt werden Aufgaben zur Verwaltung von Hersteller-Aliassen in der Produkt-Freigabe-Gruppe beschrieben, siehe Produktzuordnungen von Freigabe-Gruppen verwalten.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext der Organisation oder eines Partner Channels ausgeführt.

Hersteller-Aliasse verwalten

Das Verwalten von Hersteller-Aliassen umfasst das Anlegen und Löschen.

Hersteller-Aliasse anlegen

Um einen Hersteller-Alias anzulegen:

  1. Wählen Sie in der Navigationsleiste Master-Kataloge | Hersteller-Aliasse.

    Die Liste der Hersteller-Aliasse wird angezeigt. Alle verfügbaren Aliasse mit den zugeordneten Herstellern werden angezeigt.

  2. Klicken Sie auf Neu.

    Eine leere Detailansicht für Hersteller-Alias wird geöffnet.

  3. Geben Sie einen Namen an und klicken Sie Übernehmen.

    Der neue Hersteller-Alias wird erstellt und das Register Hersteller wird aktiviert. Sie können nun Hersteller zuordnen.

    Anderenfalls klicken Sie auf Abbrechen, um die Eingaben zu verwerfen.

Hersteller-Aliasse löschen

Um einen Hersteller-Alias zu löschen:

  1. Wählen Sie in der Navigationsleiste Master-Kataloge | Hersteller-Aliasse.

    Die Liste der Hersteller-Aliasse wird angezeigt. Alle verfügbaren Aliasse mit den zugeordneten Herstellern werden angezeigt.

  2. Markieren Sie das Kontrollkästchen des Hersteller-Alias, der entfernt werden soll.

  3. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Der gewählte Hersteller-Alias wird entfernt.

Hersteller-Zuordnungen verwalten

Das Verwalten von Hersteller-Zuordnungen umfasst das Hinzufügen von Herstellern zu einem Alias und das Entfernen von Herstellern. Hinweis: Hersteller, die einem Hersteller-Alias zugeordnet wurden, sind nicht mehr einzeln verfügbar, sondern nur unter dem gemeinsamen Alias-Namen.

Hersteller zuordnen

Um Hersteller einem Alias zuzuordnen:

  1. Wählen Sie in der Navigationsleiste Master-Kataloge | Hersteller-Aliasse.

    Die Liste der Hersteller-Aliasse wird angezeigt. Alle verfügbaren Aliasse mit den zugeordneten Herstellern werden angezeigt.

  2. Klicken Sie auf den Namen des Alias, die Sie bearbeiten wollen.

    Die Detailansicht des Hersteller-Alias wird geöffnet.

  3. Öffnen Sie das Register Hersteller.

    Die Liste von Herstellern für diesen Alias wird angezeigt.

  4. Klicken Sie auf Zuordnen.

    Eine Liste mit allen verfügbaren Herstellern wird angezeigt.

    Hinweis:

    Hersteller, die bereits einem Alias zugeordnet sind, werden nicht aufgelistet.

  5. Wählen Sie den/die gewünschten Hersteller und klicken Sie auf Übernehmen.

    Die gewählten Hersteller werden zum aktuellen Alias hinzugefügt. Mit Zurück kommen Sie zurück zum Register Hersteller der Alias-Detailansicht.

Hersteller-Zuordnungen aufheben

Um die Zuordnung von Herstellern zu einem Alias aufzuheben:

  1. Wählen Sie in der Navigationsleiste Master-Kataloge | Hersteller-Aliasse.

    Die Liste der Hersteller-Aliasse wird angezeigt. Alle verfügbaren Aliasse mit den zugeordneten Herstellern werden angezeigt.

  2. Klicken Sie auf den Namen des Alias, die Sie bearbeiten wollen.

    Die Detailansicht des Hersteller-Alias wird geöffnet.

  3. Öffnen Sie das Register Hersteller.

    Die Liste von Herstellern für diesen Alias wird angezeigt.

  4. Markieren Sie das/die Kontrollkästchen des/der Hersteller, die entfernt werden sollen.

  5. Klicken Sie auf Aufheben und bestätigen Sie die Operation.

    Die ausgewählten Hersteller werden vom Hersteller-Alias entfernt.
