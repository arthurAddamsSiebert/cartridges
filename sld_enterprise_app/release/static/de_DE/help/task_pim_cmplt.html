Die Produkt-Vollständigkeitsseite zeigt allgemeine Produktstatistiken und hilft, fehlende oder falsche Daten zu identifizieren und zu korrigieren. Die Seite ist in zwei Abschnitte geteilt, die unten beschrieben werden: Produktübersicht und Produkt-Vollständigkeit.

Produktübersicht

Der Abschnitt Produktübersicht zeigt statistische Informationen und Status-Berichte der Produkte. Die Anzeige ist nach Locale und Währung gefiltert.

Um die Produktübersicht anzuzeigen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox.

    Die Navigationsleiste für den Channel wird geöffnet.

  2. Wählen Sie in der Navigationsleiste Kataloge | Produkt-Vollständigkeit.

    Die Übersichtsseite wird angezeigt.

  3. Klicken Sie im Abschnitt Produktübersicht auf Aktualisieren.

    Hinweis:

    Die Schaltfläche Aktualisieren ist nur in Bearbeitungssystemen in Datenreplizierungsumgebungen verfügbar, siehe

    Die aktuellen Daten werden abgerufen.

Hinweis:

Die angezeigten Daten werden automatisch durch einen speziellen geplanten Job erfasst. Wenden Sie sich an Ihren Systemadministrator, um diesen Job zu konfigurieren.

Die linke Spalte der Produktübersicht zeigt allgemeine Produktstatistiken, und zwar

  • Anzahl Produkte

  • Anzahl Produktbündel

  • Anzahl gebündelte Produkte

  • Anzahl Variations-Stammprodukte

  • Anzahl Variationsprodukte

  • Anzahl Retail-Sets

  • Anzahl Retail-Set-Einzelartikel

Die rechte Spalte der Produktübersicht zeigt Berichte zum Produktstatus, und zwar

  • Produkte ohne Listenpreis

  • Produkte ohne Steuer

  • Produkte ohne Kurzbeschreibung

  • Produkte ohne ausführliche Beschreibung

  • Produkte online

  • Produkte offline

  • Produkte nicht verfügbar

  • Produkte ohne Kategoriezuordnung

Mit Klick auf einen Eintrag in der Übersicht wird die erweiterte Suche geöffnet, wobei die abgefragten Daten (der Eintrag in der Übersicht) in der Ergebnisliste gezeigt werden. Durch zusätzliche Suchkriterien können Sie das Ergebnis noch weiter eingrenzen.

Tip:

Wenn die Produktgenehmigung aktiviert ist, können Sie die Prüfung und Genehmigung direkt aus dem Suchergebnis heraus anstoßen. Für weitere Informationen siehe Produktgenehmigung.

Produkt-Vollständigkeitsprüfung

Katalog- und Produktverwalter können Produkt-Vollständigkeitsregeln definieren und sie auf ausgewählte Produkt-Sets anwenden. Im Abschnitt Produkt-Vollständigkeit werden die erstellten Prüfergebnisse dann zusammengefasst.

Produkt-Vollständigkeitsprüfungen durchführen

Um eine Produkt-Vollständigkeitsprüfung durchzuführen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox.

    Die Navigationsleiste für den Channel wird geöffnet.

  2. Wählen Sie in der Navigationsleiste Kataloge | Produkt-Vollständigkeit.

    Die Übersichtsseite wird angezeigt.

  3. Wählen Sie im Abschnitt Produkt-Vollständigkeit ein Produkt-Sortiment aus.

    Alle aktiven Regeln werden auf das ausgewählte Produktset angewendet.

  4. Klicken Sie auf Aktualisieren.

    Die aktuellen Daten werden abgerufen.

    Wird ein regel-spezifischer Eintrag geklickt, öffnet sich ein Bericht, der alle Produkte anführt, bei denen die Prüfung fehlgeschlagen ist.

    Durch Klicken eines einzelnen Produkts in der Ergebnisliste wird die entsprechende Detailansicht angezeigt, was das Hinzufügen/Bearbeiten benötigter Daten ermöglicht.

Produkt-Vollständigkeitsregeln

Die Produkt-Vollständigkeitsregel stellt das Kriterium dar, nach dem sich die Produkt-Vollständigkeitsprüfung richtet. Intershop Commerce Management unterstützt derzeit die folgenden Typen von Produkt-Vollständigkeitsregeln:

  • Standard-Produktattribut: wird genutzt, um ein konfigurierbares Set an Standardattributen auf dessen Wert hin zu prüfen.

  • Standardpreise: werden genutzte, um zu prüfen, ob die Produkte Listen- bzw. Einstandspreise für gewählte Locales haben.

  • Benutzerdefiniertes Produktattribut: wird genutzt, um ein konfigurierbares Set an benutzerdefinierten Produktattributen auf dessen Wert hin zu prüfen.

  • Kategoriezuordnung: wird genutzt, um zu prüfen, ob Produkte einer Kategorie eines Standardkatalogs oder eines Klassifikationskatalogs zugeordnet sind.

  • Kundenspezfische Preisliste: wird genutzt, um, zu prüfen, ob Produkte in einer bestimmten Preisliste einen gültigen Preis haben.

  • Bildzuordnung: wird genutzt, um zu prüfen, ob den Produkten die benötigten Bilder zugeordnet sind; es wird geprüft, ob alle obligatorischen Bilder für Bilder-Sets und mindestens ein definiertes Bild pro Produkt vorhanden sind.

Das Verwalten von Produkt-Vollständigkeitsregeln umfasst das Erstellen, Bearbeiten, (De-)Aktivieren und Löschen von Regeln.

Produkt-Vollständigkeitsregeln erstellen

Um eine Produkt-Vollständigkeitsregel zu erstellen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produkt-Vollständigkeit.

    Die Übersichtsseite wird angezeigt.

  2. Klicken Sie im Abschnitt Produkt-Vollständigkeit auf Regeln konfigurieren.

    Die Liste der Produkt-Vollständigkeitsregeln wird angezeigt.

  3. Klicken Sie Neu.

    Der Dialog Produkt-Vollständigkeitsregel wird angezeigt.

  4. Geben Sie die Details wie erforderlich an.

    Erforderliche Angaben sind der Name und Typ der Regel. Die weiteren Daten hängen vom gewählten Typ ab.

  5. Falls erforderlich, wählen Sie eine Aktion für die Regel.

    Diese Aktion wird durchgeführt, wenn die Prüfung eines Produkts (basierend auf der aktuellen Regel) fehlgeschlagen ist.

  6. Klicken Sie auf OK.

    Die Regel wird erstellt und Sie werden zur Liste der Produkt-Vollständigkeitsregeln zurückgeleitet.

Produkt-Vollständigkeitsregeln bearbeiten

Um eine Produkt-Vollständigkeitsregel zu bearbeiten:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produkt-Vollständigkeit.

    Die Übersichtsseite wird angezeigt.

  2. Klicken Sie im Abschnitt Produkt-Vollständigkeit auf Regeln konfigurieren.

    Die Liste der Produkt-Vollständigkeitsregeln wird angezeigt.

  3. Klicken Sie auf den Namen der Regel, die Sie bearbeiten möchten.

    Der Dialog Produkt-Vollständigkeitsregel wird angezeigt.

  4. Bearbeiten Sie die Details wie erforderlich und klicken Sie OK.

    Die Regel-Einstellungen werden aktualisiert und Sie werden zur Liste der Produkt-Vollständigkeitsregeln zurückgeleitet.

Produkt-Vollständigkeitsregeln aktivieren

Um eine Produkt-Vollständigkeitsregel zu de-/aktivieren:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produkt-Vollständigkeit.

    Die Übersichtsseite wird angezeigt.

  2. Klicken Sie im Abschnitt Produkt-Vollständigkeit auf Regeln konfigurieren.

    Die Liste der Produkt-Vollständigkeitsregeln wird angezeigt.

  3. Markieren Sie die Regel, die Sie aktivieren oder deaktivieren möchten.

    Markieren Sie das Kontrollkästchen in der Zeile der entsprechenden Regel.

  4. Klicken Sie entsprechend auf Aktivieren oder Deaktivieren.

    Die gewählte Regel wird aktiviert bzw. deaktiviert.

Produkt-Vollständigkeitsregeln löschen

Um eine Produkt-Vollständigkeitsregel zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produkt-Vollständigkeit.

    Die Übersichtsseite wird angezeigt.

  2. Klicken Sie im Abschnitt Produkt-Vollständigkeit auf Regeln konfigurieren.

    Die Liste der Produkt-Vollständigkeitsregeln wird angezeigt.

  3. Markieren Sie die Regel, die Sie löschen möchten.

    Markieren Sie das Kontrollkästchen in der Zeile der entsprechenden Regel.

  4. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Die gewählte Regel wird gelöscht.

Produkt-Sortimente

Mithilfe von Produkt-Sortimenten kann eine Anzahl an Produkten gesammelt werden, die zusammen geprüft werden sollen. Intershop Commerce Management unterstützt derzeit die folgenden Produkt-Sortimente:

  • alle Produkte in allen Katalogen

  • alle Produkte in ausgewählten Kategorien

Das Verwalten von Produkt-Sortimenten umfasst das Erstellen und Löschen von Sets.

Produkt-Sortimente erstellen

Um ein Produkt-Sortiment zu erstellen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produkt-Vollständigkeit.

    Die Übersichtsseite wird angezeigt.

  2. Klicken Sie im Abschnitt Produkt-Vollständigkeit auf Produkt-Sortimente konfigurieren.

    Die Liste der Produkt-Sortimente wird angezeigt.

  3. Klicken Sie Neu.

    Der Dialog Produkt-Sortimente wird angezeigt.

  4. Geben Sie die Details wie erforderlich an.

    Erforderliche Angaben sind der Name und Typ des Sortiments. Die weiteren Daten hängen vom gewählten Typ ab.

  5. Klicken Sie auf OK.

    Das Sortiment wird erstellt und Sie werden zur Liste der Produkt-Sortimente zurückgeleitet.

Produkt-Sortimente löschen

Um Produkt-Sortimente zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produkt-Vollständigkeit.

    Die Übersichtsseite wird angezeigt.

  2. Klicken Sie im Abschnitt Produkt-Vollständigkeit auf Produkt-Sortimente konfigurieren.

    Die Liste der Produkt-Sortimente wird angezeigt.

  3. Markieren Sie das Sortiment, das Sie löschen möchten.

    Markieren Sie das Kontrollkästchen in der Zeile des entsprechenden Sortiments.

  4. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Das gewählte Sortiment wird gelöscht.
