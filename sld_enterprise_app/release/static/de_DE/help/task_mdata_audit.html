Auditbericht-Verwaltung

  • Auditbericht konfigurieren
  • Auditbericht erzeugen
  • Audit-Ergebnisse anzeigen

Dieser Abschnitt beschreibt, wie für den Management-Kontext ein Protokoll von den von Benutzern ausgelösten Commerce Management-Ereignissen erstellt werden kann. Sie können nach Änderungen von Commerce Management-Objekten suchen und diese nach Benutzer-Informationen sortieren. Ereignisse werden standardmäßig für zwei Jahre im System erfasst und gespeichert.

Mit dem Auditbericht kann der Administrator Ereignisberichte aufgrund von Benutzeraktivitäten auf Organisations-, Channel- und Applikations-Ebene erstellen. Berichte sind so konfiguriert, dass sie alle Ereignisse zu Erzeugen, Aktualisieren und Löschen enthalten. Wurde der Bericht erzeugt, sind die Ergebnisse sortierbar und können zur Archivierung oder weiteren Bearbeitung exportiert werden. Auditberichte sind im Commerce Management in Ihrem Management-Kontext im Register Massendaten unter dem Punkt Auditbericht verfügbar.

Hinweis:

Die gesammelten Audit-Informationen können durch einen zugehörigen geplanten Job automatisch exportiert und bereinigt werden. Wenden Sie sich an Ihren Systemadministrator, um diesen Job zu konfigurieren.

Auditbericht konfigurieren

Die Ansicht für die Auditbericht-Konfiguration zeigt eine Reihe von Filtern zur Konfiguration eines Berichts. Standardmäßig sind alle Objektereignisse der letzten zwei Jahre im Bericht enthalten. Die folgende Tabelle zeigt eine Beschreibung von jedem konfigurierbaren Filter.

Tabelle 108: Filter für Auditberichte konfigurieren
Filter Beschreibung
Kontext Dieser Filter ist auf Organisations- und Channel-Ebene verfügbar. Er erlaubt es, den Management-Kontext auszuwählen, von dem aus Ereignis-/Objektaktivitäten wiedergegeben werden.
Benutzer Der Filter erlaubt Ihnen, einen Bericht zu konfigurieren, der die Aktivitäten der gewählten Benutzer anzeigt. Wird ein bestimmter Management-Kontext ausgewählt, werden nur die Benutzer dieses Management-Kontextes angezeigt und können ausgewählt werden.
Objekttyp Der Filter erlaubt Ihnen, einen Bericht zu konfigurieren, der nur die ausgewählten Objekte anzeigt. Wird ein bestimmter Management-Kontext ausgewählt, werden nur die Objekte dieses Management-Kontextes angezeigt und können ausgewählt werden.
Transaktions-Typ Der Filter erlaubt Ihnen, einen Bericht zu konfigurieren, der nur die ausgewählten Ereignisse anzeigt.
Datumsbereich Mit der Datumswahl können Sie das Start- und Enddatum für die Konfiguration des Berichts festlegen.
Hinweis:

Standardmäßg sind alle Filter auf "Alle auswählen" gesetzt. Sie haben die Option, manuell mehrere Werte zu wählen, jedoch müssen Sie mindestens einen Wert auswählen. Andernfalls wird das System auf Standard zurückgesetzt.

Auditbericht erzeugen

Um einen Auditbericht zu erzeugen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Massendatenverwaltung | Auditbericht.

    Das Modul Auditbericht wird angezeigt.

  2. Wählen Sie den Management-Kontext aus dem Kontextfilter.

    Nur die Aktivität des Management-Kontextes, den Sie ausgewählt haben, ist im Bericht enthalten.

  3. Wählen Sie aus dem Filter Benutzer-ID die Benutzer aus.

    Nur die Aktivitäten derjenigen Benutzer-IDs, die Sie auswählen, sind im Bericht enthalten.

  4. Wählen Sie aus dem Objekttyp-Filter die Objekte aus.

    Nur die Aktivitäten derjenigen Objekte, die Sie auswählen, sind im Bericht enthalten.

  5. Wählen Sie aus dem Filter Transaktionsyp die Ereignisse aus.

    Nur die ausgewählten Transaktionen sind im Bericht enthalten.

  6. Wählen Sie aus der Datumsauswahl den Datumsbereich aus.

    Sie können einen bestimmten Tag oder einen Zeitraum festlegen, für den der Bericht erzeugt wird.

    Hinweis:

    Der Datumsbereich ist durch den Zeitraum begrenzt, für den die Informationen im System gespeichert werden.

  7. Klicken Sie Bericht erzeugen.

    Ein Bericht basierend auf den zuvor angegebenen Einstellungen wird erzeugt.

Audit-Ergebnisse anzeigen

Wenn Sie auf Bericht erzeugen geklickt haben, erscheint ein Fenster mit den Ergebnissen des Auditberichts, wobei die Konfiguration und der Bericht in einer Tabelle unten auf der Seite angezeigt werden. Folgende Optionen stehen zur Verfügung:

  • Klicken Sie Export in CSV, um den Dialog "Speichern als" zu öffnen.

    Hier können Sie die Datei in das lokale Dateisystem speichern.

  • Die Ergebnistabelle kann durch Klicken der Spaltenüberschriften sortiert werden.

  • Sie können die Filtereinstellungen ändern und einen neuen Bericht erzeugen.

  • Klicken Sie auf Details anzeigen für einen bestimmten Eintrag.

    Folgendes wird angezeigt:

    Tabelle 109: Bestimmte Ereignisdetails anzeigen
    Ereignistyp Details
    Aktualisieren Zeigt die Objektattribut-Daten vor und nach der Aktualisierung. Sie können sehen, was sich verändert hat.
    Löschen Zeigt die Objektattribut-Daten vor dem Löschen.
    Anlegen Zeigt die aktuellen Attribute des erzeugten Objekts.
