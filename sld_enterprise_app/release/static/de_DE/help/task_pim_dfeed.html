Produktdaten-Feeds sind ein Instrument, um den Inhalt eines Produktbestands in verschiedenartige Ziele auszulagern: entweder in Dateien eines bestimmten Formats oder in ein anderes Intershop Commerce Management-System (Zielsystem).

Für eine Einführung zu Produktdaten-Feeds, siehe Produktdaten-Feeds: Konzepte.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Neue Konfiguration erzeugen

Um eine neue Konfiguration für einen Produktdaten-Feed zu erzeugen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produktdaten-Feed.

    Eine Liste aller aktuell definierten Konfigurationen wird angezeigt.

  2. Klicken Sie auf Neu.

    Die Seite Allgemein der neuen Konfiguration wird angezeigt.

  3. Bearbeiten Sie die allgemeinen Angaben wie erforderlich.

    Dazu gehören als notwendige Angaben Name und ID für die neue Konfiguration. Optional können Sie eine Beschreibung angeben. Um die Konfiguration zu aktivieren/deaktivieren, markieren Sie das Kontrollkästchen Aktiviert bzw. entfernen die Markierung.

  4. Klicken Sie auf Übernehmen.

    Die neue Konfiguration wird angelegt. Alle anderen Register werden aktiviert.

  5. Öffnen Sie die anderen Register, um entsprechend Ziel, Job-Ablaufplan und Produktzuordnungen festzulegen.

    Für nähere Informationen zu vordefinierten Zielen, siehe Ziele von Produktdaten-Feeds.

    Für weitere Informationen zu Produkt-Zuweisungen, siehe Produkte suchen und zuordnen und Produkte durchsuchen und zuordnen.

Produkte suchen und zuordnen

Um Produkte über die Suchfunktion zu einer Konfiguration zuzuordnen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produktdaten-Feed.

    Eine Liste aller aktuell definierten Konfigurationen wird angezeigt.

  2. Klicken Sie auf den Namen der Konfiguration, der Sie Produkte zuordnen wollen.

  3. Öffnen Sie das Register Produkte.

    Die Liste zeigt alle derzeit der Konfiguration zugeordneten Produkte.

  4. Klicken Sie auf Zuordnen über Suche.

    Die Produkt-Listenansicht wird angezeigt.

  5. Weisen Sie die Produkte wie erforderlich zu.

    • Um der aktuellen Konfiguration alle Produkte zuzuordnen, klicken Sie auf Alle zuordnen.

    • Um Produkte einzeln zuzuordnen, markieren Sie die entsprechenden Kontrollkästchen und klicken auf Auswahl zuordnen.

Produkte durchsuchen und zuordnen

Mit dieser Option lassen sich alle oder einzelne Produkte eines Kataloges bzw. einer Kategorie zur aktuellen Konfiguration zuordnen. Beim Ausgeben eines Kataloges oder einer Kategorie werden die Produkte aus allen jeweiligen Unterkategorien automatisch mit erfasst.

Um Kataloge aller verfügbaren Produktbestände nach zu syndizierenden Produkten zu durchsuchen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produktdaten-Feed.

    Eine Liste aller aktuell definierten Konfigurationen wird angezeigt.

  2. Klicken Sie auf den Namen der Konfiguration, der Sie Produkte zuordnen wollen.

  3. Öffnen Sie das Register Produkte.

    Die Liste zeigt alle derzeit der Konfiguration zugeordneten Produkte.

  4. Klicken Sie auf Zuordnen über Durchsuchen.

    Eine Liste der verfügbaren Kataloge im Channel-Produktbestand wird angezeigt.

  5. Weisen Sie die Produkte wie erforderlich zu.

    • Um alle Produkte eines Kataloges in die aktuelle Konfiguration aufzunehmen, klicken Sie in der Zeile des entsprechenden Kataloges auf Alle zuordnen.

    • Um bestimmte Katalogkategorien zu durchsuchen, markieren Sie das Optionsfeld für den Katalog und klicken Sie auf Weiter. Klicken Sie auf die Kategorien und Unterkategorien, bis Sie die Kategorie erreicht haben, aus der Sie Produkte ausgeben wollen. Alternativ können Sie den Katalog so weit durchsuchen, bis Sie die Produktliste, die der gewünschten Kategorie zugeordnet ist, geöffnet haben.

    • Um alle Produkte einer Kategorie in die Konfiguration aufzunehmen, markieren Sie das Kontrollkästchen für die Kategorie und klicken Sie auf Zuordnen.

    • Um bestimmte Produkte einer Kategorie aufzunehmen, öffnen Sie alle Unterkategorien, bis die gewünschte Produktliste geöffnet ist. Klicken Sie auf Alle zuordnen, um die gesamte Liste zu syndizieren, oder markieren Sie bestimmte Produkte, und klicken Sie auf Auswahl zuordnen.

Attributzuordnungsregeln bearbeiten

Der Dialog Attributzuordnungsregel wird automatisch angezeigt, wenn eine Konfiguration mit dem Ziel "Standard Transformed" erzeugt wurde. Mit den Standardeinstellungen wird nichts kopiert oder beibehalten. Das heißt, dass zuerst die Attributzuordnungen bearbeitet werden müssen, bevor Produktdaten-Feeds ein Ergebnis liefern können. Um die Zuordnungsregeln für eine Konfiguration zu bearbeiten:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produktdaten-Feed.

    Eine Liste aller aktuell definierten Konfigurationen wird angezeigt.

  2. Klicken Sie auf Regeln bearbeiten neben der Konfiguration, für die Sie Zuordnungsregeln bearbeiten wollen.

    Der Dialog Zuordnungsregeln wird geöffnet.

  3. Bearbeiten Sie die Einstellungen wie erforderlich, und klicken Sie auf Übernehmen.

    Für einen Überblick über die verfügbaren Regeln, siehe Zuordnungsregeln bearbeiten.

Produktdaten-Feeds manuell ausführen

Um einen Job für Produktdaten-Feeds mit einer bestimmten Konfiguration manuell zu starten:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produktdaten-Feed.

    Eine Liste aller aktuell definierten Konfigurationen wird angezeigt.

  2. Markieren Sie die Konfiguration, die für den Prdouktdaten-Feed-Job verwendet werden soll, und klicken Sie auf Ausführen.

    Dadurch wird der Produktdaten-Feed-Prozess gestartet. Klicken Sie auf Aktualisieren, um den Seiteninhalt neu zu laden.

Tip:

Klicken Sie nach Ablauf des Jobs auf den Namen der Exportdatei, um die Datei herunterzuladen. Um die Log-Datei anzuzeigen, klicken Sie auf den entsprechenden Eintrag in der Spalte Letzte Syndizierung.

Produktdaten-Feeds automatisch ausführen

Produktdaten-Feeds können nicht nur manuell, sondern auch automatisch durch einen geplanten Job gestartet werden.

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Produktdaten-Feed.

    Eine Liste aller aktuell definierten Konfigurationen wird angezeigt.

  2. Klicken Sie auf den Namen der Konfiguration.

  3. Öffnen Sie das Register Zeitplan.

  4. Legen Sie das Ausführungsintervall für die aktuelle Konfiguration fest und klicken Sie dann auf Übernehmen.
