Partner: Designverwaltung

  • Designpaket laden
  • Designpaket installieren
  • Designpaket deinstallieren
  • Designpaket entfernen
Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Partner Channels ausgeführt.

Designpaket laden

Um ein neues Designpaket für einen Partner zu laden:

  1. Öffnen Sie das Modul Partner.

    Wählen Sie den Partner-Channel aus der Kontextauswahlbox, wählen Sie dann Partner. Die Partnerliste wird angezeigt.

  2. Klicken auf den Namen oder die ID des Partners.

    Die Detailansicht des Partners wird angezeigt.

  3. Öffnen Sie das Register Design.

    Eine Liste der bereits geladenen Designpakete wird angezeigt.

  4. Klicken Sie auf Upload.

    Der Dialog Neues Designpaket wird angezeigt.

  5. Geben Sie einen Namen für das Designpaket, eine Beschreibung sowie Pfadangabe und Name der Archivdatei an.

  6. Klicken Sie auf Übernehmen.

    Das Designpaket wird geladen und kann jetzt installiert werden.

Designpaket installieren

Um ein Designpaket für einen Partner zu installieren:

  1. Öffnen Sie das Modul Partner.

    Wählen Sie den Partner-Channel aus der Kontextauswahlbox, wählen Sie dann Partner. Die Partnerliste wird angezeigt.

  2. Klicken auf den Namen oder die ID des Partners.

    Die Detailansicht des Partners wird angezeigt.

  3. Öffnen Sie das Register Design.

    Eine Liste der bereits geladenen Designpakete wird angezeigt.

  4. Klicken Sie in der Zeile des Designpakets, das installiert werden soll, auf Installieren.

    Eine Liste der vorhandenen Installationen dieses Designpakets wird angezeigt.

  5. Wählen Sie im Feld Neue Installation die Locale aus, für die das Paket installiert werden soll.

    Wählen Sie aus der Dropdown-Liste eine der vorhandenen Locales aus, und klicken Sie auf Installieren.

Designpaket deinstallieren

Um ein Designpaket für einen Partner zu deinstallieren:

  1. Öffnen Sie das Modul Partner.

    Wählen Sie den Partner-Channel aus der Kontextauswahlbox, wählen Sie dann Partner. Die Partnerliste wird angezeigt.

  2. Klicken auf den Namen oder die ID des Partners.

    Die Detailansicht des Partners wird angezeigt.

  3. Öffnen Sie das Register Design.

    Eine Liste der bereits geladenen Designpakete wird angezeigt.

  4. Klicken Sie in der Zeile des Designpakets, das deinstalliert werden soll, auf Installieren.

    Eine Liste der vorhandenen Installationen dieses Designpakets wird angezeigt.

  5. Markieren Sie die Installationen, die Sie entfernen wollen.

    Markieren Sie die Kontrollkästchen neben den Installationen, die Sie löschen wollen.

  6. Klicken Sie auf Löschen.

    Die Installation wird entfernt.

Designpaket entfernen

Um ein Designpaket für einen Partner zu entfernen:

  1. Öffnen Sie das Modul Partner.

    Wählen Sie den Partner-Channel aus der Kontextauswahlbox, wählen Sie dann Partner. Die Partnerliste wird angezeigt.

  2. Klicken auf den Namen oder die ID des Partners.

    Die Detailansicht des Partners wird angezeigt.

  3. Öffnen Sie das Register Design.

    Eine Liste der bereits geladenen Designpakete wird angezeigt.

  4. Markieren Sie das Kontrollkästchen für das Designpaket, das Sie löschen wollen.

  5. Klicken Sie auf Löschen und bestätigen Sie das Löschen.

    Das Designpaket wird aus dem System entfernt.
