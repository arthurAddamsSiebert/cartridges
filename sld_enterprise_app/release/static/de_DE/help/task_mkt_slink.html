Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Die Verwaltung von Kurz-URLs beinhaltet das Erstellen und Bearbeiten individueller Kurz-URLs und URL-Gruppen sowie das Exportieren und Importieren von URLs. Alle diese Aufgaben werden mithilfe des Moduls Kurz-URLs durchgeführt.

Hinweis:

Um Kurz-URLs zu nutzen, sind zusätzliche Konfigurationen und Einstellungen nötig. Kurz-URLs benötigen z. B. einen DNS-Eintrag. Bitte wenden Sie sich zur Erstellung von URLs für den Management-Kontext an Ihren Systemadministrator. Für weitere Informationen, siehe Kurz-URLs und URL-Weiterleitung.

Kurz-URLs anzeigen

Um Details zu einer Kurz-URL anzuzeigen:

  1. Öffnen Sie das Modul Kurz-URL.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie Kurz-URLs.

  2. Klicken Sie auf den Link einer Kurz-URL.

    Die Detailansicht mit dem allgemeinen Register für Kurz-URLs wird angezeigt.

Kurz-URLs anlegen

Um eine Kurz-URL anzulegen:

  1. Öffnen Sie das Modul Kurz-URL.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie Kurz-URLs.

  2. Klicken Sie auf Neu.

    Die Detailansicht für die neuen Kurz-URLs wird angezeigt.

  3. Geben Sie Konfigurationsdetails für die Kurz-URL ein.

    Details zu den verfügbaren Optionen entnehmen Sie aus der folgenden Tabelle:

    Tabelle 49: Konfigurationsoptionen Kurz-URLs
    Option Beschreibung
    Ausgangs-URL Spezifiziert die Kurz-URL
    Ziel-URL Spezifiziert die komplette Intershop Commerce Management-URL, zu der die Kurz-URL zugeordnet ist.
    Aktiviert Mit diesem Kontrollkästchen können Kurz-URLs entweder aktiviert oder deaktiviert werden.
    Standard-URL Wenn mehrere Kurz-URLs auf die gleiche Ziel-URL verweisen, kann eine Kurz-URL als Standard-URL festgelegt werden. Sollte eine dieser Kurz-URLs nicht mehr gültig sein (weil z. B. ihr Ablaufdatum verstrichen ist), wird stattdessen die Standard-URL verwendet.
    Weiterleitung Diese Option definiert URL-Weiterleitungen für eine gewählte Ausgangs-URL, wobei 301-Header (dauerhaft verschoben) normalerweise zur Storefront-Homepage weiterleiten, und 302-Header (vorübergehend verschoben) zu Kategorien oder Produkten weiterleiten. Für Kurz-URLs verwenden Sie keine.
    Beschreibung Die Beschreibung gibt z. B. den Grund für eine URL-Weiterleitung an.
    Startdatum, Enddatum Die Angabe von Start- und Enddatum kann dazu verwendet werden, die Gültigkeit der Kurz-URL auf einen bestimmten Zeitraum zu beschränken.
    Hinweis:

    Die Quell-URL und die Ziel-URL können entweder absolut oder relativ sein. Für weitere Informationen, siehe Kurz-URLs und URL-Weiterleitung.

  4. Klicken Sie auf Übernehmen.

    Die neue Kurz-URL wird in der URL-Listenansicht angezeigt.

  5. Testen Sie die neue Kurz-URL.

    Klicken Sie auf URL in der Tabellenzeile der entsprechenden Kurz-URL. Die Ziel-URL wird in einem neuen Browser-Fenster geöffnet.

Hinweis:

Antwortseiten für Kurz-URL-Aufrufe werden vom Webadapter innerhalb des (in vanity.pathinfo, Vorgabe: SLDSystem) definierten Site-Kontexts verarbeitet und zwischengespeichert. Bei Änderungen der Kurz-URL-Konfiguration muss der Seiten-Cache der entsprechenden Site (z. B. SLDSystem) gelöscht werden, damit die Modifikationen wirksam werden. Nähere Informationen erhalten sie von Ihrem Systemadministrator.

Kurz-URLs bearbeiten

Kurz-URLs können aktiviert/deaktiviert oder verändert werden.

Kurz-URLs aktivieren/deaktivieren

Um Kurz-URLs in der Listenansicht zu aktivieren/deaktivieren:

  1. Öffnen Sie das Modul Kurz-URL.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie Kurz-URLs.

    Eine Liste aller bestehenden Kurz-Urls wird angezeigt.

  2. Klicken Sie neben dem betreffenden Link auf das entsprechende Symbol in der Status-Spalte.

    Mit Klick auf das Symbol wird die Kurz-URL aktiviert, und mit Klick auf das Symbol wird die Kurz-URL deaktiviert.

Kurz-URLs bearbeiten

Um bestehende Kurz-URs zu bearbeiten:

  1. Öffnen Sie das Modul Kurz-URL.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie Kurz-URLs.

    Eine Liste aller bestehenden Kurz-Urls wird angezeigt.

  2. Klicken Sie auf die Ausgangs- oder Ziel-URL der Kurz-URL, die Sie bearbeiten möchten.

    Die Detailansicht der Kurz-URL wird angezeigt.

  3. Bearbeiten Sie die Einstellungen wie erforderlich.

    Weitere Informationen zu den verfügbaren Optionen, siehe Kurz-URLs anlegen .

  4. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Löschen von Kurz-URLs

Um Kurz-URLs zu löschen:

  1. Öffnen Sie das Modul Kurz-URL.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie Kurz-URLs.

    Eine Liste aller bestehenden Kurz-Urls wird angezeigt.

  2. Markieren Sie das Kontrollkästchen für die Kurz-URL, die Sie löschen wollen.

  3. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

Kurz-URLs können auch über die Schaltfläche Löschen in der Detailansicht gelöscht werden.

URL-Gruppen erstellen

Um eine URL-Gruppe für Kurz-URLs anzulegen:

  1. Öffnen Sie das Modul URL-Gruppen-Verwaltung.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie dann URL-Gruppen.

    Eine Liste der bestehenden URL-Gruppen wird angezeigt.

  2. Klicken Sie auf Neu.

    Der Dialog Neue URL-Gruppe wird geöffnet.

  3. Geben Sie einen Namen und eine Beschreibung für die URL-Gruppe ein.

  4. Klicken Sie auf Übernehmen.

    Die neue URL-Gruppe wird angelegt. Sie können dieser Gruppe nun Kurz-URLs zuordnen.

Kurz-URLs zu URL-Gruppen zuordnen

Um Kurz-URLs zu URL-Gruppen zuzuordnen:

  1. Öffnen Sie das Modul URL-Gruppen-Verwaltung.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie dann URL-Gruppen.

    Eine Liste der bestehenden URL-Gruppen wird angezeigt.

  2. Klicken Sie auf die URL-Gruppe, zu der Sie Kurz-URLs zuordnen wollen und öffnen Sie das Register Verweise.

    Eine Liste mit Kurz-URLs, die dieser URL-Gruppe bereits zugeordnet wurden, wird angezeigt.

  3. Klicken Sie auf Neu.

    Eine Liste der für diesen Management-Kontext definierten Kurz-URLs wird angezeigt.

  4. Markieren Sie die Kontrollkästchen für die Kurz-URLs, die Sie der URL-Gruppe zuordnen wollen, und klicken Sie auf Zuordnen.

    Die gewählten Kurz-URLs werden zugeordnet und Sie werden auf das URL-Register der Detailseite der URL-Gruppe zurückgeleitet.

URLs oder URL-Gruppen exportieren

Um Kurz-URLs oder URL-Gruppen zu exportieren:

  1. Öffnen Sie das Modul Kurz-URLs oder URL-Gruppen-Verwaltung.

    Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | URL-Verwaltung. Klicken Sie dann Kurz-URLs bzw. URL-Gruppen.

    Eine Liste der bestehenden Kurz-URLs bzw. URL-Gruppen wird angezeigt.

  2. Markieren Sie die zu exportierenden Kurz-URLs bzw. URL-Gruppen.

  3. Klicken Sie auf Export.

  4. Geben Sie einen Namen für die Exportdatei an, und klicken Sie erneut auf Export.

    Der Exportvorgang wird sofort gestartet. Das Modul Kurz-URL Import und Export wird geöffnet und zeigt den Prozessstatus.

  5. Laden Sie die Exportdatei herunter.

    Für weitere Informationen, siehe Download von Export-Dateien .
