Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden von Benutzerverwaltern im Kontext der Organisation ausgeführt.

Commerce Management-Benutzer suchen

Es gibt zwei Suchoptionen: die einfache Suche und die erweiterte Suche. Um nach bestimmten Benutzern zu suchen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Benutzer.

    Eine Liste aller Benutzer in Ihrer Vertriebsorganisation wird angezeigt.

  2. Geben Sie den Namen oder die ID des Benutzers in das Suchdialogfeld ein.

    Aktivieren Sie die Erweiterte Suche, um auf Grundlage weiterer Benutzerangaben eine detailliertere Suche durchzuführen. Suchen Sie bspw. nach einem Zugriffsrecht oder einer Abteilung.

  3. Klicken Sie auf Suchen.

    Die Suchergebnisse werden in einer Liste angezeigt.

Commerce Management-Benutzer anlegen

Benutzerkonten werden vom Benutzerverwalter angelegt. Der Benutzerverwalter gibt die erforderlichen Benutzerdaten wie Anmeldename oder Kennwort in das System ein. Der Benutzerverwalter legt außerdem Zugriffsrechte fest und weist Abteilungsrollen zu.

Hinweis:

Stellen Sie sicher, dass Ihr Intershop Commerce Management-System einen E-Mail-Service zum Versenden der Kennwort-/Bestätigungs-E-Mail konfiguriert hat.

Um einen neuen Benutzer anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Benutzer.

    Eine Liste aller Benutzer in Ihrer Vertriebsorganisation wird angezeigt.

  2. Klicken Sie auf Neu.

    Der Dialog Neuen Benutzer anlegen wird angezeigt.

  3. Geben Sie die notwendigen Daten an.

    Erforderliche Angaben sind Vorname, Nachname, E-Mail-Adresse, Benutzer-ID und Anmeldename. Die anderen Angaben sind optional.

  4. Klicken Sie auf Übernehmen.

    Der Benutzer wird angelegt, und die anderen Register werden aktiviert.

  5. Bearbeiten Sie die Informationen in den anderen Registern wie erforderlich.

    Die verfügbaren Informationen beinhalten Attribute, Zugriffsrechte, Abteilungsrollen und Adressen.

    Speichern Sie Ihre Änderungen eines Registers, bevor Sie zum nächsten wechseln.

    Tip:

    Sobald Benutzer angelegt sind, gehören sie zum "Benutzer-Pool" der Organisation im Commerce Management. Um sie in Channels oder Applikationen verfügbar zu machen, muss ihnen explizit Zugriff auf diese Management-Kontexte gegeben werden. Für weitere Informationen, siehe Zugriffsrechte in Channels zuweisen und Applikations-Zugriffsrechte verwalten.

Benutzerinformationen des Commerce Management bearbeiten

Um einen bestehenden Benutzer zu bearbeiten:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Benutzer.

    Eine Liste aller Benutzer in Ihrer Vertriebsorganisation wird angezeigt.

    Wenn der gesuchte Benutzer nicht angezeigt wird, verwenden Sie die Suche, um den Benutzer zu finden, siehe Commerce Management-Benutzer suchen.

  2. Klicken Sie auf den Namen des Benutzers, den Sie bearbeiten wollen.

    Die Benutzer-Detailansicht wird geöffnet.

  3. Bearbeiten Sie in jedem Register die Informationen wie erforderlich.

    Die verfügbaren Informationen beinhalten Attribute, Zugriffsrechte, Abteilungsrollen und Adressen.

    Speichern Sie Ihre Änderungen eines Registers, bevor Sie zum nächsten wechseln.

Zugriffsrechte/Rollen zuordnen

Um Zugriffsrechte oder Rollen zu einem bestehenden Kunden zuzuordnen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Benutzer.

    Eine Liste aller Benutzer in Ihrer Vertriebsorganisation wird angezeigt.

    Wenn der gesuchte Benutzer nicht angezeigt wird, verwenden Sie die Suche, um den Benutzer zu finden, siehe Commerce Management-Benutzer suchen.

  2. Klicken Sie auf den Namen des Benutzers, den Sie bearbeiten wollen.

    Die Benutzer-Detailansicht wird geöffnet.

  3. Öffnen Sie das Register Zugriffsrechte oder Abteilungsrollen und bearbeiten Sie die Angaben wie erforderlich.

    Speichern Sie Ihre Änderungen eines Registers, bevor Sie zum nächsten wechseln.

    Tip:

    Eine Liste verfügbarer Zugriffsrechte finden Sie hier: Zugriffsrechte Organisation.

Commerce Management-Benutzer löschen

Um einen oder mehrere Benutzer zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Benutzer.

    Eine Liste aller Benutzer in Ihrer Vertriebsorganisation wird angezeigt.

    Wenn der gesuchte Benutzer nicht angezeigt wird, verwenden Sie die Suche, um den Benutzer zu finden, siehe Commerce Management-Benutzer suchen.

  2. Markieren Sie das Kontrollkästchen des Benutzers, der entfernt werden soll.

  3. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Die markierten Benutzer werden aus dem System entfernt.

Tip:

Ein Benutzer kann auch gelöscht werden, indem Sie im Dialog Benutzer auf Löschen klicken.

Commerce Management-Benutzer publizieren

In Datenreplizierungs-Umgebungen können Organisationsverwalter Benutzer aus dem Bearbeitungssystem in das Zielsystem publizieren. Um dies zu tun:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Benutzer.

    Eine Liste aller Benutzer in Ihrer Vertriebsorganisation wird angezeigt.

    Wenn der gesuchte Benutzer nicht angezeigt wird, verwenden Sie die Suche, um den Benutzer zu finden, siehe Commerce Management-Benutzer suchen.

  2. Markieren Sie das Kontrollkästchen des Benutzers, den Sie im Zielsystem publizieren wollen.

  3. Klicken Sie Publizieren im Zielsystem.

    Die Liste konfigurierter Zielsysteme wird angezeigt.

  4. Wählen Sie ein Zielsystem aus und klicken Sie auf Fertigstellen.

    Die gewählten Benutzer werden an das gewählte Zielsystem repliziert.

Deaktivierte Commerce Management-Benutzer entsperren

Standardmäßig sperrt Intershop Commerce Management nach 6 Anmeldeversuchen mit einem falschen Kennwort Commerce Management-Benutzerkonten für 30 Minuten. Um ein gesperrtes Benutzerkonto manuell wieder zu aktivieren:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Benutzer.

    Eine Liste aller Benutzer in Ihrer Vertriebsorganisation wird angezeigt. Bei gesperrten Benutzerkonten wird der Status "Vorübergehend deaktiviert" angezeigt.

  2. Markieren Sie das Kontrollkästchen des Benutzers, der wieder aktiviert werden soll.

  3. Klicken Sie auf Konto reaktivieren und bestätigen Sie die Operation mit OK.

    Das ausgewählte Benutzerkonto wird unmittelbar entsperrt, und der entsprechende Benutzer kann sich wieder am System anmelden.
