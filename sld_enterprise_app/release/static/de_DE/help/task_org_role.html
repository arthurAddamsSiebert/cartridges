Verwaltung von globalen Rollen

  • Globale Rolle anlegen
  • Globale Rolle bearbeiten
  • Globale Rolle entfernen
Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext der Organisation ausgeführt.

Globale Rollen sind vordefinierte Rollen, die in jeder Abteilung der Vertriebsorganisation existieren müssen, siehe Globale Rollen und lokale Rollen. Das Verwalten globaler Rollen umfasst das Anlegen, Bearbeiten und Entfernen von Rollen.

Globale Rolle anlegen

Um eine neue globale Rolle anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Globale Rollen.

    Eine Liste aller globalen Rollen wird angezeigt.

  2. Klicken Sie auf Neu.

    Der Dialog Neue Globale Rolle wird geöffnet.

  3. Geben Sie die notwendigen Daten an.

    Erforderliche Angaben sind Name und ID. Das Feld Beschreibung ist optional.

  4. Klicken Sie auf Übernehmen.

    Die globale Rolle wird angelegt und das Register Zugriffrechte wird aktiviert.

  5. Wechseln Sie zum Register Zugriffsrechte, um der neuen Rolle Zugriffsrechte zuzuordnen.

    Eine Liste verfügbarer Zugriffsrechte der Organisation, siehe Zugriffsrechte.

  6. Klicken Sie auf Übernehmen, um Ihre Änderungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Globale Rolle bearbeiten

Um eine globale Rolle zu bearbeiten:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Globale Rollen.

    Eine Liste aller globalen Rollen wird angezeigt.

  2. Klicken Sie auf die globale Rolle, die Sie bearbeiten wollen.

    Die Detailansicht der globalen Rolle wird geöffnet.

  3. Bearbeiten Sie die Informationen wie erforderlich.

    Wechseln Sie zum Register Zugriffsrechte, um der Rolle Zugriffsrechte zuzuordnen oder zu entziehen. Eine Liste verfügbarer Zugriffsrechte der Organisation, siehe Zugriffsrechte.

  4. Klicken Sie auf Übernehmen, um Ihre Änderungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Globale Rolle entfernen

Um eine oder mehrere globale Rollen zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Globale Rollen.

    Eine Liste aller globalen Rollen wird angezeigt.

  2. Markieren Sie das Kontrollkästchen der globalen Rolle, die entfernt werden soll.

  3. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Die markierten globalen Rollen werden aus dem System entfernt.

Tip:

Eine globale Rolle kann auch gelöscht werden, indem Sie in der Detailansicht der globale Rolle auf Löschen klicken.
