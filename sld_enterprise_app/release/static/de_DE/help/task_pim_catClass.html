Klassifikationskatalog anlegen

Um einen Klassifikationskatalog anzulegen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie im Abschnitt Klassifikationskataloge auf Neu.

    Der Dialog Klassifikationskatalog wird angezeigt.

  3. Wählen Sie aus der Dropdown-Liste die Locale (Sprachumgebung) aus, für die der Katalog angelegt werden soll.

    Sie sollten immer zuerst die Standard-Locale des Systems auswählen. Auf diese Locale wird zurückgegriffen, wenn eine Locale, die ein Benutzer zum Katalogdurchsuchen ausgewählt hat, vom System nicht unterstützt wird. Wenn Sie nicht wissen, welche die Standard-Locale Ihres Systems ist, wenden Sie sich an Ihren zentralen Administrator.

  4. Geben Sie einen Namen, eine Katalog-ID und eine Beschreibung in die entsprechenden Felder ein.

    Das Feld Beschreibung ist optional.

  5. Klicken Sie auf Übernehmen.

    Der neue Katalog wird angelegt.

    Der Katalog kann bearbeitet werden, wenn Sie in der Katalog-Listenansicht auf Bearbeiten klicken.

Klassifikationskatalog bearbeiten

Um einen Klassifikationskatalog zu bearbeiten:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf Bearbeiten neben dem Katalog, den Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird angezeigt.

Klassifikationskatalog online stellen

Kataloge können online oder offline sein. Ein Offline-Katalog ist außer für Katalogverwalter für keinen Benutzer sichtbar. Um einen Katalog online oder offline zu setzen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf Bearbeiten neben dem Katalog, den Sie online oder offline setzen wollen.

    Der Dialog Standardkatalog wird angezeigt.

  3. Öffnen Sie das Register Online/Offline.

  4. Setzen Sie im Feld Status "Online" oder "Offline" als Wert.

  5. Geben Sie die Gültigkeit des ausgewählten Status an.

    Die Gültigkeit legt fest, wann der gewählte Status auf den Katalog angewendet wird. Wählen Sie eine der zwei Optionen "Beschränkter Zeitraum" oder "Immer". Wenn Sie "Beschränkter Zeitraum" wählen, müssen Sie eine Start- und eine Endzeit mit genauem Datum (dd-MM-yyyy) und genauer Zeit (HH:mm), getrennt durch Komma und Leerzeichen, festlegen, z. B.: 01-01-2003, 20:30.

Klassifikationskatalog löschen

Um einen Katalog zu löschen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf Löschen neben dem Katalog oder der Kategorie, die Sie löschen wollen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  3. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Der Katalog oder die Kategorie wird aus dem System entfernt.

eCl@ss- oder UN/SPSC-Kategorien anzeigen

Um eCl@ss- oder UN/SPSC-Kategorien in einem Klassifikationskatalog anzuzeigen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf den Katalog mit den Kategorien, die Sie anzeigen wollen.

    Eine Liste aller Kategorien der obersten Ebene wird angezeigt.

  3. Klicken Sie auf die Kategorien, um die Unterkategorien der nächsten Ebene anzuzeigen.

    Klicken Sie weiter auf die Unterkategorien, bis die Kategorie aufgelistet wird, die Sie anzeigen oder bearbeiten wollen.

Hinweis:

Klicken Sie auf Leere Klassifikationen ausblenden bzw. Leere Klassifikationen anzeigen, um Kategorien, denen bisher keine Produkte zugeordnet sind, auszublenden bzw. anzuzeigen.

Klassifikationskatalog-Kategorien anlegen

Um Katalogkategorien in einem benutzerdefinierten Klassifikationskatalog anzulegen:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf Neu.

    Der Dialog Kategorie wird angezeigt.

  3. Tragen Sie in die Felder Name, Kategorie-ID und Beschreibung die notwendigen Angaben ein.

    Das Feld Beschreibung ist optional.

  4. Klicken Sie auf Übernehmen.

    Alle anderen Register werden aktiviert.

Klassifikationskatalog-Kategorien bearbeiten

Um Kategorien in Klassifikationskatalogen zu bearbeiten:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf Bearbeiten neben der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Kategorie wird angezeigt.

  3. Bearbeiten Sie oder fügen Sie die erforderlichen Angaben in die vorhandenen Register ein.

Klassifikationskatalog-Kategorien löschen

Um Kategorien in Klassifikationskatalogen zu löschen:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf Löschen neben der Kategorie, die Sie löschen wollen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  3. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Die Kategorie wird aus dem System entfernt.

Hinweis:

eCl@ss-Kategorien können nicht gelöscht werden.

Klassifikationskatalog-Kategorien kopieren

Sie können Kopien von Katalogkategorien erzeugen. Dem Namen der Kopie ist "Copy of" vorangestellt (in lokalisierten Systemen werden entsprechende Übersetzungen verwendet). Das System erzeugt automatisch eine ID für die neue Kategorie. Die neue Kategorie ist offline.

  • Alle allgemeinen und benutzerdefinierten Attribute werden von der Originalkategorie in die kopierte Kategorie übertragen, einschließlich der Stammkategorie.

  • Produktzuordnungen zur Kategorie bleiben erhalten. Hinweis: Die Produkte selbst werden nicht kopiert.

Hinweis:

Beim Kopieren einer Katalogkategorie werden die Unterkategorien nicht mit übernommen.

Um eine Katalogkategorie zu kopieren:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf Kopieren.

    Die neue Kategorie wird in der Kategorie-Listenansicht angezeigt.

Sie können eine Kategorie auch aus ihrer Detailansicht heraus kopieren.

Klassifikationsattribute verwalten

Für Kategorien in benutzerdefinierten Klassifikationskatalogen können Klassifikationsattribute festgelegt werden, um die Attribute der diesen Kategorien zugeordneten Produkte vorzudefinieren. Beim Zuordnen von Produkten in Klassifikationskategorien mit Klassifikationsattributen werden diese Attribute den Produktverwaltern in einer Liste angeboten. Hinweis: Der Produktverwalter muss Attributwerte festlegen, bevor die Attribute auf die Produkte übertragen werden. Nach der Übertragung auf ein Produkt sind die Attribute über das Register Attribute in der Produkt-Detailansicht verfügbar.

Um Klassifikationsattribute zu verwalten:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf Bearbeiten neben der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Kategorie wird angezeigt.

  3. Öffnen Sie das Register Klassifikationsattribute.

    Eine Liste der bereits für diese Kategorie definierten Attribute wird angezeigt. Klicken Sie auf ein Attribut, um es zu bearbeiten.

  4. Klicken Sie auf Neu.

    Der Dialog Neues Klassifikationsattribut wird angezeigt.

  5. Tragen Sie in die Felder Name, Kategorie-ID und Beschreibung die notwendigen Angaben ein.

    Das Feld Beschreibung ist optional.

  6. Legen Sie den Datentyp, den Wertebereich und einen Standardwert fest.

    Der Standard-Datentyp ist "String". Zusätzlich sind "Double" und "Integer" als Datentypen verfügbar. Wenn Sie mehrere Attributwerte vorgeben wollen, tragen Sie diese im Feld Wertebereich ein. Jeder mögliche Wert muss explizit angegeben werden; mehrere Werte müssen durch | (Verkettungszeichen) getrennt werden. Diese Werte werden dann den Produktverwaltern in einer Dropdown-Liste angeboten. Schließlich kann ein Standardwert festgelegt werden, der gelten soll, wenn keine andere Auswahl getroffen wurde.

  7. Mit Klick auf Übernehmen werden Ihre Eingaben wirksam.

Sortierattribute verwalten

Für die Produktsortierung können allgemeine Sortiereinstellungen vorgenommen werden, die für alle Kataloge des Channels gelten. Für diese allgemeinen Sortiereinstellungen können Sie ein kategoriespezifische Sortierattribut festlegen, das die allgemeine Einstellung überschreibt. Um Sortiereinstellungen für eine Katalogkategorie zu verwalten:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf Bearbeiten neben der Kategorie, die Sie bearbeiten wollen.

  3. Klicken Sie auf Bearbeiten neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Das allgemeine Register der Detailseite für Katalog/Kategorie wird mit einem Abschnitt für die Sortierung unten angezeigt.

  4. Markieren Sie das Optionsfeld Benutzerdefiniert, um die allgemeine Sortiereinstellung zu überschreiben.

  5. Wählen Sie ein Standardattribut für die Sortierung (Produktname, Produktpreis) oder geben Sie ein spezielles Produktattribut an.

    Wählen Sie die Option keine, wenn Sie die allgemeine Sortiereinstellung des Channels für diese Kategorie außer Kraft setzen wollen, ohne eine kategoriespezifische Festlegung zu treffen.

  6. Legen Sie die Sortierrichtung fest.

    Verfügbare Optionen sind aufsteigend und absteigend.

  7. Klicken Sie auf Übernehmen, um die Zuordnung zu starten.

Hinweis:

Kategoriespezifische Sortiereinstellungen gelten nur für die Kategorie, für die sie definiert wurden. Sie werden nicht an Unterkategorien vererbt.

Kategorie-/Produktverweise verwalten

Kategorie- und Produktverweise verbinden eine Katalogkategorie mit bestimmten Produkten oder anderen Katalogkategorien. Kategorie- und Produktverweise können verschiedene Beziehungen beschreiben und für verschiedene Zwecke verwendet werden. Diese Unterschiede werden durch die verschiedenen Verweistypen deutlich. Weitere Informationen zu Verweistypen, siehe Kategorie- und Produktverweise. Kategorie- und Produktverweise können manuell sortiert werden.

Kategorie-/Produktverweise festlegen

Um Kategorie- oder Produktverweise festzulegen:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf Bearbeiten neben der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird geöffnet.

  3. Öffnen Sie das Register Verweise.

    Es werden bereits definierte Kategorie- und Produktverweise angezeigt. Die Listen können nach Verweistyp gefiltert werden.

  4. Klicken Sie im gewünschten Abschnitt (Produktverweise|Kategorieverweise) auf Zuordnen.

    Der Dialog Verweistyp wird angezeigt.

  5. Wählen Sie einen Verweistyp (Optionsfeld) aus, und klicken Sie auf Weiter.

    Für Produktverweise wird die Produkt-Listenansicht angezeigt. Kontrollkästchen für Produkte, zu denen bereits ein Verweis definiert wurde, sind ausgegraut.

    Für Kategorieverweise werden die verfügbaren Kataloge angezeigt. Wählen Sie einen Katalog und klicken Sie auf Weiter, um verfügbare Kategorien anzuzeigen. Kontrollkästchen für Kategorien, zu denen bereits ein Verweis definiert wurde, sind ausgegraut.

  6. Ermitteln Sie das Produkt oder die Kategorie, die als Verweisziel gelten soll, wählen Sie die entsprechenden Kontrollkästchen und klicken Sie OK.

    Die gewählten Kategorien oder Produkte werden nun im entsprechenden Abschnitt (Produktverweise|Kategorieverweise) im Register Verweise des Dialogs Standardkatalog angezeigt.

Kategorie-/Produktverweise sortieren

Um Kategorie- oder Produktverweise zu sortieren:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf Bearbeiten neben der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird geöffnet.

  3. Öffnen Sie das Register Verweise.

    Es werden bereits definierte Kategorie- und Produktverweise angezeigt.

  4. Markieren Sie die Kontrollkästchen für die Verweise, die Sie manuell sortieren wollen.

  5. Klicken Sie auf die Pfeile, um die Produktverweise neu anzuordnen.

    Die neu sortierte Ansicht wird sofort geändert.
