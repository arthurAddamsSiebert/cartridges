Label anlegen

Um ein neues Label anzulegen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Massendatenverwaltung | Label.

    Alle Label für den gewählten Management-Kontext werden angezeigt.

  2. Klicken Sie auf Neu.

    Die Detailansicht des Labels wird angezeigt.

  3. Geben Sie einen Label-Namen und eine ID an.

    Optional können Sie eine Beschreibung angeben.

  4. Klicken Sie auf Übernehmen.

    Das neue Label wird angelegt. Alle anderen Register werden aktiviert.

Objekte mit Labeln anzeigen

Um alle Objekte, die über ein Label referenziert werden, anzuzeigen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Massendatenverwaltung | Label.

    Alle Label für den gewählten Management-Kontext werden angezeigt.

  2. Wählen sie aus der Liste das Label aus, dessen Objekte Sie anzeigen wollen.

    Die Detailansicht des Labels wird angezeigt.

  3. Öffnen Sie das Register Objekte.

    Die Liste zeigt alle derzeit diesem Label zugeordneten Objekte.

Ein Label mehreren Produkten zuordnen

Um ein Label mehreren Produkten gleichzeitig zuzuordnen:

  1. Suchen Sie die Produkte, die Sie bearbeiten wollen.

    Markieren Sie die Kontrollkästchen der Produkte in der Listenansicht, und klicken Sie auf Auswahl bearbeiten. Alternativ können Sie eine Suche durchführen (siehe Produkte finden) und auf Alles bearbeiten klicken, um das gesamte Suchergebnis zu bearbeiten.

  2. Markieren Sie die Aktion Label zuordnen, und klicken Sie auf Weiter.

    Eine Liste der bestehenden Label wird angezeigt.

  3. Wählen Sie ein Label aus der Liste, um ein bestehendes Label zuzuordnen, oder geben Sie Namen und ID für ein neues Label ein, un weisen Sie es dem gegenwärtigen Produkt zu.

  4. Klicken Sie auf Fertigstellen.

    Das Label wird den ausgewählten Produkten zugewiesen.

Label einem Katalog / einer Kategorie zuweisen

Um ein Label einem Katalog / einer Kategorie zuzuordnen:

  1. Navigieren Sie zum Katalog oder zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

  3. Öffnen Sie das Register Label.

    Die aktuell zugewiesenen Label werden angezeigt.

  4. Klicken Sie auf Neu, um eine neue Label-Zuordnung zu definieren.

    Eine Liste der verfügbaren Label wird angezeigt.

  5. Wählen Sie ein Label aus der Liste, um ein bestehendes Label zuzuordnen, oder geben Sie Namen und ID für ein neues Label ein, un weisen Sie es dem gegenwärtigen Produkt zu.

  6. Klicken Sie auf OK.

    Das Label wird dem Katalog / der Kategorie zugewiesen.

Label-Katalog/Kategorie-Zuordnung aufheben

Um eine Label-Katalog/Kategorie-Zuordnung aufzuheben:

  1. Navigieren Sie zum Katalog oder zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

  3. Öffnen Sie das Register Label.

    Die aktuell zugewiesenen Label werden angezeigt.

  4. Wählen Sie das Label, dessen Zuordnung aufgehoben werden soll.

  5. Klicken Sie auf Aufheben und bestätigen Sie mit Ok.

    Die Label-Katalog/Kategorie-Zuordnung wird aufgehoben, das Label bleibt jedoch verfügbar.

Aktionen für Objekte mit Labeln durchführen

Um eine Aktion mit Objekten mit Labeln auszuführen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox, wählen Sie dann Massendatenverwaltung | Label.

    Alle Label für den gewählten Management-Kontext werden angezeigt.

  2. Wählen Sie das gewünschte Label aus.

    Markieren Sie mit dem Kontrollkästchen das Label, das die Objekte definiert, auf die die Aktion angewendet werden soll.

  3. Klicken Sie in der Zeile des markierten Labels in der Spalte Aktion auf Alle bearbeiten.

    Die Schaltfläche ist nur dann verfügbar, wenn das Label Objekten zugeordnet ist.

  4. Wählen sie eine Aktion aus.

    Die Auswahl der auf dieser Seite verfügbaren Aktionen hängt vom Typ der Objekte ab, die das Label referenziert. Für weitere Informationen, siehe Label-basierte Massendaten-Operationen.

  5. Klicken Sie auf Weiter.

    Die weiteren Schritte hängen von der hier gewählten Operation ab. Um z.B. einen Export zu starten, müssen Sie einen Namen für die Exportdatei angeben.

  6. Geben Sie auf diesen Seiten alle notwendigen Parameter an; klicken Sie dann auf Fertigstellen, um die Aktion zu starten.

    Damit wird der Prozess für die ausgewählten Objekte ausgeführt.
