Applikation: Designverwaltung

  • Modul Design öffnen
  • Design-Archivdatei laden
  • Designpaket installieren
  • Designpaket deinstallieren
  • Designpaket entfernen

Die Design-Verwaltung wird von Channel-Verwaltern als Teil der Applikationsverwaltung ausgeführt.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines s ausgeführt.

Modul Design öffnen

Um das Modul Design für eine Appliaktion zu öffnen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox.

    Die Navigationsleiste für den Channel wird geöffnet.

  2. Klicken Sie in der Navigationsleiste auf Applikationen.

    Eine Liste aller Applikationen in diesem Channel wird angezeigt.

  3. Klicken Sie auf die Applikation (Name oder ID), die Sie bearbeiten wollen.

    Das Register Allgemein der Detailansicht wird geöffnet.

  4. Öffnen Sie das Register Design.

    Eine Liste der bereits geladenen Designpakete wird angezeigt.

Design-Archivdatei laden

Durch das Laden einer Design-Archivdatei wird ein neues Designpaket erzeugt. Um eine Design-Archivdatei zu laden:

  1. Öffnen Sie das Modul Design.

    Für weitere Informationen, siehe Modul Design öffnen.

  2. Klicken Sie auf Upload.

    Der Dialog Neues Designpaket wird angezeigt.

  3. Geben Sie einen Namen für das Designpaket, eine Beschreibung sowie Pfadangabe und Name der Archivdatei an.

  4. Klicken Sie auf Übernehmen.

    Das Designpaket wird geladen und kann jetzt installiert werden.

Designpaket installieren

Um ein Designpaket zu installieren:

  1. Öffnen Sie das Modul Design.

    Für weitere Informationen, siehe Modul Design öffnen.

  2. Klicken Sie in der Zeile des Designpakets, das installiert werden soll, auf Installieren.

    Eine Liste der vorhandenen Installationen dieses Designpakets wird angezeigt.

  3. Wählen Sie im Feld Neue Installation die Locale aus, für die das Paket installiert werden soll.

    Wählen Sie aus der Dropdown-Liste eine der vorhandenen Locales aus, und klicken Sie auf Installieren.

Designpaket deinstallieren

Um ein Designpaket zu deinstallieren:

  1. Öffnen Sie das Modul Design.

    Für weitere Informationen, siehe Modul Design öffnen.

  2. Klicken Sie in der Zeile des Designpakets, das deinstalliert werden soll, auf Installieren.

    Eine Liste der vorhandenen Installationen dieses Designpakets wird angezeigt.

  3. Markieren Sie die Installationen, die Sie entfernen wollen.

    Markieren Sie die Kontrollkästchen neben den Installationen, die Sie löschen wollen.

  4. Klicken Sie auf Löschen.

    Die Installation wird entfernt.

Designpaket entfernen

Um ein Designpaket von einer Applikation zu entfernen:

  1. Öffnen Sie das Modul Design.

    Für weitere Informationen, siehe Modul Design öffnen.

  2. Markieren Sie das Kontrollkästchen für das Designpaket, das Sie löschen wollen.

  3. Klicken Sie auf Löschen und bestätigen Sie das Löschen.

    Das Designpaket wird aus dem System entfernt.
