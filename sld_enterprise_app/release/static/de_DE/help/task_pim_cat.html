Kataloge anlegen

Um einen Standardkatalog anzulegen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie im Abschnitt Standardkataloge auf Neu.

    Der Dialog Standardkatalog wird angezeigt.

  3. Wählen Sie aus der Dropdown-Liste die Locale (Sprachumgebung) aus, für die der Katalog angelegt werden soll.

    Sie sollten immer zuerst die Standard-Locale des Systems auswählen. Auf diese Locale wird zurückgegriffen, wenn eine Locale, die ein Benutzer zum Katalogdurchsuchen ausgewählt hat, vom System nicht unterstützt wird. Wenn Sie nicht wissen, welche die Standard-Locale Ihres Systems ist, wenden Sie sich an Ihren zentralen Administrator.

  4. Geben Sie einen Namen, eine Katalog-ID und eine Beschreibung in die entsprechenden Felder ein.

    Das Feld Beschreibung ist optional.

  5. Klicken Sie Übernehmen.

    Der neue Katalog wird angelegt.

    Der Katalog kann bearbeitet werden, wenn Sie in der Katalog-Listenansicht auf Bearbeiten klicken.

Kataloge bearbeiten

Um einen Standardkatalog zu bearbeiten:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird angezeigt.

    Kataloge oder Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, können nicht bearbeitet, sondern nur angezeigt werden ().

Katalog-Channel-Zuweisungen verwalten

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext der Organisation oder eines Partner Channels ausgeführt.

Kataloge Channels zuordnen

Um einen Katalog einem Channel zuzuordnen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird angezeigt.

  3. Öffnen Sie das Register Channels.

    Eine Liste aller Channel-Zuordnungen des aktuellen Katalogs wird angezeigt.

  4. Klicken Sie Neu.

    Eine Liste aller verfügbaren Channels, denen der aktuelle Katalog nicht zugeordnet ist, wird angezeigt.

  5. Markieren Sie den entsprechenden Channel und klicken Sie auf OK.

    Markieren Sie das Kontrollkästchen neben dem Channel, dem Sie den aktuellen Katalog zuordnen wollen, und klicken Sie auf OK.

Zuordnung von Kataloganzeigen aufheben

Um die Zuordnung eines Katalogs zu einem Channel zu entfernen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird angezeigt.

  3. Öffnen Sie das Register Channels.

    Eine Liste aller Channel-Zuordnungen des aktuellen Katalogs wird angezeigt.

  4. Markieren Sie die zu entfernenden Zuordnungen, und klicken Sie auf Löschen.

    Markieren Sie das Kontrollkästchen neben der Zuordnung, die Sie entfernen wollen, und klicken Sie auf Löschen.

Katalog online setzen

Kataloge können online oder offline sein. Ein Offline-Katalog ist außer für Katalogverwalter für keinen Benutzer sichtbar. Um einen Katalog online oder offline zu setzen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird geöffnet.

    Kataloge oder Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, können nicht bearbeitet werden, lassen sich jedoch online bzw. offline setzen. Klicken Sie dazu auf das Symbol , um die Detailansicht zu öffnen.

  3. Öffnen Sie das Register Online/Offline.

  4. Setzen Sie im Feld Status "Online" oder "Offline" als Wert.

  5. Geben Sie die Gültigkeit des ausgewählten Status an.

    Die Gültigkeit legt fest, wann der gewählte Status auf den Katalog angewendet wird. Wählen Sie eine der zwei Optionen "Beschränkter Zeitraum" oder "Immer". Wenn Sie einen beschränkten Zeitraum wählen, müssen Sie eine Start- und eine Endzeit mit genauem Datum und genauer Zeit in den entsprechenden Feldern angeben.

Kataloge löschen

Um einen Katalog zu löschen:

  1. Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge.

    Die Katalog-Listenansicht wird angezeigt.

  2. Klicken Sie auf Löschen neben dem Katalog oder der Kategorie, die Sie löschen wollen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  3. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Der Katalog oder die Kategorie wird aus dem System entfernt.

Katalogkategorien anlegen

Um Katalogkategorien anzulegen:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie Neu.

    Der Dialog Kategorie wird angezeigt.

  3. Tragen Sie in die Felder Name, Kategorie-ID und Beschreibung die notwendigen Angaben ein.

    Das Feld Beschreibung ist optional.

  4. Klicken Sie Übernehmen.

    Alle anderen Register werden aktiviert.

Katalogkategorien bearbeiten

Um Katalogkategorien zu bearbeiten:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf das Symbol in der Zeile der Kategorie, die Sie bearbeiten wollen.

    Die Kategorie-Detailansicht wird geöffnet.

    Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, können nicht bearbeitet, sondern nur angezeigt werden ().

  3. Bearbeiten Sie oder fügen Sie die erforderlichen Angaben in die vorhandenen Register ein.

Katalogkategorien löschen

Um Katalogkategorien zu löschen:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf Löschen neben der Kategorie, die Sie löschen wollen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  3. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Die Kategorie wird aus dem System entfernt.

Katalogkategorien kopieren

Sie können Kopien von Katalogkategorien erzeugen. Dem Namen der Kopie ist "Copy of" vorangestellt (in lokalisierten Systemen werden entsprechende Übersetzungen verwendet). Das System erzeugt automatisch eine ID für die neue Kategorie. Die neue Kategorie ist offline.

  • Alle allgemeinen und benutzerdefinierten Attribute werden von der Originalkategorie in die kopierte Kategorie übertragen, einschließlich der Stammkategorie.

  • Produktzuordnungen zur Kategorie bleiben erhalten. Hinweis: Die Produkte selbst werden nicht kopiert.

Hinweis:

Beim Kopieren einer Katalogkategorie werden die Unterkategorien nicht mit übernommen.

Um eine Katalogkategorie zu kopieren:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Wählen Sie die gewünschte Kategorie.

    Markieren Sie das Kontrollkästchen der Kategorie, die Sie kopieren wollen.

  3. Klicken Sie auf Kopieren.

    Die neue Kategorie wird in der Kategorie-Listenansicht angezeigt.

Sie können eine Kategorie auch aus ihrer Detailansicht heraus kopieren.

Katalogkategorien publizieren

In Datenreplizierungs-Umgebungen können Katalogverwalter Katalogkategorien aus dem Bearbeitungssystem in das Zielsystem detailliert veröffentlichen. Um dies zu tun:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Wählen Sie die gewünschte Kategorie.

    Markieren Sie das Kontrollkästchen der Kategorie, die Sie ins Zielsystem publizieren wollen.

  3. Klicken Sie Publizieren im Zielsystem.

    Die Liste konfigurierter Zielsysteme wird angezeigt.

  4. Wählen Sie ein Zielsystem aus und klicken Sie auf Fertigstellen.

    Die Kategorie wird an das gewählte Zielsystem repliziert.

Katalog-Content-Ansichten verwalten

Jede Katalogkategorie kann zu einer oder mehreren Content-Ansichten, d.h. Seiten oder Includes, zugeordnet werden. Im Allgemeinen sind Seiten und Includes abhängig vom Content Management, siehe Content-Management: Begriffe. Zuordnungen von Seiten und Includes werden von Katalogverwaltern über das Register Content des Kategorie-Dialogs verwaltet.

Das Register Content ist in zwei Abschnitte geteilt:

  • Seiten: listet die Kategorieseiten der aktuellen Kategorie auf.

  • Includes: listet die Includes der aktuellen Kategorie auf.

Kategorien/Includes zu Seiten zuordnen

Um Seiten oder Includes einer Kategorie zuzuordnen:

  1. Navigieren Sie zur Katalog- oder zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird geöffnet.

    Kataloge oder Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, können nicht bearbeitet, sondern nur angezeigt werden ().

  3. Öffnen Sie das Register Content.

    Es werden alle Content-Ansichten (Seiten/Includes) angezeigt, die dieser Kategorie angehören.

  4. Klicken Sie im gewünschten Bereich (Seiten/Includes) auf Neu.

    Die verfügbaren Seitentypen werden angezeigt.

  5. Wählen Sie den gewünschten Content-Typen und klicken Sie auf Übernehmen.

    Die Seite/Include des gewählten Typs wird der Kategorie zugeordnet. Sie werden auf das Register Content zurückgeleitet, welches die aktualisierte Content-Liste anzeigt.

Kategorieseiten/Includes bearbeiten

Um einer Kategorie zugehörige Seiten oder Includes zu bearbeiten:

  1. Navigieren Sie zur Katalog- oder zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird geöffnet.

    Kataloge oder Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, können nicht bearbeitet, sondern nur angezeigt werden ().

  3. Öffnen Sie das Register Content.

    Es werden alle Content-Ansichten (Seiten/Includes) angezeigt, die dieser Kategorie angehören.

  4. Klicken Sie auf den Namen der Seite/Include, die Sie bearbeiten wollen.

    Der Dialog zum Bearbeiten des Content wird geöffnet.

    Für Seiten können Sie bestehende Seitenvarianten zuordnen oder neue Seitenvarianten erzeugen. Für weitere Informationen siehe Seitenvarianten zu Seite hinzufügen.

    Für Includes können Sie bestehende Komponenten zuordnen oder neue Komponenten erzeugen. Für weitere Informationen siehe Komponenten zu Includes hinzufügen.

    Um zum Register Content des Dialogs Kategorie zurückzukehren, klicken Sie Zurück zur Liste.

Kategorieseiten/Includes löschen

Um Kategorieseiten oder Includes zu löschen:

  1. Navigieren Sie zur Katalog- oder zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird geöffnet.

    Kataloge oder Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, können nicht bearbeitet, sondern nur angezeigt werden ().

  3. Öffnen Sie das Register Content.

    Es werden alle Content-Ansichten (Seiten/Includes) angezeigt, die dieser Kategorie angehören.

  4. Wählen Sie die Content-Ansicht, die entfernt werden soll.

    Im entsprechenden Abschnitt wählen Sie das/die Kontrollkästchen der Seiten/Includes, die Sie löschen wollen.

  5. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Die gewählten Content-Ansichten werden entfernt.

Produkte in Katalogkategorien verwalten

Kategorieseiten mit Produkten zeigen eine Liste der zugeordneten Produkte. Diese Produktlisten erlauben Zugriff auf eine Reihe von Funktionen zum Bearbeiten von Produkten.

Produkte in Katalogkategorien bearbeiten

Bestimmte Operationen in der Produktverwaltung können mittels Stapelprozessen mit mehreren Produkte gleichzeitig ausgeführt werden. Folgende Aufgaben sind möglich:

  • Standard-Produktattribute aktualisieren (Status, Verfügbarkeit, Bestellmenge, ...)

  • Benutzerdefiniertes Produktattribut aktualisieren/anlegen

  • Benutzerdefiniertes Produktattribut löschen

  • Produkte Katalogkategorien zuordnen

  • Produkte Klassifikationskatalogkategorie zuordnen

  • Kategorie-/Produktverweise zuordnen

  • Produkt-/Kategorieverweise aufheben

  • Produkte kopieren

  • Produkte löschen

  • Label zuordnen

  • Produkte in einen anderen Intershop Commerce Management-Cluster replizieren

  • Export in CSV

Hinweis:

Für das Bearbeiten mit Stapelprozessen müssen die Produkte nicht gesperrt werden. Produkte, die beim Ausführen der Stapelprozesse von anderen Benutzern gesperrt sind, werden nicht geändert. Stellen Sie deshalb sicher, dass die in den Stapelprozess einbezogenen Produkte entsperrt sind.

Um Produkte in Stapelprozessen zu bearbeiten:

  1. Wählen Sie die Produkte aus, die Sie bearbeiten wollen.

    Markieren Sie die Kontrollkästchen der gewünschten Produkte in der Listenansicht, und klicken Sie auf Auswahl bearbeiten. Alternativ können Sie auf Alle bearbeiten klicken, um Ihre Änderungen auf alle Produkte in der aktuellen Kategorie anzuwenden.

  2. Markieren Sie eine Aktion, und klicken Sie auf Weiter.

    Die weiteren Schritte hängen von der hier gewählten Operation ab.

Hinweis:

Stapelprozesse sind nicht für die Verarbeitung von Massendaten vorgesehen. Für die Aktualisierung großer Datenmengen empfiehlt Intershop die Verwendung der Importfunktion.

Hinweis:

In Abhängigkeit von Typ und Konfiguration des Intershop Commerce Management-Systems und der Menge an Daten, die geändert werden sollen, kann eine Bulk-Operation (z. B. Stapelprozesse, Import- und Exportptozesse, Auditberichte) negative Effekte für die Systemleistung bzw. die Datenkonsistenz haben. Kontaktieren Sie Ihren Intershop Commerce Management-Systemadministrator, bevor Sie so eine Bulk-Operation auslösen.

Produkte zuordnen

Um Produkte zur aktuellen Kategorie zuzuordnen:

  1. Klicken Sie auf Hinzufügen.

    Alle Produkte im Produktbestand werden angezeigt.

  2. Wählen Sie die Produkte aus, die Sie dieser Kategorie zuordnen wollen.

    Sie können durch die Liste blättern oder eine Suche ausführen, um die Auswahl einzuschränken. Markieren Sie die Kontrollkästchen der gewünschten Produkte.

  3. Klicken Sie auf OK, um die ausgewählten Produkte der aktuellen Kategorie zuzuordnen.

    Anderenfalls klicken Sie auf Abbrechen, um die Änderungen zu verwerfen.

Tip:

Sie können Produkte auch einzeln Katalogkategorien zuordnen. Für weitere Informationen siehe Katalogkategorie-Zuordnungen verwalten.

Hinweis:

Für die Änderung von Produktzuordnungen in Commerce Management muss der vorkonfigurierte Job "Update Product Assignments" ausgeführt werden. Wenden Sie sich für das Planen bzw. Ausführen des Jobs an Ihren Intershop Commerce Management-Systemadministrator.

Produktzuordnungen aufheben

Um Zuordnungen von Produkten zur aktuellen Kategorie aufzuheben:

  1. Wählen Sie die Produkte aus, deren Zuordnung Sie aufheben wollen.

    Markieren Sie die Kontrollkästchen der gewünschten Produkte in der Listenansicht.

  2. Klicken Sie auf Aufheben und bestätigen Sie die Operation.

    Die Zuordnung der Produkte zur aktuellen Kategorie wird entfernt.

    Tip:

    Mit Klick auf Alle Aufheben lassen sich alle Produktzuordnungen zu dieser Kategorie aufheben, ohne einzelne Produkte auswählen zu müssen.

Tip:

Sie können die Zuordnung von Produkten zu Katalogkategorien auch einzeln aufheben. Für weitere Informationen siehe Katalogkategorie-Zuordnungen verwalten.

Hinweis:

Für die Änderung von Produktzuordnungen in Commerce Management muss der vorkonfigurierte Job "Update Product Assignments" ausgeführt werden. Wenden Sie sich für das Planen bzw. Ausführen des Jobs an Ihren Intershop Commerce Management-Systemadministrator.

Produkte mithilfe von Filtern zuordnen

Produkte können Kategorien dynamisch mithilfe von Produktfiltern zugeordnet werden, siehe Vordefinierte Produktfilter.

Hinweis:

Produktfilter benötigen eine integrierte Suchmaschine dritter Anbieter, siehe Suchindex-Services.

Um dynamische Produkt-Kategorie-Zuordnungen zu verwalten:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf das Symbol in der Zeile der Kategorie, die Sie bearbeiten wollen.

    Die Kategorie-Detailansicht wird geöffnet.

    Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, können nicht bearbeitet, sondern nur angezeigt werden ().

  3. Wählen Sie im Abschnitt Produktzuordnung im allgemeinen Register Produkte dynamisch zuordnen.

    Dies öffnet den Bearbeitungsdialog Produktfilter wie beschrieben in Produktfilter-Regeln bearbeiten.

  4. Bearbeiten Sie für jeden Abschnitt die Regeln wie erforderlich.

    Folgende Optionen stehen zur Verfügung:

    • Kategorie

      Gibt die Katalogkategorie an, für die die Produktfilter-Suchoperation vorbehalten sein soll. Ein Klick auf Auswählen öffnet einen Assistenten zur Auswahl der gewünschten Kategorie. Ein Klick auf Leeren entfernt die Kategoriebeschränkung.

    • Filterbedingung

      Spezifiziert die Textwerte (für Textattribute) oder Wertbereiche (für numerische Attribute), die vom Filter gefunden werden sollen.

    • Sortierung

      Gibt die Sortierung der gefunden Ergebnissets an. Das Sortierungsattribut ist von der definierten Filterbedingung unabhängig.

    • Gewichtung

      Legt eine Gewichtung der spezifischen Attributwerte an.

  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Hinweis:

Für die Änderung von Produktzuordnungen in Commerce Management muss der vorkonfigurierte Job "Update Product Assignments" ausgeführt werden. Wenden Sie sich für das Planen bzw. Ausführen des Jobs an Ihren Intershop Commerce Management-Systemadministrator.

Katalog/Kategorie-Bilder verwalten

Katalogverwalter können Bilder für jeden Katalog, jede Kategorie festlegen, die in der Storefront angezeigt werden. Um dies zu tun:

  1. Navigieren Sie zur Katalog- oder zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird geöffnet.

    Kataloge oder Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, können nicht bearbeitet, sondern nur angezeigt werden ().

  3. Klicken Sie auf Auswählen neben dem Feld Bild.

    Der Dialog Bild auswählen wird geöffnet.

    Für Kataloge oder Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, ist der Auswählen Schalter nicht verfügbar.

  4. Navigieren sie zum gewünschten Content-Verzeichnis und wählen Sie ein Bild.

    Sie können auch nach einem Bild-Namen suchen oder ein neues Bild hochladen.

  5. Nachdem Sie ein Bild ausgewählt haben, klicken Sie OK.

    Das Bild wird dem aktuellen Katalog / der aktuellen Kategorie zugeordnet, der Fokus wechselt zurück zum Register Allgemein des Dialogs Standardkatalog.

Tip:

Die Zuweisung des Bilds gilt für alle Locales.

SEO-Funktionen für Kataloge and Kategorien verwalten

Das Verwalten von SEO-Funktionen für Kataloge und Kategorien beinhaltet die Pflege von angereichertem Kategorie-Content sowie die Verwaltung SEO-spezifischer Seitenattribute.

Angereicherten Inhalt für Kategorien verwalten

Katalogverwalter können angereicherten Content (Beschreibungen) für Kataloge und Kategorien hinzufügen und pflegen. Um dies zu tun:

  1. Navigieren Sie zur Katalog- oder zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  2. Klicken Sie auf das Symbol neben dem Katalog oder der Kategorie, die Sie bearbeiten wollen.

    Der Dialog Standardkatalog wird geöffnet.

    Kataloge oder Kategorien, die von einem Stamm-Management-Kontext freigegeben wurden, können nicht bearbeitet, sondern nur angezeigt werden ().

  3. Bearbeiten Sie den angereicherten Content im Content-Feld des Registers Allgemein wie erforderlich.

Kategorieebene überschreiben, indem für den Katalog oder eine bestimmte Kategorie ein spezifisches Sortierattribut festgelegt wird.

Um Sortiereinstellungen auf Katalog- oder Kategorieebene zu verwalten:

  1. Navigieren Sie zum Katalog oder zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

    Das allgemeine Register der Detailseite für Katalog/Kategorie wird mit einem Abschnitt für die Sortierung unten angezeigt.

  3. Markieren Sie das Optionsfeld Benutzerdefiniert, um die allgemeine Sortiereinstellung zu überschreiben.

  4. Wählen Sie ein Standardattribut für die Sortierung (Produktname, Produktpreis) oder geben Sie ein spezielles Produktattribut an.

Kategorie-/Produktverweise verwalten

Kategorie- und Produktverweise verbinden eine Katalogkategorie mit bestimmten Produkten oder anderen Katalogkategorien. Kategorie- und Produktverweise können verschiedene Beziehungen beschreiben und für verschiedene Zwecke verwendet werden. Diese Unterschiede werden durch die verschiedenen Verweistypen deutlich. Weitere Informationen zu Verweistypen, siehe Kategorie- und Produktverweise.

Kategorieverweise und Produktverweise werden mit dem Register Verweise verwaltet. Dieser Tab listet sowohl "ausgehende" als auch "eingehende" Links auf. Ausgehende Links definieren ein anderes Produkt oder eine Kategorie als das "Verweisziel" ("<Verweistyp> für"), eingehende Links legen die aktuelle Kategorie als "Verweisziel" eines anderen Produktes oder einer Kategorie fest ("ist <Verweistyp> von").

Die Verweistypauswahl ist in zwei Spalten unterteilt: die linke zeigt die ausgehenden Links, die rechte zeigt die eingehenden Links. Verweistyp auswählen zeigt die dazugehörige Linkliste, die die tatsächlichen Links anzeigt. Multiple Verweislisten sind nach Verweistypen gruppiert, wo ausgehende Links über den eingehenden Links desselben Typs stehen.

Ausgehende Kategorie-/Produktverweise können manuell sortiert werden.

Kategorie-/Produktverweise festlegen

Um Kategorie- oder Produktverweise festzulegen:

  1. Navigieren Sie zur Kategorieebene, auf der Sie arbeiten wollen.

    Wählen Sie den gewünschten Management-Kontext aus der Kontextauswahlbox und wählen Sie Katalog | Kataloge. Klicken Sie auf den Katalog und die Kategorien, bis das gewünschte Kategorie-Level aufgelistet wird.

  3. Öffnen Sie das Register Verweise.

    Es werden bereits definierte Kategorie- und Produktverweise angezeigt. Die Listen können nach Verweistyp gefiltert werden.

  4. Wählen Sie den beabsichtigten Verweistyp.

    Das zeigt die dazugehörige Verweisliste an.

  5. Klicken Sie Zuordnen im beabsichtigten Bereich (Produkte|Kategorien) auf der Verweisliste.

    Für Produktverweise wird die Produkt-Listenansicht angezeigt. Kontrollkästchen für Produkte, zu denen bereits ein Verweis definiert wurde, sind ausgegraut.

    Die verfügbaren Kategorien für Kategorieverweise werden in einer Baumansicht angezeigt. Klappen Sie den Baum auf, um die gewünschte Kategorie anzuzeigen. Kontrollkästchen für Kategorien, zu denen bereits ein Verweis definiert wurde, sind ausgegraut.

  6. Ermitteln Sie das Produkt oder die Kategorie, auf die verwiesen werden soll, und wählen Sie die entsprechenden Kontrollkästchen aus.

    Damit wird das Produkt oder die Kategorie ausgewählt. die als Verweisziel für ausgehende Links dienen soll oder als Ursprung für eingehende Links.

  7. Klicken Sie auf Ok für Produktverweise und klicken Sie auf Beenden für Kategorieverweise.

    Die gewählten Kategorien oder Produkte werden nun im entsprechenden Bereich (Produkte|Kategorien) der gewählten Verweistypliste angezeigt.
