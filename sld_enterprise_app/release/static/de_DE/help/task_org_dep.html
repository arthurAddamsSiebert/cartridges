Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext der Organisation ausgeführt.

Abteilungen anzeigen

Um Abteilungen und deren Details anzuzeigen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

  2. Klicken Sie auf das Pluszeichen (+) neben einer Abteilung.

    Die zugehörigen Unterabteilungen werden angezeigt.

  3. Klicken Sie einen Abteilungsnamen.

    Die Detailansicht der Abteilung wird angezeigt.

Abteilungen verwalten

Das Verwalten von Abteilungen umfasst das Anlegen, Bearbeiten und Entfernen von Abteilungen.

Abteilung anlegen

Um eine neue Abteilung anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

  2. Klicken Sie auf Neu.

    Der Dialog Neue Abteilung wird angezeigt.

  3. Tragen Sie in die Felder im Register Allgemein alle erforderlichen Angaben zu Namen und ID ein.

    Die anderen Felder sind optional.

  4. Legen Sie eine Stammabteilung fest.

    Um die neue Abteilung einer Stammabteilung zuzuordnen, tragen Sie die Abteilungs-ID in das entsprechende Feld ein. Alternativ können Sie auf Auswählen klicken und die gesuchte Stammabteilung im Dialog auswählen.

  5. Klicken Sie auf Übernehmen.

    Die neue Abteilung wird angelegt, und die anderen Register Detailansicht der Abteilung werden aktiviert.

Abteilung bearbeiten

Um eine bestehende Abteilung zu bearbeiten:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

  2. Klicken Sie auf die Abteilung, die Sie bearbeiten wollen.

  3. Bearbeiten Sie unter jedem Register die erforderlichen Angaben.

  4. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Abteilung entfernen

Um eine oder mehrere Abteilungen zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

    Klicken Sie auf das Pluszeichen (+) neben einer Abteilung, um deren Unterabteilungen anzuzeigen.

  2. Markieren Sie die Kontrollkästchen neben den Abteilungen, die Sie löschen wollen.

  3. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Die markierten Abteilungen werden aus dem System entfernt.

Tip:

Eine Abteilung kann auch in der Detailansicht der Abteilung gelöscht werden, indem Sie im Register Allgemein auf Löschen klicken.

Abteilungsbenutzer verwalten

Das Verwalten von Abteilungsbenutzern umfasst das Hinzufügen von Benutzern zu einer Abteilung sowie das Entfernen derselben.

Beachten Sie, dass das Hinzufügen und Entfernen von Benutzern von/zu Abteilungen nur die Benutzer-Abteilung-Beziehung betrifft. Die Benutzer werden nicht angelegt oder entfernt.

Abteilungs-Benutzer hinzufügen

Um Benutzer zu eine Abteilung hinzuzufügen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

    Klicken Sie auf das Pluszeichen (+) neben einer Abteilung, um deren Unterabteilungen anzuzeigen.

  2. Klicken Sie auf die Abteilung, dem Sie einen Benutzer hinzufügen wollen.

  3. Öffnen Sie das Register Benutzer.

    Alle der aktuellen Abteilung zugeordneten Benutzer werden angezeigt.

  4. Klicken Sie auf Neu.

    Eine Liste zeigt alle Benutzer dieser Organisation, die noch nicht Mitglied dieser Abteilung sind.

  5. Markieren Sie den gewünschten Benutzer und klicken Sie auf Weiter.

    Die verfügbaren Abteilungsrollen werden angezeigt.

  6. Markieren Sie die gewünschte Rolle und klicken Sie auf Fertigstellen.

    Der ausgewählte Benutzer wird dieser Abteilung hinzugefügt und ihm wird die gewählte Rolle zugeordnet.

Abteilungs-Benutzer entfernen

Um Benutzer aus einer Abteilung zu entfernen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

    Klicken Sie auf das Pluszeichen (+) neben einer Abteilung, um deren Unterabteilungen anzuzeigen.

  2. Klicken Sie auf die Abteilung, aus dem Sie einen Benutzer entfernen wollen.

  3. Öffnen Sie das Register Benutzer.

    Alle der aktuellen Abteilung zugeordneten Benutzer werden angezeigt.

  4. Markieren Sie das Kontrollkästchen des Benutzers, den Sie entfernen wollen.

  5. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Der gewählte Benutzer wird aus der Abteilung entfernt.

Abteilungsrollen verwalten

Das Verwalten von Abteilungenrollen umfasst das Anzeigen, Anlegen, Bearbeiten und Entfernen von Rollen.

Abteilungsrollen anzeigen

Um Abteilungsrollen anzuzeigen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

    Klicken Sie auf das Pluszeichen (+) neben einer Abteilung, um deren Unterabteilungen anzuzeigen.

  2. Klicken Sie auf die Abteilung, für die Sie die Rollen anzeigen wollen.

  3. Öffnen Sie das Register Rollen.

    Eine Liste aller Abteilungsrollen wird angezeigt.

Abteilungsrollen anlegen

Um eine neue Abteilungsrolle anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

    Klicken Sie auf das Pluszeichen (+) neben einer Abteilung, um deren Unterabteilungen anzuzeigen.

  2. Klicken Sie auf die Abteilung, für die Sie eine Rolle anlegen wollen.

  3. Öffnen Sie das Register Rollen.

    Eine Liste aller Abteilungsrollen wird angezeigt.

  4. Klicken Sie auf Neu.

  5. Tragen Sie die notwendigen Angaben in die Felder Name und ID ein.

    Die anderen Felder sind optional.

  6. Klicken Sie auf Übernehmen.

    Die Rolle wird angelegt.

  7. Klicken Sie auf das Register Zugriffsrechte, um der neuen Rolle Zugriffsrechte zuzuordnen.

  8. Klicken Sie auf Übernehmen, um Ihre Änderungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Abteilungsrolle bearbeiten

Um eine Abteilungsrolle zu bearbeiten:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

    Klicken Sie auf das Pluszeichen (+) neben einer Abteilung, um deren Unterabteilungen anzuzeigen.

  2. Klicken Sie auf die Abteilung, für die Sie eine Rolle anlegen wollen.

  3. Öffnen Sie das Register Rollen.

    Eine Liste aller Abteilungsrollen wird angezeigt.

  4. Klicken Sie auf die Rolle, die Sie bearbeiten wollen.

  5. Bearbeiten Sie die Felder nach Bedarf.

  6. Klicken Sie auf Übernehmen, um Ihre Änderungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Abteilungsrolle löschen

Um eine oder mehrere Abteilungsrollen zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Abteilungen.

    Die Hierarchie der Abteilungen wird in einer Baumstruktur angezeigt.

    Klicken Sie auf das Pluszeichen (+) neben einer Abteilung, um deren Unterabteilungen anzuzeigen.

  2. Klicken Sie auf die Abteilung, für die Sie eine Rolle anlegen wollen.

  3. Öffnen Sie das Register Rollen.

    Eine Liste aller Abteilungsrollen wird angezeigt.

  4. Markieren Sie die Kontrollkästchen neben den Rollen, die Sie löschen wollen.

  5. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Die markierten Rollen werden aus dem System entfernt.

Hinweis:

Abteilungsrollen, die auf globalen Rollen basieren, können nicht gelöscht werden.
