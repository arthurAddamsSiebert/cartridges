Empfehlungs-Datenfeed

  • Neue Konfiguration erzeugen
  • Empfehlungs-Datenfeed manuell ausführen
  • Empfehlungs-Datenfeed automatisch ausführen

Intershop Commerce Management kann als optionale Komponenten Recommendation-Engines integrieren. Die zugehörigen Recommendation-Daten-Feeds können als spezieller Anwendungsfall von Produktdaten-Feeds gesehen werden, siehe Produktdaten-Feeds: Konzepte. Somit ist deren Handling und Verhalten ähnlich.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Neue Konfiguration erzeugen

Um eine neue Konfiguration für einen Empfehlungs-Datenfeed zu erzeugen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Empfehlungen - Datenfeeds.

    Eine Liste aller Produkte aller verfügbaren Feed-Konfigurationen.

  2. Klicken Sie auf Neu.

    Die Seite Allgemein der neuen Konfiguration wird angezeigt.

  3. Bearbeiten Sie die allgemeinen Angaben wie erforderlich.

    Dazu gehören als notwendige Angaben Name und ID für die neue Konfiguration. Optional können Sie eine Beschreibung angeben. Um die Konfiguration zu aktivieren/deaktivieren, markieren Sie das Kontrollkästchen Aktiviert bzw. entfernen die Markierung.

  4. Geben Sie den Datenfeed-Typen an.

    Abhängig von den integrierten Empfehlungs-Services sind verschiedene Datenfeed-Typen verfügbar. Wählen Sie den Typ des gewünschten Services aus der Dropdown-Liste aus.

  5. Klicken Sie auf Übernehmen.

    Die neue Konfiguration wird erzeugt und der Abschnitt Parameter wird angezeigt.

  6. Geben Sie die Parameter wie erforderlich an.

    Die Tabelle unten listet die zusätzlichen Parameter auf.

    Tabelle 47: Parameter für Empfehlungs-Datenfeeds
    Parameter Beschreibung
    Währung Legt die Währung an, die bei angelegten Produktdaten enthalten sein soll.
    Bestellungsdaten speichern Gibt an, ob Bestellungsdaten enhalten sind. Wenn ja, geben Sie den Zeitraum an, der für die Abfrage von Bestellungsdaten verwendet werden soll.

  7. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Empfehlungs-Datenfeed manuell ausführen

Um einen Job für Empfehlungs-Datenfeeds mit einer bestimmten Konfiguration auszuführen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Empfehlungen - Datenfeeds.

    Eine Liste aller Produkte aller verfügbaren Feed-Konfigurationen.

  2. Markieren Sie die Konfiguration, die für den Empfehlungs-Datenfeed-Job verwendet werden soll, und klicken Sie auf Ausführen.

    Dadurch wird der Empfehlungs-Datenfeed-Prozess gestartet. Klicken Sie auf Aktualisieren, um den Seiteninhalt neu zu laden.

Empfehlungs-Datenfeed automatisch ausführen

Empfehlungs-Datenfeeds können nicht nur manuell, sondern auch automatisch durch einen geplanten Job gestartet werden.

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Empfehlungen - Datenfeeds.

    Eine Liste aller Produkte aller verfügbaren Feed-Konfigurationen.

  2. Klicken Sie auf den Namen der Konfiguration.

  3. Öffnen Sie das Register Zeitplan.

  4. Legen Sie das Ausführungsintervall für die aktuelle Konfiguration fest und klicken Sie dann auf Übernehmen.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.
