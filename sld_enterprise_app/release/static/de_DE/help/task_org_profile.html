Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext der Organisation ausgeführt.

Profile anzeigen und bearbeiten

Um die allgemeinen Daten im Organisationsprofil anzuzeigen und zu bearbeiten:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Organisationsprofil.

    Ihr Organisationsprofil wird angezeigt.

  2. Bearbeiten Sie die Informationen wie erforderlich.

  3. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Organisationsadressen verwalten

Die Verwaltung von Organisationsadressen beinhaltet das Anlegen, Bearbeiten und Löschen von Adressen.

Hinweis:

Jede Organisation hat mindestens eine Adresse. Diese Adresse kann nicht gelöscht werden und dient der Organisation als Kontaktadresse.

Organisationsadresse bearbeiten

Um Ihre Organisationsadresse zu bearbeiten:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Organisationsprofil.

    Ihr Organisationsprofil wird angezeigt.

  2. Öffnen Sie das Register Adressen.

    Dieses Register listet alle Adressen, die für Ihre Organisation angegeben sind, auf.

  3. Klicken Sie auf den Namen der Adresse, die Sie bearbeiten wollen.

    Die Adress-Detailansicht wird geöffnet.

  4. Bearbeiten Sie die Informationen wie erforderlich.

    Geben Sie Adress-Attribute im Register Attribute an (optional).

  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Organisationsadresse anlegen

Um eine neue Adresse für Ihre Organisation anzulegen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Organisationsprofil.

    Ihr Organisationsprofil wird angezeigt.

  2. Öffnen Sie das Register Adressen.

    Dieses Register listet alle Adressen, die für Ihre Organisation angegeben sind, auf.

  3. Klicken Sie auf Neu.

    Der Dialog Neue Adresse wird angezeigt.

  4. Fügen Sie die Informationen hinzu wie erforderlich.

    Geben Sie Adress-Attribute im Register Attribute an (optional).

  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Abbrechen, um die Änderungen zu verwerfen.

Organisationsadresse entfernen

Um eine Organisations-Adresse zu löschen:

  1. Klicken Sie in der Navigationsleiste auf Organisation | Organisationsprofil.

    Ihr Organisationsprofil wird angezeigt.

  2. Öffnen Sie das Register Adressen.

    Dieses Register listet alle Adressen, die für Ihre Organisation angegeben sind, auf.

  3. Markieren Sie das Kontrollkästchen der Adresse, die entfernt werden soll.

  4. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Die markierte Adresse wird entfernt.

Tip:

Eine Adresse kann auch gelöscht werden, indem Sie in der Adress-Detailansicht auf Löschen klicken.
