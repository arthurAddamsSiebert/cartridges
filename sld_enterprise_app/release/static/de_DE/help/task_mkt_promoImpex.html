Wie bereits erwähnt in Aktions-Import und -Export, Intershop Commerce Management kann

  • Aktionen aus XML-Dateien importieren und ihn sie exportieren,

  • Aktionscodes aus CSV-Dateien importieren und in sie exportieren sowie

  • Aktionscode-Gruppen aus XML-Dateien importieren und in sie exportieren.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Upload von Importdateien

Bevor Sie Aktion-Daten in Intershop Commerce Management importieren können, müssen Sie eine entsprechende Datei hochladen. Um eine Datei für den Import hochzuladen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Klicken Sie auf Upload im Abschnitt Dateien.

    Eine Liste der bereits für den Import geladenen Dateien wird angezeigt.

  3. Geben Sie im Feld Datei hochladen den Datei-Pfad an oder klicken Sie auf Durchsuchen, um die Datei zu suchen.

  4. Klicken Sie auf Upload.

    Die hochgeladene Datei wird in der Liste angezeigt. Sie kann jetzt importiert werden.

Tip:

Um Systemressourcen wieder freizugeben, empfiehlt Intershop Ihnen, die hochgeladenen Importdateien nach dem erfolgtem Importvorgang zu löschen.

Download von Export-Dateien

Exportierte Dateien können aus dem System heruntergeladen werden. Um eine exportierte Datei herunterzuladen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Klicken Sie auf Download in der Zeile Import- und Exportdateien.

    Eine Liste der bereits exportierten Dateien wird angezeigt.

  3. Klicken Sie auf einen Dateinamen, um die entsprechende Datei zu speichern.

    Der Dialog Download wird geöffnet und fordert Sie auf, die Datei zu speichern.

Tip:

Nach erfolgtem Download empfiehlt Intershop die Exportdateien zu löschen, um Systemressourcen wieder freizugeben.

Aktionen importieren

Um Aktionen aus XML-Dateien zu importieren:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Laden Sie die zu importierende Datei hoch.

    Für weitere Informationen, siehe Upload von Importdateien .

  3. Klicken Sie im Abschnitt Aktionen (XML) auf Import.

    Eine Liste der zu importierenden Dateien wird angezeigt.

  4. Markieren Sie die zu importierende Datei und klicken Sie auf Weiter.

    Die Importdatei wird jetzt im Hinblick auf das vordefinierte XML-Schema für Aktionen validiert. Ist die Validierung erfolgreich, kann die Datei importiert werden. Bei Fehlern werden Fehlermeldungen generiert. Sie können die Log-Datei auch herunterladen.

  5. Wählen Sie einen Importmodus aus.

    Die verfügbaren Importmodi sind Auslassen, Ignorieren, Aktualisieren, Ersetzen, Löschen und Erstimport. Für weitere Informationen zu Importmodi, siehe Standard-Importmodi.

  6. Klicken Sie auf Import.

    Der Importvorgang wird sofort gestartet.

    Ist der Prozess erfolgreich, wird die Anzahl importierter Aktionen angezeigt. Bei Fehlern werden Fehlermeldungen generiert. Sie können die Log-Datei auch herunterladen.

Aktionen exportieren

Der Aktionenexport basiert auf individuellen Konfigurationen oder "Szenarien". Die Konfiguration umfasst alle Einstellungen, die für die Exportprozesse zum erfolgreichen Ausführen von entsprechenden Aufgaben notwendig sind. Es können mehrere Konfigurationen angelegt, gespeichert und ausgeführt werden - entsprechend den Zielsystemen, die die exportierten Daten dann verwenden.

Neue Konfigurationen anlegen

Um eine neue Konfiguration für einen Aktionsexport zu erzeugen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Klicken Sie in der Zeile Aktionen (XML) auf Export.

    Die Seite Aktionsexporte wird angezeigt.

  3. Klicken Sie auf Neu.

    Das Register Allgemein für eine neue Konfiguration für einen Aktionsexport wird angezeigt.

  4. Bearbeiten Sie die allgemeinen Angaben wie erforderlich.

    Name, ID und Exportdateiname sind erforderlich. Optional können Sie eine Beschreibung angeben.

  5. Klicken Sie auf Übernehmen.

    Die neue Konfiguration wird angelegt. Alle anderen Register werden aktiviert.

  6. Wechseln Sie in die anderen Register, um alle nötigen Angaben zu bestimmen.

    Dazu gehören Zeitplan und Aktionen, siehe Aktions-Import und -Export.

    • Im Register Zeitplan legen Sie fest, ob der Export-Prozess, der auf der aktuellen Konfiguration basiert, manuell oder in regelmäßigen Intervallen mit einem geplanten Job ausgeführt wird.

    • Im Register Aktionen legen Sie fest, ob Aktionen dynamisch auf verschiedenen Eigenschaften basierend eingeschlossen werden sollen (letzte Änderungen, Aktivitätsstatus, bestimmtes Start-/Enddatum), oder statisch durch direkte Zuordnung.

Manuelle Aktionsexporte

Um einen Aktionsexport mit einer bestimmten Konfiguration durchzuführen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Klicken Sie in der Zeile Aktionen (XML) auf Export.

    Die Seite Aktionsexporte wird angezeigt.

  3. Markieren Sie die Konfiguration, die für den Export verwendet werden soll, und klicken Sie auf Ausführen.

    Der Export mit der gewählen Konfiguration wird gestartet. Klicken Sie auf Aktualisieren, um den Seiteninhalt neu zu laden.

Automatische Aktionsexporte

Aktionsexporte können nicht nur manuell, sondern auch automatisch durch einen geplante Jobs gestartet werden.

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Klicken Sie in der Zeile Aktionen (XML) auf Export.

    Die Seite Aktionsexporte wird angezeigt.

  3. Klicken Sie auf den Namen der gewünschten Konfiguration.

    Das Register Allgemein der Konfiguration für den Aktionsexport wird angezeigt.

  4. Öffnen Sie das Register Zeitplan.

  5. Legen Sie das Ausführungsintervall für die aktuelle Konfiguration fest und klicken Sie dann auf Übernehmen.

    Der Aktionsexport mit der gewählten Konfiguration wird als geplanter Job im definierten Intervall ausgeführt.

Aktionscodes importieren

Um Aktionscodes aus einer hochgeladenen Importdatei zu importieren, führen Sie folgende Schritte aus:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Laden Sie die zu importierende Datei hoch.

    Für weitere Informationen, siehe Upload von Importdateien .

  3. Im Abschnitt Aktionscodes (CSV) klicken Sie auf Import.

    Attribute der Importdaten und eine Liste von hochgeladenen Dateien wird angezeigt.

  4. Legen Sie Attribute abhängig von Ihren Importdaten fest.

    Die folgende Tabelle zeigt dies ausführlicher:

    Tabelle 48: Attribute von Aktionscode-Importdaten
    Attribut Beschreibung
    Zuordnungsvorlage Öffnen Sie für den Import von Aktionscodes das Template PromotionCodeCSVConvert.isml.
    Feldtrenner Zeichen für die Trennung von Feldwerten in jeder Zeile. Da Aktionscodes nur ein Feld pro Eintrag besitzen, ist dies hier nicht relevant.
    Formatzeile Bestimmen Sie die Anzahl an Formatzeilen oben in der Importdatei.
    Benutzerdefinierte Formatzeile Ermöglicht die Anpassung einer benutzerdefinierten Formatzeile.

  5. Wählen Sie das Optionsfeld neben der Datei, die Sie importieren wollen, und klicken Sie auf Weiter.

    Die Importdatei wird nun validiert, und die Zahl an Aktionscodes in der Datei wird angezeigt. Bei Fehlern wird eine entsprechende Fehlernachricht erzeugt, die Sie überprüfen können.

  6. Klicken Sie auf Weiter.

    Eine Liste verfügbarer Aktionscode-Gruppen wird angezeigt.

  7. Markieren Sie eine Aktionscode-Gruppe und klicken Sie auf Weiter.

    Der Import wird an einer einzigen Codegruppe durchgeführt, wählen Sie die zu bearbeitende aus.

  8. Wählen Sie einen Importmodus.

    Um Aktionscodes zu importieren, sind die Modi Erstimport und Löschen möglich. Für weitere Informationen zu Importmodi, siehe Standard-Importmodi.

  9. Klicken Sie auf Import.

    Der Import startet und eine Statusnachricht wird angezeigt, wenn er beendet ist.

Aktionscodes exportieren

Um Aktionscodes in eine Datei zu exportieren und herunterzuladen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Im Abschnitt Aktionscodes (CSV) klicken Sie auf Export.

  3. Bestimmen Sie die Export-Optionen und klicken Sie auf Weiter.

    Sie können entweder alle Codes exportieren oder nur die, die einen gewissen Status (eingelöst, nicht eingelöst oder deaktiviert) haben. Wählen Sie aus der Dropdown-Liste die gewünschte Option.

    Wenn Sie nur die Codes einer bestimmten Code-Gruppe exportieren möchten, entfernen Sie die Markierung beim Kontrollkästchen Alle Aktionscode-Gruppen. Somit werden Sie im nächsten Schritt aufgefordert, die Code-Gruppen auszuwählen.

  4. Optional: Wählen Sie eine oder mehrere Code-Gruppen aus und klicken Sie auf Weiter.

  5. Geben Sie einen Namen für die Exportdatei an, inklusive Dateierweiterung (*.csv).

  6. Klicken Sie auf Export.

    Sie werden auf die Übersichtsseite des Moduls Import & Export zurückgeleitet.

    Der Seitenstatus wird in regelmäßigen Intervallen aktualisiert. Warten Sie, bis der Export fertiggestellt wurde.

  7. Laden Sie die Exportdatei herunter.

    Für weitere Informationen, siehe Download von Export-Dateien.

Aktionscode-Gruppen importieren

Um Aktionscode-Gruppen aus XML-Dateien zu importieren:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Laden Sie die zu importierende Datei hoch.

    Für weitere Informationen, siehe Upload von Importdateien .

  3. Klicken Sie im Abschnitt Aktionscode-Gruppen (XML) auf Import.

    Eine Liste der zu importierenden Dateien wird angezeigt.

  4. Markieren Sie die zu importierende Datei und klicken Sie auf Weiter.

    Die Importdatei wird jetzt validiert. Bei Erfolg wird die Anzahl der Aktionscode-Gruppen angezeigt. Bei Fehlern werden Fehlermeldungen generiert. Sie können die Log-Datei auch herunterladen.

  5. Wählen Sie einen Importmodus aus.

    Die verfügbaren Importmodi sind Auslassen, Ignorieren, Aktualisieren, Ersetzen, Löschen und Erstimport. Für weitere Informationen zu Importmodi, siehe Standard-Importmodi.

  6. Klicken Sie auf Import.

    Der Importvorgang wird sofort gestartet. Der Import- und Exportstatus wird angezeigt.

Aktionscode-Gruppen exportieren

Um Aktionscode-Gruppen in eine XML-Datei zu exportieren und herunterzuladen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Import & Export.

    Das Modul Aktions-Import und -Export wird geöffnet.

  2. Klicken Sie im Abschnitt Aktionscode-Gruppen (XML) auf Export.

    Die Seite zur Auswahl der Aktionscode-Gruppen wird angezeigt.

  3. Markieren Sie die zu exportierende(n) Aktionscode-Gruppe(n), und klicken Sie auf Weiter.

    Verwenden Sie die Suche, um die Auswahl an Aktionscode-Gruppen einzuschränken. Verwenden Sie die Kontrollkästchen zur Auswahl einzelner Aktionscode-Gruppen.

    Klick auf Weiter öffnet die Seite zur Export-Konfiguration.

  4. Konfigurieren Sie die Exportoptionen, und klicken Sie dann auf Export.

    • Wählen Sie, ob Sie entweder das gesamte Suchergebnis oder nur die markierten Code-Gruppen exportieren wollen.

    • Geben Sie einen Namen für die Exportdatei an, inklusive Dateierweiterung (*.xml).

    Klick auf Export startet unmittelbar den Exportvorgang. Der Import- und Exportstatus wird angezeigt.

  5. Laden Sie die Exportdatei herunter.

    Für weitere Informationen, siehe Download von Export-Dateien.
