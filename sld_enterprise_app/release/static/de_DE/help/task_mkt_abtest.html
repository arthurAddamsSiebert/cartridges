Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext einer Applikation ausgeführt und setzen das Zugriffsrecht "Marketing-Verwalter" für die Applikation voraus.

Folgendes ist bei der Arbeit mit A/B-Tests zu beachten:

  • Die Benutzer der Kontrollgruppe verhalten sich wie Benutzer, die nicht dem A/B-Test zugeordnet wurden. Die Sichtbarkeit ihres Contents richtet sich nach den Standardoptionen für Sichtbarkeit, ungeachtet der Einstellungen oder des Status der A/B-Tests. Der dieser Gruppe zugewiesene Content muss sich daher wie jeder anderer Site-Content verhalten, mit dem einzigen Unterschied, dass er nicht für die A/B-Testgruppen sichtbar ist.

  • Ziehen Sie in Betracht, die Nutzer- und Zielgruppen des gewählten Contents zu überprüfen, weil sie möglicherweise die Zielgruppen des A/B-Tests beeinträchtigen.

  • Beachten Sie, dass Seitenbesuche während A/B-Tests durch alle Applikationen des Channels gezählt werden. Stellen Sie also sicher, dass der Test-Content allen Applikationen des Channels zugewiesen und in allen sichtbar ist.

  • Stellen Sie sicher, dass die Start-/End-/Publizierungszeiträume des ausgewählten Contents zu den Start- und Endzeiten des A/B-Tests passen.

A/B-Tests anzeigen

Um einen A/B-Test anzuzeigen:

  1. Wählen Sie den Management-Kontext aus der Kontextauswahlbox.

    Die Navigationsleiste für den Managementkontext wird geöffnet.

  2. Klicken Sie in der Navigationsleiste auf Marketing | A/B-Tests.

    Das Modul A/B-Tests wird geöffnet und zeigt eine Liste mit allen im Management-Kontext definierten A/B-Tests.

  3. Um die Liste der angezeigten A/B-Tests einzuschränken, können Sie die Filter der einfachen oder erweiterten Suche benutzen.

  4. Klicken Sie auf den Namen eines A/B-Tests, um Details anzuzeigen.

    Die Detailansicht mit dem Register Zielgruppen wird angezeigt.

A/B-Tests anlegen

Um einen A/B-Test anzulegen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | A/B Tests.

    Das Modul A/B Test wird angezeigt.

  2. Klicken Sie Neu und geben Sie die notwendigen Informationen an.

    Ein Formular zur Eingabe allgemeiner Eigenschaften von A/B-Tests wird angezeigt. Geben Sie die Informationen an, wie in folgender Tabelle beschrieben:

    Tabelle 45: Allgemeine Eigenschaften A/B-Tests
    Eigenschaft Beschreibung
    Name, Beschreibung Geben Sie einen Namen und eine Beschreibung für den neuen Test ein. Der Name wird im gesamten Commerce Management für diesen neuen Test verwendet.
    Start-/Enddatum Geben Sie ein Start-/Enddatum und eine Start-/Endzeit für den Test an.
    Maximale Teilnehmeranzahl Optional. Die Anzahl der Teilnehmer kann nach oben begrenzt werden. Wenn diese angegeben wurde, so wird der Test abgeschlossen, sobald die Teilnehmerzahl erreicht wurde, ohne das Enddatum des Tests zu berücksichtigen.
    Verhältnis der Benutzerteilnahme Legt die Prozentzahl an Kunden fest, die am Test teilnehmen. Kunden, die nicht am Test teilnehmen, erhalten den gleichen Content wie die Kontrollgruppe.
    aktiviert Markieren Sie dieses Kontrollkästchen zur Aktivierung des A/B-Tests. Neu erstellte Test sind standardmäßig deaktiviert.

  3. Klicken Sie auf Übernehmen, um einen neuen Test anzulegen.

    Der neue A/B-Test wird erstellt und die Detailansicht angezeigt.

    Zum Fertigstellen des neuen A/B-Tests fahren Sie mit folgenden Aufgaben fort:

    • Definieren Sie die Zielgruppe des Tests. Für weitere Informationen, siehe Zielgruppe verwalten .

    • Definieren Sie Testgruppen. Für weitere Informationen, siehe Testgruppen verwalten .

    • Definieren Sie Test-Content. Für weitere Informationen, siehe Test-Content verwalten .

Sie können jetzt Test- und Zielgruppen, Rabatte und Content für den A/B-Test definieren. Markieren Sie das Kontrollkästchen Aktiviert, um den A/B-Test zu aktivieren.

A/B-Test aktivieren

Damit ein A/B-Test aktiviert ist, müssen zwei Bedingungen erfüllt sein:

  • der A/B-Test-Zeitraum und

  • Aktivierungsstatus

Um einen A/B-Test zu aktivieren:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | A/B Tests.

    Das Modul A/B Test wird angezeigt.

  2. Klicken Sie auf den Namen eines A/B-Tests, um Details anzuzeigen.

    Die Detailansicht mit dem Register Zielgruppen wird angezeigt.

  3. Öffnen Sie das Register Eigenschaften.

    Das Register Eigenschaften wird angezeigt.

  4. Legen Sie den A/B-Test-Zeitraum fest.

    Geben Sie ein Start-/Enddatum und eine Start-/Endzeit für den Gültigkeitszeitraum des Tests an.

  5. Aktivieren Sie den A/B-Test

    Markieren Sie das Kontrollkästchen Aktiviert.

  6. Klicken Sie auf Übernehmen.

    Somit wird der A/B-Test aktiviert und in der Storefront für den geplanten Zeitraum verfügbar gemacht.

Deaktivierung A/B-Test

Um einen A/B-Test zu deaktivieren:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | A/B Tests.

    Das Modul A/B Test wird angezeigt.

  2. Klicken Sie auf den Namen eines A/B-Tests, um Details anzuzeigen.

    Die Detailansicht mit dem Register Zielgruppen wird angezeigt.

  3. Öffnen Sie das Register Eigenschaften.

    Das Register Eigenschaften wird angezeigt.

  4. Löschen Sie die Markierung im Kontrollkästchen Aktiviert und klicken Sie auf Übernehmen.

Löschen von A/B-Tests

Hinweis:

Wenn ein A/B-Test aktiviert und das Enddatum noch nicht erreicht ist, müssen Sie den A/B-Test vor dem Löschen zunächst deaktivieren. Für weitere Informationen, siehe Deaktivierung A/B-Test .

Um einen A/B-Test zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | A/B Tests.

    Das Modul A/B Test wird angezeigt.

  2. Zum Löschen eines A/B-Tests markieren Sie das Kontrollkästchen neben dem Test, den Sie löschen wollen. Klicken Sie dann auf Löschen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  3. Klicken Sie auf OK.

    Der A/B-Test wird aus dem System entfernt.

Zielgruppe verwalten

Sie können den Test auf bestimmte Kunden beschränken, indem Sie sie der Zielgruppe des Tests hinzufügen. Kunden werden der Zielgruppe implizit zugeordnet, indem Kundensegmente oder Affiiliate-Partner der Zielgruppe zugeordnet werden.

Um der Zielgruppe Kundensegmente oder Affiliate zuzuordnen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | A/B Tests.

    Das Modul A/B Test wird angezeigt.

  2. Klicken Sie auf den Namen eines A/B-Tests, um Details anzuzeigen.

    Die Detailansicht mit dem Register Zielgruppen wird angezeigt. Das Register zeigt eine Liste von bereits zugeordneten Kundensegmenten und Affiliate-Partnern.

  3. Klicken Sie auf Zuordnen.

    Eine Liste verfügbarer Kundensegmente bzw. Affiliate-Partner wird angezeigt. Elemente, die bereits der Zielgruppe zugeordnet wurden, haben eine Markierung im Kontrollkästchen.

  4. Markieren Sie das Kontrollkästchen des Elements, das Sie zur Zielgruppe hinzufügen wollen, und klicken Sie auf Übernehmen.

    Das Element wird zugeordnet und alle Kunden der Gruppe werden der Zielgruppe zugeordnet.

Hinweis:

Um Kundensegmente als Zielgruppe zuordnen zu können, muss mindestens ein Segmentierungsdienst für den Channel konfiguriert sein, siehe Kundensegmente.

Testgruppen verwalten

Um die Testgruppen für einen A/B-Test zu bearbeiten:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | A/B Tests.

    Das Modul A/B Test wird angezeigt.

  2. Klicken Sie auf den Namen eines A/B-Tests, um Details anzuzeigen.

    Die Detailansicht mit dem Register Zielgruppen wird angezeigt.

  3. Öffnen Sie das Register Testgruppen.

    Eine Liste aller Testgruppen wird angezeigt. Beim Anlegen des A/B-Tests erzeugt das System automatisch zwei Standard-Testgruppen (Control Group, Test Group), die jeweils 50% der Mitglieder der Zielgruppe enthalten.

  4. Klicken Sie auf Neu, um eine neue Testgruppe anzulegen.

    Geben Sie einen Namen und eine Beschreibung an.

  5. Klicken Sie auf Übernehmen, um Ihre Änderungen zu speichern.

Hinweis:

Sie können den prozentualen Anteil der Mitglieder der Zielgruppe an dieser Kontrollgruppe festlegen. Der prozentuale Anteil der Teilnehmer der übrigen Testgruppen wird automatisch angepasst (siehe Mit A/B-Tests arbeiten).

Test-Content verwalten

Im Register Content kann Content definiert werden, der für verschiedene Testgruppen angezeigt werden soll.

Um Content für eine bestimmte Testgruppe zu definieren:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | A/B Tests.

    Das Modul A/B Test wird angezeigt.

  2. Klicken Sie auf den Namen eines A/B-Tests, um Details anzuzeigen.

    Die Detailansicht mit dem Register Zielgruppen wird angezeigt.

  3. Öffnen Sie das Register Content.

    Das Register zeigt für alle vorher definierten Testgruppen eine Liste der derzeit zugeordneten Content-Objekte.

  4. Um einer Testgruppe neuen Content zuzuordnen, klicken Sie auf Zuordnen.

    Eine Liste von Content-Kategorien wird angezeigt.

  5. Wählen Sie aus der Liste eine der angezeigten Optionen und klicken Sie auf Weiter.

    Eine Liste verfügbarer Objekte wird angezeigt.

  6. Wählen Sie aus der Liste ein oder mehr Elemente und klicken Sie auf Zuordnen.

    Die gewählten Objekte werden der Testgruppe zugeordnet.

Um eine Content-Beziehung zu entfernen, wählen Sie das entsprechende Content-Objekt und klicken Sie auf Aufheben. Beachten Sie, dass Sie Content nur auf derselben Management-Kontext-Ebene aufheben können, auf der Sie die Beziehung erstellt haben.

A/B-Test-Ergebnisse anzeigen

Das Register Zusammenfassung zeigt auf einen Blick die wichtigsten Test-Ergebnisse und detaillierte Ansichten der verschiedenen Kundenaktivitäten, die das System protokolliert.

Um A/B-Test-Ergebnisse anzuzeigen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | A/B Tests.

    Das Modul A/B Test wird angezeigt.

  2. Klicken Sie auf den Namen eines A/B-Tests, um Details anzuzeigen.

    Die Detailansicht mit dem Register Zielgruppen wird angezeigt.

  3. Öffnen Sie das Register Zusammenfassung.

    Die Zusammenfassung des A/B-Tests wird angezeigt.

  4. Klappen Sie den Abschnitt/die Abschnitte aus, die Sie ansehen möchten.

    Durch Klicken auf den Titel eines Abschnitts wird der Abschnitt ausgeklappt und detaillierte Statistiken werden angezeigt.

    Neben der Übersicht sind folgende Abschnitte verfügbar:

    • Gesamtwerte

    • Konversionsraten

    • Mittelwerte

  5. Optional können Sie auf Übernehmen für Zielsystem klicken.

    Ein Dialog für die Publikation der Content-Zuordnungen einer Testgruppe in den aktivierten Shop wird geöffnet.

Durch Klicken auf Zurück zur Liste kehren Sie auf die Listenansicht des A/B-Tests zurück.
