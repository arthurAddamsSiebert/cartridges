Kataloganzeigen ermöglichen dem Katalogadministrator des Channels, die Anzeige von Produktkatalog-Inhalt kundensegmentspezifisch zu steuern.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Kataloganzeige erstellen

Um eine Kataloganzeige anzulegen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Kataloganzeigen.

    Eine Liste der bestehenden Kataloganzeigen wird angezeigt.

  2. Klicken Sie auf Neu.

    Der Dialog Kataloganzeige wird angezeigt.

  3. Geben Sie einen Namen und eine Beschreibung für die Kataloganzeige an.

  4. Markieren Sie das Kontrollkästchen Online (oder entfernen Sie die Markierung), um die Kataloganzeige online (bzw. offline) zu setzen.

  5. Wählen Sie den Anwendungsmodus (manuell oder automatisches Intervall) aus.

    Für weitere Informationen, siehe Kataloganzeigen publizieren.

  6. Öffnen Sie das Register Kataloge, um Kataloge einzufügen.

    Für weitere Informationen, siehe Kataloge in Kataloganzeigen aufnehmen.

  7. Öffnen Sie das Register Ausnahmen, um Kategorien auszublenden.

    Für weitere Informationen, siehe Kategorien oder Produkte aus Kataloganzeigen ausblenden.

  8. Wechseln Sie zum Register Zielgruppe, um Kunden oder Kundensegmente der Kataloganzeige zuzuordnen.

    Für weitere Informationen, siehe Kataloganzeigen Kunden zuordnen.

Kataloganzeige bearbeiten

Um eine Kataloganzeige zu bearbeiten:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Kataloganzeigen.

    Eine Liste der bestehenden Kataloganzeigen wird angezeigt.

  2. Klicken Sie auf den Namen der Kataloganzeige, die Sie bearbeiten wollen.

    Die Detailseite der Kataloganzeige wird angezeigt.

  3. Bearbeiten Sie die Details der Kataloganzeige wie erforderlich.

    Klicken Sie auf Übernehmen, um Ihre Änderungen zu speichern.

Kataloge in Kataloganzeigen aufnehmen

Um Produktkataloge, Kategorien oder Produkte in Kataloganzeigen aufzunehmen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Kataloganzeigen.

    Eine Liste der bestehenden Kataloganzeigen wird angezeigt.

  2. Klicken Sie auf den Namen der Kataloganzeige, die Sie bearbeiten wollen.

    Die Detailseite der Kataloganzeige wird angezeigt.

  3. Öffnen Sie das Register Kataloge.

    Eine Liste der verfügbaren Produktkataloge wird angezeigt.

  4. Markieren Sie das Kontrollkästchen des Produktkatalogs, der in die Kataloganzeige aufgenommen werden soll.

    Um nur einzelne Kategorien oder Produkte eines Kataloges in die Kataloganzeige aufzunehmen, klicken Sie auf den entsprechenden Katalognamen. Klicken Sie durch die Katalogstruktur, bis Sie die anzuzeigenden Kategorien/Produkte erreicht haben, und markieren Sie zur Auswahl die entsprechenden Kontrollkästchen.

  5. Klicken Sie auf Übernehmen.

    Ihre Einstellungen werden gespeichert, sind aber in der Kataloganzeige noch nicht wirksam.

Kategorien oder Produkte aus Kataloganzeigen ausblenden

Um Kategorien oder Produkte in Kataloganzeigen auszublenden:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Kataloganzeigen.

    Eine Liste der bestehenden Kataloganzeigen wird angezeigt.

  2. Klicken Sie auf den Namen der Kataloganzeige, die Sie bearbeiten wollen.

    Die Detailseite der Kataloganzeige wird angezeigt.

  3. Öffnen Sie das Register Ausnahmen.

    Eine Liste aller in die Kataloganzeige aufgenommenen Kataloge wird angezeigt.

  4. Markieren Sie das Kontrollkästchen des Katalogs, der in der Kataloganzeige ausgeblendet werden soll.

    Um nur einzelne Kategorien oder Produkte eines Kataloges in der Kataloganzeige auszublenden, klicken Sie auf den entsprechenden Katalognamen. Klicken Sie durch die Katalogstruktur, bis Sie die auszublendenden Kategorien/Produkte erreicht haben, und markieren Sie zur Auswahl die entsprechenden Kontrollkästchen.

  5. Klicken Sie auf Übernehmen.

    Ihre Einstellungen werden gespeichert, sind aber in der Kataloganzeige noch nicht wirksam.

Kataloganzeigen Kunden zuordnen

Sie können festlegen, für welche Zielgruppe d.h. Kunden oder Kundensegmente, die Kataloganzeige gilt. Um dies zu tun:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Kataloganzeigen.

    Eine Liste der bestehenden Kataloganzeigen wird angezeigt.

  2. Klicken Sie auf den Namen der Kataloganzeige, die Sie bearbeiten wollen.

    Die Detailseite der Kataloganzeige wird angezeigt.

  3. Öffnen Sie das Register Zielgruppe.

    Dieses Register ist in zwei Abschnitte unterteilt:

    • Kundensegmente: Zeigt alle Kundensegmente, denen die Kataloganzeige zugeordnet ist.

      Hinweis:

      Um Kundensegmente als Zielgruppe zuordnen zu können, muss mindestens ein Segmentierungsdienst für den Channel konfiguriert sein, siehe Kundensegmente.

    • Kundensegmente: Zeigt alle Kunden, denen die Kataloganzeige zugeordnet ist.

  4. Klicken Sie im entsprechenden Abschnitt (Kundensegmente/Kunden) auf Zuordnen.

    Die verfügbaren Kundensegmente bzw. Kunden werden angezeigt.

  5. Wählen Sie das gewünschte Kundensegment oder den Kunden und klicken Sie auf Ok.

    Das gewählte Kundensegment oder der Kunde ist der Kataloganzeige zugeordnet. Sie werden auf das Zielgruppenregister zurückgeleitet, welches die aktualisierte Kundensegment-/Kundenliste anzeigt.

Hinweis:

Sie können Kataloganzeigen zu Kunden oder Kundensegmenten zuordnen, auch während Kunden- oder Kundensegmentdetails bearbeitet werden, siehe dazu Kataloganzeigen für Kunden verwalten oder Kataloganzeigen für Kundensegmente verwalten.

Zuordnung von Kataloganzeigen zu Kunden aufheben

Um die Zuordnung einer Kataloganzeige zu einem Kundensegment oder Kunden aufzuheben:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Kataloganzeigen.

    Eine Liste der bestehenden Kataloganzeigen wird angezeigt.

  2. Klicken Sie auf den Namen der Kataloganzeige, die Sie bearbeiten wollen.

    Die Detailseite der Kataloganzeige wird angezeigt.

  3. Öffnen Sie das Register Zielgruppe.

    Öffnen Sie das Register Zielgruppe.

    Dieses Register ist in zwei Abschnitte unterteilt:

    • Kundensegmente: Zeigt alle Kundensegmente, denen die Kataloganzeige zugeordnet ist.

      Hinweis:

      In diesem Abschnitt werden nur Kundensegmente angezeigt, die von einem aktiven Segmentierungsdienst bereitgestellt wurden. Ist ein bestimmter Dienst deaktiviert, werden alle von ihm bereitgestellten Kundensegmente als Zielgruppen entfernt.

    • Kundensegmente: Zeigt alle Kunden, denen die Kataloganzeige zugeordnet ist.

  4. Wählen Sie die Zuordnung, die Sie löschen möchten.

    Markieren Sie die Kontrollkästchen der Kundensegmente/Kunden im entsprechenden Abschnitt (Kundensegment/Kunden), deren Zuordnung zur Kataloganzeige Sie entfernen wollen.

  5. Klicken Sie auf Zuordnung aufheben und dann auf Ok, um den Vorgang zu bestätigen.

    Die Zuordnungen der ausgewählten Kundensegmente/Kunden zur Kataloganzeige werden gelöscht.

Kataloganzeigen publizieren

Bevor die Einstellungen für eine Kataloganzeige wirksam werden, muss die Kataloganzeige angewendet werden. Eine Kataloganzeige kann manuell oder automatisch angewendet werden.

Kataloganzeige manuell anwenden

Um Kataloganzeigen manuell anzuwenden:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Kataloganzeigen.

    Eine Liste der bestehenden Kataloganzeigen wird angezeigt.

  2. Markieren Sie das Kontrollkästchen für die Kataloganzeige, die Sie anwenden wollen.

  3. Klicken Sie auf Jetzt anwenden.

    Alle zuvor durchgeführten und noch nicht angewendeten Änderungen werden sofort wirksam.

Kataloganzeigen automatisch publizieren

Um Kataloganzeigen automatisch anzuwenden:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Kataloganzeigen.

    Eine Liste der bestehenden Kataloganzeigen wird angezeigt.

  2. Klicken Sie auf den Namen der Kataloganzeige, die Sie bearbeiten wollen.

    Die Detailseite der Kataloganzeige wird angezeigt.

  3. Wählen Sie das Anwendungsintervall aus.

    Öffnen Sie die Anwendungsintervall-Dropdown-Liste und wählen Sie einen der vordefinierten Intervallwerte aus (Stündlich, Alle 6 Stunden, Alle 12 Stunden, Täglich, Wöchentlich).

  4. Klicken Sie auf Übernehmen.

    Ihre Einstellungen werden gespeichert und ein entsprechender geplanter Job wird erzeugt und aktiviert.

Kataloganzeige löschen

Um eine Kataloganzeige zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Kataloganzeigen.

    Eine Liste der bestehenden Kataloganzeigen wird angezeigt.

  2. Wählen Sie die zu löschende Kataloganzeige aus.

    Markieren Sie das Kontrollkästchen für die Kataloganzeige, die Sie löschen wollen.

  3. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Die Kataloganzeige wird aus dem System entfernt.
