Produktfilter verwalten

  • Ein Produktfilter anlegen
  • Produktfilter-Regeln bearbeiten
  • Produktfilter löschen

Katalogverwalter können Produktfilter vordefinieren, die dann z. B. für die dynamische Zuordnung von Produkten zu Katalogkatgorien oder das automatische Einpflegen von Produkten in Seitenkomponenten verfügbar sind.

Hinweis:

Produktfilter benötigen eine integrierte Suchmaschine dritter Anbieter, siehe Suchindex-Services.

Hinweis:

Um Änderungen an Produktfiltern wirksam werden zu lassen, ist es notwendig, die Produktzuordnungen zu aktualisieren. Um dies zu erreichen, führen Sie folgende Schritte aus:

  1. Führen Sie den SLD System Job Update Product Assignments.

    Hierdurch werden die Produkte ausgewählt und zugeordnet, die über die Filterbedingungen/Kategorie des vordefinierten Produktfilters, der den Solr-Suchindex abfragt, abgerufen werden. Die Ergebnisse werden von der Solr-Suchmaschine sortiert zurückgegeben, abhängig von dem Attribut, das für die Sortierung im vordefinierten Produktfilter angegeben wurde.

  2. Erzeugen Sie den Solr-Suchindex neu, um die Produkt-Kategorie-Zuordnung im Index selbst zu aktualisieren.

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Ein Produktfilter anlegen

Um einen neuen Produktfilter anzulegen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | vordefinierte Produktfilter.

    Eine Liste mit allen für diesen Channel definierten Produktfiltern wird angezeigt.

  2. Klicken Sie auf Neu.

    Eine leere Detailansicht wird geöffnet.

  3. Geben Sie die Filter-Details wie erforderlich an.

    Geben Sie Name, ID und Beschreibung (optional) sowie die gewünschten Regeln an.

    Weitere Informationen zu Produktfilter-Regeln, siehe Produktfilter-Regeln bearbeiten.

  4. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Produktfilter-Regeln bearbeiten

Um Produktfilter-Regeln zu bearbeiten:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | vordefinierte Produktfilter.

    Eine Liste mit allen für diesen Channel definierten Produktfiltern wird angezeigt.

  2. Klicken Sie auf den Namen oder ID des Produktfilters, den Sie bearbeiten wollen.

    Die Detailansicht des Produktfilters wird geöffnet.

  3. Bearbeiten Sie für jeden Abschnitt die Regeln wie erforderlich.

    Folgende Optionen stehen zur Verfügung:

    • Kategorie

      Gibt die Katalogkategorie an, für die die Produktfilter-Suchoperation vorbehalten sein soll. Ein Klick auf Auswählen öffnet einen Assistenten zur Auswahl der gewünschten Kategorie. Ein Klick auf Leeren entfernt die Kategoriebeschränkung.

    • Filterbedingung

      Spezifiziert die Textwerte (für Textattribute) oder Wertbereiche (für numerische Attribute), die vom Filter gefunden werden sollen.

    • Sortierung

      Gibt die Sortierung der gefunden Ergebnissets an. Das Sortierungsattribut ist von der definierten Filterbedingung unabhängig.

    • Ranking

      Legt eine Gewichtung der spezifischen Attributwerte an.

  4. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Produktfilter löschen

Um Produktfilter zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | vordefinierte Produktfilter.

    Eine Liste mit allen für diesen Channel definierten Produktfiltern wird angezeigt.

  2. Wählen Sie die zu löschenden Produktfilter aus.

    Markieren Sie die Kontrollkästchen neben den Produktfiltern, die Sie löschen wollen.

  3. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Die ausgewählten Produktfilter werden entfernt.
