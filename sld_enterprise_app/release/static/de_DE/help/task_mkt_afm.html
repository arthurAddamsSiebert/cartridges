Affiliate-Partner-Verwaltung

  • Affiliate-Partner anlegen
  • Affiliate-Partner löschen
  • Anlegen von Programmen für Affiliate-Partner
  • Programme von Affiliate-Partnern löschen
  • Anzeige von Zugriffsstatistiken
Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Affiliate-Partner anlegen

Um einen Affiliate-Partner anzulegen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Affiliate-Partner.

    Eine Liste aller Affiliate-Partner wird angezeigt.

  2. Klicken Sie auf Neu.

    Die Detailansicht des neuen Affiliate-Partners wird angezeigt.

  3. Legen Sie die allgemeinen Angaben für den Affiliate-Partner fest.

    Geben Sie einen Namen, einen eindeutigen ID-String und einen Beschreibungstext für den Affiliate-Partner an.

  4. Klicken Sie auf Übernehmen.

    Wenn Sie einen neuen Affiliate-Partner anlegen, so wird eine spezifische URL für den Shop des Affiliate-Partners erzeugt.

    Die URL für den Shopeingang des neuen Affiliate-Partners wird erzeugt und angezeigt. Kunden, die diese Website über diese URL erreichen, werden der spezifischen Kundegruppe zugeordnet und können entsprechend bearbeitet werden. Übermitteln Sie diese URL an den Affiliate-Partner, damit Sie in die Site des Partners integriert werden kann.

    Sie können nun die Kontaktinformation für den Partner bearbeiten, die Statistiken für den Partner einsehen, und spezielle Partnerprogramme für diesen Affiliate-Partner anlegen.

Affiliate-Partner löschen

Um einen Affiliate-Partner zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Affiliate-Partner.

    Eine Liste aller Affiliate-Partner wird angezeigt.

  2. Markieren Sie die Kontrollkästchen für die Affiliate-Partner, die Sie löschen wollen.

  3. Klicken Sie auf Löschen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  4. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Die Affiliate-Partner werden aus dem System entfernt, einschließlich aller Partnerprogramme, die für diese Partner angelegt wurden.

Hinweis:

Die URL, die für den Affiliate-Partner erzeugt wurde, bleibt auch nach Löschen des Partners gültig.

Anlegen von Programmen für Affiliate-Partner

Um ein Partnerprogramm für einen Affiliate-Partner anzulegen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Affiliate-Partner.

    Eine Liste aller Affiliate-Partner wird angezeigt.

  2. Klicken Sie auf den Affiliate-Partner, zu dem das Partnerprogramm gehört.

    Die Detailansicht des Affiliate-Partners wird angezeigt.

  3. Öffnen Sie das Register Programme.

    Eine Liste aller bestehenden Partnerprogramme wird angezeigt.

  4. Klicken Sie auf Neu.

    Die Detailansicht des Affiliate-Partnerprogamms wird angezeigt.

  5. Tragen Sie die für das Affiliate-Partnerprogramm notwendigen Daten ein.

    Die ID ist notwendig. Zusätzlich können Sie Start- und Enddatum für das Programm festlegen. Markieren Sie das Kontrollkästchen Aktiviert, um das Partnerprogramm zu aktivieren.

  6. Klicken Sie auf Übernehmen.

    Die URL für das Affiliate-Partnerprogramm wird erzeugt und angezeigt. Übermitteln Sie diese URL an den Affiliate-Partner, damit Sie in die Site des Partners integriert werden kann. Sie können nun Zugriffsstatistiken für dieses Programm einsehen.

Programme von Affiliate-Partnern löschen

Um ein Partnerprogramm eines Affiliate-Partners zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Affiliate-Partner.

    Eine Liste aller Affiliate-Partner wird angezeigt.

  2. Klicken Sie auf den Affiliate-Partner, zu dem das Partnerprogramm gehört.

    Die Detailansicht des Affiliate-Partners wird angezeigt.

  3. Öffnen Sie das Register Programme.

    Eine Liste aller bestehenden Partnerprogramme wird angezeigt.

  4. Markieren Sie das Programm, das Sie löschen wollen.

  5. Klicken Sie auf Löschen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  6. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Das Partnerprogramm wird aus dem System entfernt.

Hinweis:

Die URL, die für das Affiliate-Partnerprogramm erzeugt wurde, bleibt auch nach Löschen des Programms gültig.

Anzeige von Zugriffsstatistiken

Um Zugriffsstatistiken für einen Affiliate-Partner oder ein Partnerprogramm anzeigen zu lassen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Marketing | Affiliate-Partner.

    Eine Liste aller Affiliate-Partner wird angezeigt.

  2. Klicken Sie auf den gewünschten Affiliate-Partner.

    Die Detailansicht des Affiliate-Partners wird angezeigt.

  3. Öffnen Sie das Register Statistik.

    Eine Gesamtstatistik für diesen Affiliate-Partner wird angezeigt, einschließlich aller Benutzer und aller Partnerprogramme.

  4. Öffnen Sie das Register Programme und klicken Sie auf das gewünschte Partnerprogramm.

    Die Detailansicht des Affiliate-Partnerprogramms wird angezeigt.

  5. Öffnen Sie das Register Statistik.

    Eine Gesamtstatistik für dieses Affiliate-Partnerprogramm wird angezeigt, einschließlich aller Benutzer, die die Storefront über dieses Programm erreicht haben.
