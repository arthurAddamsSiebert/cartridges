Kundensegmente anlegen

Um ein neues benutzerdefiniertes Kundensegment anzulegen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Kunden | Kundensegmente.

    Eine Liste aller benutzerdefinierten Kundensegmente wird angezeigt.

    Hinweis:

    Implizite Kundensegmente werden nicht angezeigt.

  2. Klicken Sie auf Neu.

    Die Detailansicht für Kundensegmente wird angezeigt.

  3. Geben Sie im Register Allgemein alle notwendigen Daten für das neue Kundensegment an.

    Name und ID sind erforderlich. Das Feld Beschreibung ist optional.

  4. Klicken Sie auf Übernehmen.

    Ein neues Kundensegment wird angelegt. Das Register Kunden wird aktiviert und Sie können nun Kunden zuordnen.

  5. Öffnen Sie das Register Kunden.

  6. Klicken Sie auf Neu, um Kunden einem Kundensegment zuzuordnen.

    Die Kunden-Listenansicht wird angezeigt.

  7. Wählen Sie die Kunden für das Kundensegment aus und klicken Sie auf Zuordnen.

Zugeordnete Kunden verwalten

Die Verwaltung zugeordneter Kunden umfasst das Hinzufügen und Löschen von Kundenzuordnungen.

Kunden hinzufügen

Um Kunden zu einem Kundensegment hinzuzufügen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Kunden | Kundensegmente.

    Eine Liste aller benutzerdefinierten Kundensegmente wird angezeigt.

    Hinweis:

    Implizite Kundensegmente werden nicht angezeigt.

  2. Wählen Sie den Namen des Kundensegments, das bearbeitet werden soll.

    Die Detailansicht für Kundensegmente wird angezeigt.

  3. Öffnen Sie das Register Kunden.

    Eine Liste aller Kunden des aktuellen Kundensegments wird angezeigt.

  4. Klicken Sie auf Zuordnen.

    Eine Liste verfügbarer Kunden wird angezeigt.

  5. Markieren Sie das Kontrollkästchen des Kunden, der zugeordnet werden soll.

  6. Klicken Sie auf Zuordnen.

    Der ausgewählte Kunde wird dem aktuellen Kundensegment zugeordnet.

Kunden löschen

Um Kunden aus einem Kundensegment zu entfernen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Kunden | Kundensegmente.

    Eine Liste aller benutzerdefinierten Kundensegmente wird angezeigt.

    Hinweis:

    Implizite Kundensegmente werden nicht angezeigt.

  2. Wählen Sie den Namen des Kundensegments, das bearbeitet werden soll.

    Die Detailansicht für Kundensegmente wird angezeigt.

  3. Öffnen Sie das Register Kunden.

    Eine Liste aller Kunden des aktuellen Kundensegments wird angezeigt.

  4. Wählen Sie das Kontrollkästchen der Kundenzuordnung aus, die gelöscht werden soll.

  5. Klicken Sie auf Aufheben und bestätigen Sie die Operation.

    Die ausgewählte Kundenzuordnung wird entfernt.

Kundensegment bearbeiten

Um ein benutzerdefiniertes Kundensegment zu bearbeiten:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Kunden | Kundensegmente.

    Eine Liste aller benutzerdefinierten Kundensegmente wird angezeigt.

    Hinweis:

    Implizite Kundensegmente werden nicht angezeigt.

  2. Wählen Sie den Namen des Kundensegments, das bearbeitet werden soll.

    Die Detailansicht für Kundensegmente wird angezeigt.

  3. Bearbeiten Sie die Einstellungen wie erforderlich und klicken Sie auf Übernehmen.

Hinweis:

Implizite Kundensegmente können nicht bearbeitet werden.

Preislisten für Kundensegmente verwalten

Preislisten ermöglichen die Festlegung von gestaffelten, zielgruppenspezifischen Preisen für ein Produkt, siehe Spezialpreislisten und Staffelpreise.

Neue Preislisten für Kundensegmente erstellen

Um eine neue Preisliste für ein Kundensegment anzulegen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Kunden | Kundensegmente.

    Eine Liste aller benutzerdefinierten Kundensegmente wird angezeigt.

    Hinweis:

    Implizite Kundensegmente werden nicht angezeigt.

  2. Klicken auf den Namen oder die ID des Kundensegments.

    Die Detailansicht des Kunden wird angezeigt.

  3. Zum Register Preisliste wechseln.

    Dieses Register zeigt die Preislisten, die dem Kundensegment zugeordnet sind.

  4. Klicken Sie auf Neu.

    Dadurch wird der neue Preislisten-Dialog geöffnet.

  5. Definieren Sie Namen und ID für die Preisliste, geben Gültigkeitsdaten an, falls erforderlich, und klicken Sie auf Übernehmen.

    Werden keine Gültigkeitsdaten angegeben, gilt die Preisliste für einen unbegrenzten Zeitraum.

  6. Öffnen Sie das Register Preise, um alle Produkte anzuzeigen, die Preise für diese Preisliste definiert haben.

    Für weitere Informationen zum Definieren von Preisen in Spezialpreislisten. siehe Staffelpreise verwalten.

  7. Öffnen Sie das Register Zielgruppe, um festzulegen, für welche Kundensegmente und Kunden diese Preislisten verfügbar sein sollen.

    Die neue Preisliste wird standardmäßig dem aktuellen Kundensegment zur Verfügung gestellt.

    Um zum Register Preisliste der Kundensegment-Detailansicht zurückzukehren, klicken Sie in der Liste auf Zurück.

Preislisten Kundensegmenten zuordnen

Um eine bestehende Preisliste einem Kundensegment zuzuordnen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Kunden | Kundensegmente.

    Eine Liste aller benutzerdefinierten Kundensegmente wird angezeigt.

    Hinweis:

    Implizite Kundensegmente werden nicht angezeigt.

  2. Klicken auf den Namen oder die ID des Kundensegments.

    Die Detailansicht des Kunden wird angezeigt.

  3. Zum Register Preisliste wechseln.

    Dieses Register zeigt die Preislisten, die dem Kundensegment zugeordnet sind.

  4. Klicken Sie auf Zuordnen.

    Die verfügbaren Preislisten werden angezeigt.

  5. Wählen Sie die gewünschte Preisliste aus und klicken Sie auf OK.

    Die gewählte Preisliste ist dem Kundensegment zugeordnet. Sie werden auf das Register Preisliste zurückgeleitet, das den aktualisierten Abschnitt mit Preislisten anzeigt.

Zuordnung von Preislisten zu Kundensegmenten aufheben

Um eine Zuordnung einer Preisliste zu einem Kundensegment aufzuheben:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Kunden | Kundensegmente.

    Eine Liste aller benutzerdefinierten Kundensegmente wird angezeigt.

    Hinweis:

    Implizite Kundensegmente werden nicht angezeigt.

  2. Klicken auf den Namen oder die ID des Kundensegments.

    Die Detailansicht des Kunden wird angezeigt.

  3. Zum Register Preisliste wechseln.

    Dieses Register zeigt die Preislisten, die dem Kundensegment zugeordnet sind.

  4. Wählen Sie die Zuordnung, die Sie löschen möchten.

    Markieren Sie im Abschnitt kundenspezifischer Preislisten die Kontrollkästchen der Preislisten, deren Zuordnung zu einem Kundensegment Sie aufheben wollen.

  5. Klicken Sie auf Aufheben und dann auf Ok, um den Vorgang zu bestätigen.

    Die Zuordnung der ausgewählten Preislisten zum Kundensegment wurden aufgehoben.

Kataloganzeigen für Kundensegmente verwalten

Kataloganzeigen ermöglichen die Kontrolle der Sichtbarkeit des Produktkatalog-Contents auf einer kundensegmentspezifischen Ebene, siehe Kataloganzeigen.

Neue Kataloganzeigen für Kundensegmente erstellen

Um eine neue Kataloganzeige für ein Kundensegment zu erstellen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Kunden | Kundensegmente.

    Eine Liste aller benutzerdefinierten Kundensegmente wird angezeigt.

    Hinweis:

    Implizite Kundensegmente werden nicht angezeigt.

  2. Klicken auf den Namen oder die ID des Kundensegments.

    Die Detailansicht des Kunden wird angezeigt.

  3. Zum Register Kataloganzeige wechseln.

    Dieses Register zeigt die Kataloganzeigen, die dem Kundensegment zugeordnet sind.

  4. Klicken Sie auf Neu.

    Der Dialog Neue Kataloganzeige wird geöffnet.

  5. Geben Sie einen Namen und eine Beschreibung für die Kataloganzeige an.

  6. Markieren Sie das Kontrollkästchen Online (oder entfernen Sie die Markierung), um die Kataloganzeige online (bzw. offline) zu setzen.

  7. Wählen Sie den Anwendungsmodus (manuell oder automatisches Intervall) aus.

    Für weitere Informationen, siehe Kataloganzeigen publizieren.

  8. Öffnen Sie das Register Kataloge, um Kataloge einzufügen.

    Für weitere Informationen, siehe Kataloge in Kataloganzeigen aufnehmen.

  9. Öffnen Sie das Register Ausnahmen, um Kategorien auszublenden.

    Für weitere Informationen, siehe Kategorien oder Produkte aus Kataloganzeigen ausblenden.

  10. Wechseln Sie zum Register Zielgruppe, um Kunden oder Kundensegmente der Kataloganzeige zuzuordnen.

    Die neue Kataloganzeige wird standardmäßig dem aktuellen Kundensegment zur Verfügung gestellt. Für weitere Informationen, siehe Kataloganzeigen Kunden zuordnen.

    Um zum Register Kataloganzeige der Kundensegment-Detailansicht zurückzukehren, klicken Sie in der Liste auf Zurück.

Kundensegment löschen

Um ein oder mehr benutzerdefinierte Kundensegmente zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Kunden | Kundensegmente.

    Eine Liste aller benutzerdefinierten Kundensegmente wird angezeigt.

    Hinweis:

    Implizite Kundensegmente werden nicht angezeigt.

  2. Markieren Sie das Kontrollkästchen für die Kundensegmente, die Sie löschen wollen.

  3. Klicken Sie auf Löschen.

    Ein Dialog zur Bestätigung des Löschvorgangs wird angezeigt.

  4. Klicken Sie auf OK, um das Löschen zu bestätigen.

    Die Kundensegmente werden aus dem System entfernt.

Hinweis:

Implizite Kundensegmente können nicht gelöscht werden.
