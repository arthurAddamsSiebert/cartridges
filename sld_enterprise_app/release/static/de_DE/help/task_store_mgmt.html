Verwaltung von Filialen

  • Filialen anzeigen
  • Eine Filiale anlegen
  • Eine Filiale bearbeiten
  • Eine Filiale löschen
Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext eines Channels ausgeführt.

Filialen anzeigen

Um alle im Channel angelegten Filialen anzuzeigen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox.

    Die Navigationsleiste für den Channel wird geöffnet.

  2. Klicken Sie in der Navigationsleiste auf Filialen.

    Die Übersichtsseite wird angezeigt.

  3. Klicken Sie auf Filialen.

    Eine Liste aller im Channel definierten Filialen wird angezeigt.

Eine Filiale anlegen

Um eine neue Filiale anzulegen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Filialen | Filialen.

    Eine Liste aller im Channel definierten Filialen wird angezeigt.

  2. Klicken Sie auf Neu.

    Die Detailansicht der Filiale wird angezeigt.

  3. Geben Sie die notwendigen Daten an.

    Erforderliche Angaben sind Filial-ID, Land, Ort, PLZ, Straße/Nummer sowie Bundesland. Markieren Sie das Kontrollkästchen Aktiv, um die Filiale zu aktivieren.

  4. Klicken Sie auf Übernehmen.

    Die Filiale wird angelegt und das Register Attribute wird aktiviert.

  5. Optional, legen Sie die Attribute wie benötigt fest und klicken Sie auf Hinzufügen.

    Klicken Sie auf Zurück zur Liste, um zurück zur Filial-Liste zu gelangen.

Eine Filiale bearbeiten

Um Details einer Filiale zu bearbeiten:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Filialen | Filialen.

    Eine Liste aller im Channel definierten Filialen wird angezeigt.

  2. Klicken Sie auf den Namen der Filiale, die Sie bearbeiten wollen.

    Die Detailansicht der Filiale wird angezeigt.

  3. Bearbeiten Sie die Informationen wie erforderlich.

    Im Register Attribute können Sie benutzerdefinierte Filialattribute verwalten.

  4. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    Anderenfalls klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen.

Eine Filiale löschen

Um eine im Channel definierte Filiale zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Filialen | Filialen.

    Eine Liste aller im Channel definierten Filialen wird angezeigt.

  2. Markieren Sie das Kontrollkästchen für die Filiale, die Sie löschen wollen.

  3. Klicken Sie auf Löschen und bestätigen Sie die Operation.

    Die markierte Filiale wird aus dem Channel entfernt.
