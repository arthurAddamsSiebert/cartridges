Dieser Abschnitt beschreibt die Aufgaben zur Verwaltung von Produktfreigabe-Beziehungen. Dieses Modul Produktfreigaben besteht aus zwei Teilen:

  • Channels

    Zeigt die Freigabe-Beziehungen, d.h. die Channels, für die Produktdaten freigegeben sind.

  • Freigabe-Gruppen

    Zeigt die Freigabe-Gruppen (Produktdaten-Sets, die freigegeben werden sollen).

Hinweis:

Die in diesem Abschnitt beschriebenen Aufgaben werden im Kontext der Organisation oder eines Partner Channels ausgeführt.

Channel-Zuordnungen verwalten

Das Verwalten von Channel-Zuordnungen umfasst das Anlegen und Entfernen von Produktfreigabe-Beziehungen.

Channel zuweisen

Um eine Freigabe-Beziehung für einen Channel hinzuzufügen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Klicken Sie im Abschnitt Channels auf Neu.

    Der Dialog Channel hinzufügen wird geöffnet und zeigt eine Liste der Channel, die derzeit nicht als Ziele von Produktfreigaben gewählt sind.

  3. Markieren Sie den/die entsprechenden Channel(s).

    Um einen Channel für Produktfreigaben auszuwählen, markieren Sie das Kontrollkästchen neben dem entsprechenden Channel.

  4. Klicken Sie auf OK.

    Die Freigabe-Beziehung wird eingerichtet, d.h. die ausgewählten Channels können jetzt Master-Produkte mitnutzen.

    Weitere Informationen zum Verwalten von Produktdaten-Zuordnungen finden Sie in: Produktdaten-Zuordnungen verwalten.

Channel-Zuordnung aufheben

Um eine Freigabe-Beziehung für einen Channel zu entfernen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Wählen Sie im Abschnitt Channels den Channel, dessen Zuordnung zu Produktfreigaben aufgehoben werden soll.

    Um einzelne Channels zu entfernen, markieren Sie die entsprechenden Kontrollkästchen. Um alle Channels zu entfernen, klicken Sie auf Alle auswählen im Tabellenkopf.

  3. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Die gewählten Freigabe-Beziehungen werden gelöscht, d.h. die ausgewählten Channels können jetzt die Master-Produkte nicht mehr mitnutzen.

Hinweis:

Wird eine Freigabe-Beziehung, die für einen Channel eingerichtet wurde, gelöscht, werden alle freigegebenen Produkte aus dem Channel entfernt. Änderungen, die an den freigegebenen Produkten vorgenommen wurden, bleiben bestehen; wird die Freigabe-Beziehung mit den gleichen Produkten erneut eingerichtet, werden die vorhergehenden Änderungen automatisch wiederhergestellt.

Produktdaten-Zuordnungen verwalten

Um Produktzuordnungen für Freigabe-Beziehungen zu verwalten:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Klicken Sie im Abschnitt Channel auf den Namen des Channels, den Sie bearbeiten möchten.

    Der Dialog Produkt-Freigaben für den Channel wird geöffnet.

  3. Wählen Sie den Umfang der Produkte aus, auf den die Freigabe angewendet werden soll.

    Folgende Optionen (Optionsfelder) stehen zur Verfügung:

    Tabelle 25: Produktzuordnungs-Optionen
    Option Beschreibung
    Nichts freigeben Keine Produkte werden für den aktuellen Channel freigegeben.
    Alle Produkte freigeben Alle Produkte im Produktbestand werden für den aktuellen Channel freigegeben.
    Produkte über Freigabe-Gruppen freigeben Produkte einer Freigabe-Gruppe werden für den aktuellen Channel freigegeben.

  4. Klicken Sie auf Übernehmen.

    Wenn Sie "Nichts freigeben" oder "Alle Produkte freigeben" gewählt haben, sind Sie jetzt fertig. Die entsprechenden Optionen sind aktiviert.

    Wenn Sie "Produkte über Freigabe-Gruppen freigeben" gewählt haben, wird der Abschnitt Freigabe-Gruppen angezeigt. Fahren Sie mit den weiteren Schritten fort.

    Tip:

    Mit Klick auf Zurücksetzen werden die vorher gewählten Optionen wiederhergestellt.

  5. Klicken Sie im Abschnitt Freigabe-Gruppen auf Neu.

    Eine Liste der verfügbaren Freigabe-Gruppen wird angezeigt. Informationen zum Verwalten von Freigabe-Gruppen, siehe Freigabe-Gruppen verwalten.

  6. Wählen Sie die gewünschte/n Freigabe-Gruppe/n.

    Um einzelne Freigabe-Gruppen auszuwählen, markieren Sie das jeweilige Kontrollkästchen. Um alle Freigabe-Gruppen auszuwählen, wählen Sie Alle im Tabellenkopf.

  7. Klicken Sie auf OK, um die Einstellung zu speichern.

    Anderenfalls klicken Sie auf Abbrechen, um die Einstellungen zu verwerfen.

Hinweis:

Falls Sie die Markierung von "Produkte über Freigabe-Gruppen freigeben" entfernen, wird die entsprechende Freigabe-Gruppen-Zuordnung entfernt. Um die Option "Produkte über Freigabe-Gruppen freigeben" wieder herzustellen, müssen Sie erneut eine Freigabe-Gruppe zuordnen.

Freigabe-Gruppen verwalten

Das Verwalten von Freigabe-Gruppen umfasst das Anlegen und Entfernen von Freigabe-Gruppen, die Verwaltung von Produkt-Zuordnungen sowie der Channel-Zuordnungen.

Freigabe-Gruppe anlegen

Um eine Freigabe-Gruppe anzulegen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Klicken Sie im Abschnitt Freigabe-Gruppen auf Neu.

    Eine leere Detailansicht der Freigabe-Gruppe wird geöffnet.

  3. Geben Sie einen Namen an.

    Optional können Sie eine Beschreibung angeben.

  4. Klicken Sie auf Übernehmen, um die neue Freigabe-Gruppe anzulegen.

    Anderenfalls klicken Sie auf Abbrechen, um die Eingaben zu verwerfen.

Freigabe-Gruppe entfernen

Um eine Freigabe-Gruppe zu löschen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Markieren Sie im Abschnitt Freigabe-Gruppen das Kontrollkästchen neben der Freigabe-Gruppe, die Sie löschen wollen.

  3. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Die gewählte Freigabe-Gruppe wurde entfernt.

Achtung:

Wenn eine Freigabe-Gruppe gelöscht wird, werden alle mit dieser Gruppe eingerichteten Freigabe-Beziehungen entfernt, d.h. die freigegebenen Produkte sind im betroffenen Channel nicht mehr verfügbar.

Produktzuordnungen von Freigabe-Gruppen verwalten

Das Verwalten von Produktzuordnungen von Freigabe-Gruppen beinhaltet das Hinzufügen von Produkten zu Freigabe-Gruppen entweder durch Produktauswahlregeln oder durch Durchsuchen oder Suchen nach den gewünschten Produkten sowie das Entfernen von Produkten aus der Freigabe-Gruppe.

Tip:

Sie können Produkte einer Freigabe-Gruppe durch Auswahlregeln oder Suchen/Durchsuchen zuordnen. Eine Veränderung des Zuordnungsprinzips hebt alle Produktzuordnungen auf.

Produkte über Auswahlregeln zuordnen

Sie können das Produkt einer bestimmten Marke bzw. das Produkt einer ausgewählten Kategorie zuordnen. Um Produkte zu einer Freigabe-Gruppe mittels Auswahlregeln hinzuzufügen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Klicken Sie im Abschnitt Freigabe-Gruppen auf den Namen der Freigabe-Gruppe, die Sie bearbeiten wollen.

    Die Detailansicht der Freigabe-Gruppe wird geöffnet.

  3. Öffnen Sie das Register Produkte.

  4. Markieren Sie "Produkte basierend auf Produktauswahlregeln freigeben" und klicken Sie auf Übernehmen.

    Der Dialog Produktauswahlregeln wird angezeigt.

  5. Um Produkte einer bestimmten Marke/ eines Herstellers zuzuordnen, wählen Sie die gewünschten Hersteller aus und bewegen Sie sie in das Feld "Zugeordnete Hersteller".

    Tip:

    Hersteller, die einem Hersteller-Alias zugeordnet wurden (siehe Verwaltung von Hersteller-Aliassen) sind auf der Liste nicht einzeln aufgeführt, sondern nur unter dem gemeinsamen Alias-Namen.

  6. Um Produkte einer bestimmten Kategorie zuzuordnen, wählen Sie die gewünschten Kategorien.

    Durch einen Klick auf Auswählen wird ein Assistent zur Navigation durch verfügbare Kategorien und zur Auswahl einer individuellen Kategorie geöffnet.

    Wählen Sie, ob Sie die Unterkategorien einbeziehen wollen oder nicht.

  7. Klicken Sie auf Übernehmen.

    Der Prozess für die Produktzuordnung wird gestartet.

Produkte über Suchen/Durchsuchen zuordnen

Um Produkte zu einer Freigabe-Gruppe über Suchen/Durchsuchen hinzuzufügen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Klicken Sie im Abschnitt Freigabe-Gruppen auf den Namen der Freigabe-Gruppe, die Sie bearbeiten wollen.

    Die Detailansicht der Freigabe-Gruppe wird geöffnet.

  3. Öffnen Sie das Register Produkte.

  4. Wählen Sie "Nur markierte Produkte freigeben" und klicken Sie auf Übernehmen.

  5. Klicken Sie auf Zuordnen über Suche oder auf Zuordnen über Durchsuchen.

    "Zuordnen über Suche" öffnet die Produkt-Listenansicht und zeigt alle Produkte im Produktbestand, die derzeit keiner Freigabe-Gruppe zugeordnet sind. Verwenden Sie die einfache oder erweiterte Suche, um Produkte zu finden, die Sie der Freigabe-Gruppe zuordnen wollen.

    "Zuordnen über Durchsuchen" öffnet eine List der verfügbaren Kataloge. Um bestimmte Katalogkategorien zu durchsuchen, markieren Sie das Optionsfeld für den Katalog und klicken Sie auf Weiter. Klicken Sie auf die Kategorien und Unterkategorien, bis Sie die gewünschte Kategorie oder Produktliste erreicht haben.

  6. Ordnen Sie die Produkte zu.

    Wenn Sie "Zuordnen über Suchen" gewählt haben, klicken Sie auf Alle zuordnen, um alle Produkte des Suchergebnisses zur Freigabe-Gruppe hinzuzufügen. Um einzelne Produkte hinzuzufügen, markieren Sie die entsprechenden Kontrollkästchen und klicken Sie auf Auswahl zuordnen.

    Wenn Sie "Zuordnen über Durchsuchen" gewählt haben, markieren Sie das Kontrollkästchen neben einer Kategorie und klicken Sie auf Alle zuordnen, um alle Produkte dieser Kategorie der Freigabe-Gruppe zuzuordnen. Um einzelne Produkte aus einer Kategorie zuzuordnen, öffnen Sie alle Unterkategorien bis zur Produktliste und klicken dann auf Alle zuordnen, um alle Produkte aus dieser Liste zuzuordnen, oder wählen Sie einzelne Produkte aus der Liste und klicken Sie auf Auswahl zuordnen.

Produktzuordnung über Auswahlregeln aufheben

Um über Auswahlregeln zugeordnete Produkte aus einer Freigabe-Gruppe zu entfernen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Klicken Sie im Abschnitt Freigabe-Gruppen auf den Namen der Freigabe-Gruppe, die Sie bearbeiten wollen.

    Die Detailansicht der Freigabe-Gruppe wird geöffnet.

  3. Öffnen Sie das Register Produkte.

    Das Register zeigt die derzeit aktive(n) Auswahlregel(n) und die Liste einbezogener Produkte.

  4. Entfernen Sie die gewünschte Marke oder Kategorie, um die entsprechenden Produkte von der Freigabe-Gruppe auszuschließen.

    Um Produkte einer bestimmten Marke/eines Herstellers auszuschließen, verschieben Sie die gewünschte Marke/den Hersteller in die Box "Verfügbare Hersteller".

    Um Produkte einer bestimmten Kategorie auszuschließen, markieren Sie die gewünschte Kategorie in der Box "Kategorien" und klicken Sie Entfernen.

  5. Klicken Sie auf Übernehmen.

    Die Zuordnung des gewählten Herstellers oder der Kategorie zur Freigabe-Gruppe wird entfernt.

Per Suche zugeordnete Produkte entfernen

Um über Suchen/Durchsuchen zugeordnete Produkte aus einer Freigabe-Gruppe zu entfernen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Klicken Sie im Abschnitt Freigabe-Gruppen auf den Namen der Freigabe-Gruppe, die Sie bearbeiten wollen.

    Die Detailansicht der Freigabe-Gruppe wird geöffnet.

  3. Öffnen Sie das Register Produkte.

    Dieses Register zeigt eine Liste der Produkte, die derzeit der Freigabe-Gruppe zugeordnet sind.

  4. Wählen Sie die Produkte, die aus der Freigabe-Gruppe entfernt werden sollen.

    Um einzelne Produkte zu entfernen, markieren Sie die entsprechenden Kontrollkästchen. Um alle Produkte zu entfernen, klicken Sie auf Alle im Tabellenkopf.

  5. Klicken Sie auf Aufheben, dann auf OK, um das Löschen zu bestätigen.

    Die ausgewählten Produktzuordnungen werden entfernt.

Channel-Zuordnungen von Freigabe-Gruppen verwalten

Das Verwalten von Channel-Zuordnungen für Freigabe-Gruppen umfasst das Anlegen und Entfernen von Channels.

Channel einer Freigabe-Gruppe zuordnen

Um einen Channel zu einer Freigabe-Gruppe hinzuzufügen.

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Klicken Sie im Abschnitt Freigabe-Gruppen auf den Namen der Freigabe-Gruppe, die Sie bearbeiten wollen.

    Die Detailansicht der Freigabe-Gruppe wird geöffnet.

  3. Öffnen Sie das Register Channels.

    Das Register zeigt alle Channels an, die derzeit der Freigabe-Gruppe zugeordnet sind.

  4. Klicken Sie auf Neu.

    Eine Liste aller Channels, die derzeit nicht dieser Freigabe-Gruppe zugeordnet sind, wird angezeigt.

  5. Markieren Sie den/die entsprechenden Channel(s).

    Um einen Channel für Produktfreigaben auszuwählen, markieren Sie das Kontrollkästchen neben dem entsprechenden Channel.

  6. Klicken Sie auf OK.

    Eine Freigabe-Beziehung für den gewählten Channel mit einer Produktdaten-Zuordnung per Freigabe-Gruppe wird sofort eingerichtet.

Produkt-Freigabe-Channel-Zuordnung aufheben

Um einen Channel aus einer Freigabe-Gruppe zu entfernen:

  1. Wählen Sie den Channel aus der Kontextauswahlbox, wählen Sie dann Katalog | Ausgehende Produktfreigabe.

    Das Modul Produktfreigaben wird geöffnet.

  2. Klicken Sie im Abschnitt Freigabe-Gruppen auf den Namen der Freigabe-Gruppe, die Sie bearbeiten wollen.

    Die Detailansicht der Freigabe-Gruppe wird geöffnet.

  3. Öffnen Sie das Register Channels.

    Das Register zeigt alle Channels an, die derzeit der Freigabe-Gruppe zugeordnet sind.

  4. Wählen Sie die Channels, die aus der Freigabe-Gruppe entfernt werden sollen.

    Um einzelne Channels zu entfernen, markieren Sie die entsprechenden Kontrollkästchen. Um alle Channels zu entfernen, klicken Sie auf Alle auswählen im Tabellenkopf.

  5. Klicken Sie auf Löschen, dann auf OK, um das Löschen zu bestätigen.

    Die Zuordnung des Channels zur Freigabe-Gruppe wird entfernt.

Hinweis:

Wird ein Channel aus einer Freigabe-Gruppe gelöscht, wird die Freigabe-Beziehung entfernt, die diese Channel-Zuordnung verwendet.
